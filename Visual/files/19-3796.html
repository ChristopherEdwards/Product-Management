PSGW INPUT AOU INP SITE (3796)    OPTION (19)

Name Value
NAME PSGW INPUT AOU INP SITE
MENU TEXT Identify AOU INPATIENT SITE
UPPERCASE MENU TEXT IDENTIFY AOU INPATIENT SITE
DESCRIPTION 
This option will loop through the PHARMACY AOU STOCK file (#58.1) and
locate any active AOU that does not have the INPATIENT SITE field
defined.
