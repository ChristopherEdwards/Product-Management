NAME GMRV-PULSE RATE HELP
HEADER Pulse
TEXT 
Answer must be numeric between 0 and 300 with no decimal places.
 
If no letter is appended to the end of the pulse, the reading will be
considered as a RADIAL pulse.  If a letter is appended to the end of
the reading then the following is a list of valid letters, and the types
of pulse they represent:
 
                     R = RADIAL     P=PERIPHERAL
                     A = APICAL     B=BILATERAL PERIPHERALS
                     C = CAROTID    O=OTHER      or
                     site defined qualifiers.
  
Default: Site configurable
  
E.g., 76 or 76R    for 76 RADIAL
      76A or 76a   for 76 APICAL
