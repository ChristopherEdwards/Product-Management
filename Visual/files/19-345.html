NAME XMQUEUED
MENU TEXT Queues with Messages to Transmit Report
UPPERCASE MENU TEXT QUEUES WITH MESSAGES TO TRANSM
DESCRIPTION 
This option produces a report of all those queues that have messages
to go out.  The list does not say whether there is or is not a task
scheduled to send them.  Requeuing them is okay even if there is already
a task to go out.
CREATOR USER,ONE
TYPE run routine
DISPLAY OPTION? YES
PACKAGE MAILMAN
