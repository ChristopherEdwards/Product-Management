PSO BINGO AUTOSTART (8654)    OPTION (19)

Name Value
NAME PSO BINGO AUTOSTART
MENU TEXT Start Bingo Board
UPPERCASE MENU TEXT START BINGO BOARD
DESCRIPTION 
This option is used to automatically start the bingo board. It can be
setup to run on a regular schedule through the Taskman Schedule option.
CREATOR USER,ONE
TYPE run routine
PACKAGE OUTPATIENT PHARMACY
