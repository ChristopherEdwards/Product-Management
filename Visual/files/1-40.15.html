OPC ERRORS (40.15)    FILE (1)

Name Value
NAME OPC ERRORS
DESCRIPTION 
This file contains the field name(s) and number(s) which is/are
required and was/were found to be invalid.  Once the invalid
data is entered the entry will be automatically purged from the
file.  Accurate reporting on the Amis 223 is dependent on the
maintenance of this file utilizing the "OPC MENU" option
in the SD module.
