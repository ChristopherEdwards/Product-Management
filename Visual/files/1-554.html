NAME CMOP OPERATIONS
DESCRIPTION 
This file will contain information regarding the operations of the CMOP
software. The information will include data on the background jobs which
release prescriptions,file data,print transmission labels, compile cost
information and purge files. Data in this file is maintained by the 
Consolidated Mail Outpatient Pharmacy software and must not be edited 
using VA FileMan.
