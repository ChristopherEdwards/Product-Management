DESCRIPTION 
Enter/edit of the MANUFACTURER file. This file is maintained by the
Engineering Service Center. Sites may add entries as necessary, using a
MFGID of 50000 or greater. Application managers should check carefully to
insure that any local entries are not, in fact, duplications. Only the
pseudonym field can be edited for national entries.
