EAS EZ REMOVE SIGNATURE (9193)    OPTION (19)

Name Value
NAME EAS EZ REMOVE SIGNATURE
MENU TEXT Remove Signature Verification
UPPERCASE MENU TEXT REMOVE SIGNATURE VERIFICATION
DESCRIPTION 
Allows the user to remove Signature Verification from the 1010EZ
Application.  The status of the Application reverts from 'Signed'
to either 'Printed, Pending Signature' or 'In Review'.
CREATOR USER,ONE
TYPE run routine
PACKAGE ENROLLMENT APPLICATION SYSTEM
ROUTINE REMSIG^EASEZU5
