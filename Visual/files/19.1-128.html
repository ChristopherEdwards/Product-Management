RT RAD-FR-SUPERVISOR (128)    SECURITY KEY (19.1)

Name Value
NAME RT RAD-FR-SUPERVISOR
DESCRIPTION 
This key allows the radiology file room supervisor to do the following:
          - cancel record requests
          - edit record requests
          - flag records as missing
          - remove the missing flag
          - approve or disapprove the finding of a record by
            a user other than a supervisor
                (the parameter allowing non-supervisors to find
                 records must be set in order to do this.)
