Name Value
NAME FEDERAL SUPPLY CLASSIFICATION
DESCRIPTION 
  This file contains codes used to classify types of items into categories
to be used for centralized reporting of procurement.  Examples of categories
are 'DRUGS & BIOLOGICALS', 'SURG. DRESS. MATERIALS', etc.
  This file also contains new entries known as product service codes
which are applicable to services only. These codes are not allowed to be 
used with items.
