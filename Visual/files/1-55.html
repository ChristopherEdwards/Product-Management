PHARMACY PATIENT (55)    FILE (1)

Name Value
NAME PHARMACY PATIENT
DESCRIPTION 
This file holds, for each patient, information that is typically of
interest only to the pharmacy.  This should be noted when merging patient
records and deleting the 'old' record from the patient file.  That process
does not delete the corresponding pharmacy patient file entry.
 
This file is also shared with inpatient pharmacy and promises to become a
very central file to the pharmacy.
