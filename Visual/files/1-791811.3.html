BATTERY (791811.3)    FILE (1)

Name Value
NAME BATTERY
DESCRIPTION 
This file contains batteries that may be issued for line items distributed
by the Denver Distribution Center through the Remote Order/
Entry System. Each entry in the Remote Inventory Product file may have
associated batteries from this file. This file is distributed with data.
Updates to the file will be made via patch. The file should not be edited
at the local facility.
