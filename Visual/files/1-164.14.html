NAME ICDO SITE GROUPS FOR REPORTING
DESCRIPTION 
This file, pointed to by the ICDO-SITES File (#164.08), facilitates the
production of the Annual Registry Summary Report.
 
This file comes with data, which overwrite the site's data.  It is not
recommended that sites modify this file.
