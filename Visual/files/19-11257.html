HL DISPLAY ONE LINK (TCP ONLY) (11257)    OPTION (19)

Name Value
NAME HL DISPLAY ONE LINK (TCP ONLY)
MENU TEXT Display One Link (TCP Only)
UPPERCASE MENU TEXT DISPLAY ONE LINK (TCP ONLY)
DESCRIPTION 
Asks the user to select a TCP link.  It then displays the link, a count
of the messages pending on its incoming and outgoing queues, along with
the first message on the queue.
