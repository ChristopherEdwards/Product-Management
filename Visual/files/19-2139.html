DG ELIG ID RESET (2139)    OPTION (19)

Name Value
NAME DG ELIG ID RESET
MENU TEXT Specific Eligibility ID Reset (All patients)
UPPERCASE MENU TEXT SPECIFIC ELIGIBILITY ID RESET
DESCRIPTION 
IRM personnel are prompted for an eligibility code.  Next, all patients that
have that eligibility code will have that ID reset.
 
The site would use this utility if the 'ID FORMAT' field in the 
ELIGIBILITY CODE file was changed.  This utility would allow the site to
update their database with the new values.
 
This option should be executed during non-peak hours.  When the job has
completed, a Mailman message will be sent to the user.

TYPE run routine
