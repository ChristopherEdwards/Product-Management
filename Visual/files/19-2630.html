NAME PRCSECP
MENU TEXT Copy a Transaction
UPPERCASE MENU TEXT COPY A TRANSACTION
DESCRIPTION 
This functions allows a Control Point to copy a pre-existing
transaction.  The information which is required for a new
request has been deleted and the user is prompted to edit
and complete this new request.
