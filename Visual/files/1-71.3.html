RAD/NUC MED COMMON PROCEDURE (71.3)    FILE (1)

Name Value
NAME RAD/NUC MED COMMON PROCEDURE
DESCRIPTION 
This file contains the procedures used in the display of common procedures
when requesting a procedure.  Up to 40 active common procedures are
allowed per imaging type.  Active common procedures have a sequence number
that determines their display order on the common procedure selection
screen.
