ADT TEMPLATE (43.7)    FILE (1)

Name Value
NAME ADT TEMPLATE
DESCRIPTION 
This file contains those options for which a facility may specify
a local template to utilize for output purposes rather than the
one distributed by the MAS developers.  The file is uneditable
using VA filemanager and may not be added to under any circumstances
by the facility.  To specify a local template utilize the appropriate
option of the MAS module.
