RA ORDERLOGLOC (5451)    OPTION (19)

Name Value
NAME RA ORDERLOGLOC
MENU TEXT Ward/Clinic Scheduled Request Log
UPPERCASE MENU TEXT WARD/CLINIC SCHEDULED REQUEST
DESCRIPTION 
This option allows the user to generate a list of scheduled requests by
ward/clinic. The list includes the following information:
patient name, social security number, procedure and imaging
location if the request was submitted to a specific imaging location.
CREATOR USER,ONE
TYPE run routine
DISPLAY OPTION? YES
ROUTINE RAORD8
