SPONSOR PERSON (355.82)    FILE (1)

Name Value
NAME SPONSOR PERSON
DESCRIPTION 
This file is pointed to by the SPONSOR (#355.8) and contains
all sponsors who are not patients.  This file contains the
non-patient sponsor's name, date of birth, and social security
number, all of which are retrieved from the PATIENT (#2) file
for sponsors who are patients.  Other pertinent sponsor
information is stored in the SPONSOR (#355.8) file.
