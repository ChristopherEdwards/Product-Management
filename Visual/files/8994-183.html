ORB FOLLOW-UP STRING (183)    REMOTE PROCEDURE (8994)

Name Value
NAME ORB FOLLOW-UP STRING
TAG GUI
ROUTINE ORB3FUP1
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
This function returns a string of follow-up data.  Content of the data 
varies by notification.
INPUT PARAMETER
  1. INPUT PARAMETER:   XQAID
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   32
    REQUIRED:   YES
    DESCRIPTION:   
    The identifier used by the Kernel Alert utility to uniquely identify an
alert/notification.  For OE/RR alerts the format is:
"OR,<dfn>,<notif ien>;<notif date.time>"
RETURN PARAMETER DESCRIPTION 
String of information required to complete the follow-up action.
