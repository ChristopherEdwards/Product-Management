IBARXM COPAY QUERY (9194)    OPTION (19)

Name Value
NAME IBARXM COPAY QUERY
MENU TEXT Query Medication Copay Billing Events
UPPERCASE MENU TEXT QUERY MEDICATION COPAY BILLING
DESCRIPTION 
This option will produce a report that will show what medication
copayments a patient has been billed for and not billed for.  This option
will allow you to query remote facilities to get all the information and
verify its accuracy.  If there is a discrepancy, the amounts will be
updated as well.
CREATOR USER,ONE
TYPE run routine
PACKAGE INTEGRATED BILLING
ROUTINE EN^IBARXMQ
