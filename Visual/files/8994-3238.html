DSIF CNSLTS UNHIDE CONSULT (3238)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIF CNSLTS UNHIDE CONSULT
TAG UNHIDE
ROUTINE DSIFCNS
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
This RPC will unhide a Consult hidden in the FBCS Consult Tracking 
(#19655) file.
INPUT PARAMETER
  1. INPUT PARAMETER:   DSIFIEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The IEN of the Consult you wish to unhide.
RETURN PARAMETER DESCRIPTION 
The RPC will return either 1^successful, or -1^unsuccessful. In the case 
of trying to unhide an already visible consult, it will return 0^message.
