NAME ORLP ADD SPECIALTY
X ACTION PRESENT YES
MENU TEXT Specialty (Add to list)
UPPERCASE MENU TEXT SPECIALTY (ADD TO LIST)
DESCRIPTION 
You can create a patient list by selecting 1 or more specialties, or
you can add specialties to a currently existing list using this option.
Once you have added a specialty, you can add or delete patients to
customize the list.
