NAME ENFAR3
MENU TEXT Check of Equipment Capitalization
UPPERCASE MENU TEXT CHECK OF EQUIPMENT CAPITALIZAT
DESCRIPTION 
This option generates a report of equipment items which appear to have
unusual combinations in the TOTAL ASSET VALUE and CAPITALIZED? fields.
Capitalized equipment is reported when it's Type Entry is blank.
Capitalized NX equipment is reported when it's CMR does not begin with one
of the standard two digit values, or it's Standard General Ledger is
blank. Equipment items with a disposition date are excluded from the
report. This report searches the entire equipment file and may take some
time to run.
 
These equipment items should be checked to ensure that the data in the
Equipment file is accurate.
