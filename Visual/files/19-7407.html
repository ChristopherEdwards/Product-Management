PSNPMIS PRINT (7407)    OPTION (19)

Name Value
NAME PSNPMIS PRINT
MENU TEXT Print a PMI Sheet
UPPERCASE MENU TEXT PRINT A PMI SHEET
DESCRIPTION 
This option will print a Patient Medication Information Sheet for
a drug selected from the local drug file.  Drugs in the local drug
file must be linked to entries in the National Drug File at the
site and entries in the National Drug File must have been linked
to a GCNSEQNO thus a PMIS by the National Drug File Manager.  A PMIS
can be printed in either English or Spanish.
CREATOR USER,ONE
TYPE run routine
ROUTINE PSNPPIP
