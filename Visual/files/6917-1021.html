LOAD BANK, ELECTRICAL. SIMPLEX, INC. A SET (BANK) OF ELECTRICAL
ELEMENTS THAT ARE PASSIVE (RESISTIVE, INDUCTIVE, CAPACITIVE) OR ACTIVE
(MOTORS, BATTERIES). THEY ARE USUALLY LOCATED ADJACENT TO EACH OTHER TO
FORM A CONSOLIDATED SET OF ANY OF THESE ELEMENTS OR ANY COMBINATION OF
THEM. A LOAD BANK IS USED TO REPLACEDISTRIBUTED OR DISPERSED LOAD
ELEMENTS CONTAINED IN A POWER DISTRIBUTION SYSTEM OR ELECTRICAL NETWORK
FOR THE PURPOSE OF PROVIDING A CONCENTRATED LOAD 
.
