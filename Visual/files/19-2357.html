IB OUTPUT INPTS WITHOUT INS (2357)    OPTION (19)

Name Value
NAME IB OUTPUT INPTS WITHOUT INS
MENU TEXT Inpatients w/Unknown or Expired Insurance
UPPERCASE MENU TEXT INPATIENTS W/UNKNOWN OR EXPIRE
DESCRIPTION 
This option will produce a list of either current inpatients or admissions
for a date range where the patient has either unknown insurance or the 
insurance is expired.

TYPE run routine
