Name Value
NAME PRCHPM CS ADD
MENU TEXT Create Code Sheet (LOG/GSA/DLA)
UPPERCASE MENU TEXT CREATE CODE SHEET (LOG/GSA/DLA
DESCRIPTION 
This option allows the user to create a LOG, GSA or DLA code sheet, using
the pre-defined edit templates that prompt for specific fields.  It does
not pre-load any data from Purchase Orders, Requisitions or Receiving
Reports.
