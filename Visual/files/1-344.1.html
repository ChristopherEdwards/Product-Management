AR DEPOSIT (344.1)    FILE (1)

Name Value
NAME AR DEPOSIT
DESCRIPTION 
This file stores information associated with SF-215 or Deposit Tickets.
Multiple Receipts can be open by different cashiers, but they are all
tied to a deposit ticket that tracks the date the money 
is deposited to bank, who deposited the money, total deposit, etc.
 
Per VHA Directive 10-93-142, this file definition should not be modified.
