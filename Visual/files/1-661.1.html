PROSTHETIC HCPCS (661.1)    FILE (1)

Name Value
NAME PROSTHETIC HCPCS
DESCRIPTION 
This is a standardized Prosthetic HCPCS file that contains HCPCS code
used in Prosthetics.  Sites are not allowed to update, add, edit or delete
entries from this file.  Only the SYNONYM and ITEM fields are editable,
and can be edited in the sites, all other fields will only be updated by
the Prosthetics Data Validation Group and update nationally will
take place through a patch.
