NAME HL7 NON-DHCP APPLICATION PARAMETER
DESCRIPTION 
This file contains parameters associated with non-DHCP applications
from whom the DHCP system can accept HL7 transmissions.
 
This is the main file that sites must edit before they can begin receiving
HL7 transmissions from another system.  This is done by using the
Non-DHCP Application Parameters Enter/Edit option on the site manager's
menu.
