LRLEDI (299)    SECURITY KEY (19.1)

Name Value
NAME LRLEDI
DESCRIPTIVE NAME LEDI Configuration
DESCRIPTION 
This security key is used to identify users that are charged with the
setup and configuration of LEDI software functionallity. In most
cases the LRLIASON key holder would qualify to also hold this key.
LEDI setup options are exported with this security key assigned.
