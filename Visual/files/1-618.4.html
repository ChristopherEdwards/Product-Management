MENTAL HEALTH INPT (618.4)    FILE (1)

Name Value
NAME MENTAL HEALTH INPT
DESCRIPTION 
This is the main data file for Mental Health Inpatient features.  It
contains current and past treatment episodes.  Team information, clinician
responsibility and treatment data are stored.  This is a flat file that
may be easily archived and restored.
