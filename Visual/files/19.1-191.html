IVM SYS (191)    SECURITY KEY (19.1)

Name Value
NAME IVM SYS
DESCRIPTIVE NAME IVM System Manager's Key
PERSON LOOKUP LOOKUP
KEEP AT TERMINATE NO
DESCRIPTION 
This key is used to to lock the IVM SYS MGR MENU option in the IVM
package.  The purpose of this key is to restrict access to this
menu from all users except IRM staff and IVM package coordinators.
