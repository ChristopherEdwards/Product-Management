PRCP IK DISASSEMBLE (2547)    OPTION (19)

Name Value
NAME PRCP IK DISASSEMBLE
MENU TEXT Disassemble Instrument Kit
UPPERCASE MENU TEXT DISASSEMBLE INSTRUMENT KIT
DESCRIPTION 
The Disassemble Instrument Kit option will break down the instrument
kit and return the individual disposable items back to stock.  When
an instrument kit is disassembled, the quantity on-hand for the
instrument kit will be decremented and the quantity on-hand for the
disposable items will be incremented.  The quantity on-hand for
reusable items will not change.
CREATOR USER,ONE
TYPE run routine
ROUTINE PRCPCDIK
