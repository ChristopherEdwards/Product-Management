QAOS DELETE (6885)    OPTION (19)

Name Value
NAME QAOS DELETE
MENU TEXT Delete Occurrence Screen Record
UPPERCASE MENU TEXT DELETE OCCURRENCE SCREEN RECOR
DESCRIPTION 
This option is used to mark an occurrence screen record as 'deleted'.
A deleted record remains in the file, but it is invisible to the user.
A record may be 'undeleted' by using the Reopen Closed/Deleted
Occurrence Screen Record option.
CREATOR POSTMASTER
TYPE run routine
ROUTINE EN4^QAOEDT2
