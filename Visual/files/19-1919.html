DG DEATH ENTRY (1919)    OPTION (19)

Name Value
NAME DG DEATH ENTRY
MENU TEXT Death Entry
UPPERCASE MENU TEXT DEATH ENTRY
LOCK DG DETAIL
DESCRIPTION 
This option allows the user to identify a patient as expired who
was NOT an inpatient at the time of death.  Patients who expire
while inpatients should be discharged from the medical center with
a discharge type of either "DEATH" or "DEATH WITH AUTOPSY".
