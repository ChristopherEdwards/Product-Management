PSNDRUG (3227)    OPTION (19)

Name Value
NAME PSNDRUG
MENU TEXT Rematch / Match Single Drugs
UPPERCASE MENU TEXT REMATCH / MATCH SINGLE DRUGS
DESCRIPTION 
User should run this option to match one drug at a time. User is
prompted for the drug name from local file to match with National File.
This option can also be used to rematch any drug that was incorrectly
matched or to match any new drug added to your local drug file.
CREATOR USER,ONE
TYPE run routine
ROUTINE PSNDRUG
