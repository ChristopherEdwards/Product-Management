DGMST STATISTICAL PRINT (8697)    OPTION (19)

Name Value
NAME DGMST STATISTICAL PRINT
MENU TEXT Print Statistical Report
UPPERCASE MENU TEXT PRINT STATISTICAL REPORT
DESCRIPTION 
Prints the MST Statistical Report.
This report contains both Outpatient and Inpatient statistics.
The number of encounters/movements related to MST and those not related
to MST are calculated.  Averages are also calculated for encounters related and not related to MST.
For inpatients, the Total Length of Stay for MST related episodes is
determined.  Also, a breakdown for each ICD-9 code assigned to a 
patient movement or encounter, by patient sex, is calculated.
CREATOR USER,ONE
TYPE run routine
DISPLAY OPTION? YES
ROUTINE EN^DGMSTR3
