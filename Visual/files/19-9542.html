EAS MERGE DUP MT/COPAY DEP (9542)    OPTION (19)

Name Value
NAME EAS MERGE DUP MT/COPAY DEP
MENU TEXT Merge Duplicate MT/Copay Dependents
UPPERCASE MENU TEXT MERGE DUPLICATE MT/COPAY DEPEN
LOCK DG MEANSTEST
DESCRIPTION 
This menu option will allow holders of the DG MEANSTEST security key the
ability to merge dependents considered to be duplicates in the PATIENT 
RELATION File.  Data deleted during the process is stored in an ^XTMP
global for 10 days.
