ENINV EDIT (1505)    OPTION (19)

Name Value
NAME ENINV EDIT
MENU TEXT Inventory Edit
UPPERCASE MENU TEXT INVENTORY EDIT
DESCRIPTION 
Edit the record of an existing piece of equipment. The .01 field (ENTRY
NUMBER) is assigned by the system when an item is first added to the
EQUIPMENT INV. File and may not be edited.
This option gives access to both Supply and Engineering fields.
