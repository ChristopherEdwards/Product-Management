ESP CRIME DATA (912.4)    FILE (1)

Name Value
NAME ESP CRIME DATA
DESCRIPTION 
The ESP CRIME DATA file is used to capture data for the Police &
Security Monthly Crime Report.  This file allows for a specific
date range selection so that the report can be run for alternate
time periods.  The crime categories included are the only ones
authorized by Police & Security managmeent (VACO) for tracking
criminal activity by the local sites.
The file data is AUTO-GENERATED.
