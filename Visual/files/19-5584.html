IVM UPLOAD INS (5584)    OPTION (19)

Name Value
NAME IVM UPLOAD INS
MENU TEXT Insurance Upload
UPPERCASE MENU TEXT INSURANCE UPLOAD
LOCK IVM UPLOAD
DESCRIPTION 
This option is used to upload patient insurance information which has
been transmitted to the facility from the IVM Center.  This information
was provided to IVM from an employer while contacting the employer to
verify income.
 
The user has the option to accept or reject this information.  Once
action has been taken on the data, a confirmation message, in HL7
format, is sent back to the IVM Center which contains the decision
that was made at the site level.  If the information was rejected,
a reason for not uploading the data (from file #301.91) is included
in the confirmation message.
