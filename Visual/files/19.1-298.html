SCENI MEANS TEST EDIT (298)    SECURITY KEY (19.1)

Name Value
NAME SCENI MEANS TEST EDIT
DESCRIPTIVE NAME IEMM Means Test edit key.
DESCRIPTION 
This key will enable a user to complete a means test when using Load Edit
only through the IEMM correction tool.  If a user is accessing Load Edit
from outside of the IEMM software this key will have no effect.
