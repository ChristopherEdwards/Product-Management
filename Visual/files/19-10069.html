PSJU NSS REPORT (10069)    OPTION (19)

Name Value
NAME PSJU NSS REPORT
MENU TEXT Non-Standard Schedule Report
UPPERCASE MENU TEXT NON-STANDARD SCHEDULE REPORT
DESCRIPTION 
This option allows the user to print a listing of all the active orders 
(during the timeframe selected) that contain problem schedules. These are
either invalid schedules or schedules with abbreviations that JCAHO
considers dangerous.
CREATOR USER,ONE
TYPE run routine
