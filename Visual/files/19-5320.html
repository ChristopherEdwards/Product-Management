RA WORKSHEETS (5320)    OPTION (19)

Name Value
NAME RA WORKSHEETS
MENU TEXT Print Worksheets
UPPERCASE MENU TEXT PRINT WORKSHEETS
DESCRIPTION 
This function allows the user to print any number of
worksheets needed.
 
These worksheets are designed to be used by an imaging department
that does not have enough CRT's to capture exam status in a
real-time mode.
 
These worksheets should accompany the exam requisition as it proceeds
through the department.  As the exam status changes, the
appropriate entries on the worksheet should be made.
 
The data captured on the sheets should then be entered in a batch
mode later in the day, when CRT's are available.
CREATOR USER,ONE
