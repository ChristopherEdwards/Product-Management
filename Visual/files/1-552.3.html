NAME CMOP RELEASE
DESCRIPTION 
This file is maintained at the Consolidated Mail Outpatient Pharmacy host
facility and contains release information returned from the automated
dispensing system. This file manages the background filer operations and 
returns data received from the automated system at the host facility  to
the originating medical center for filing. Data is purged from this file
daily. Data in this file is maintained by the Consolidated Mail Outpatient
Pharmacy software and must not be edited using VA FileMan.
