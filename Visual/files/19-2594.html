NAME PRCSAPP
MENU TEXT Approve Requests
UPPERCASE MENU TEXT APPROVE REQUESTS
LOCK PRCSCPO
DESCRIPTION 
This function is used by those persons who have access to Approve
orders before they are transferred to A&MM for action.  The user
can review the order, make any changes and optionally forward the
order to A&MM.  The user must have an Electronic Signature to
access this function.
