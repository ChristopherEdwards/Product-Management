MONTHLY ACCRUAL (440.7)    FILE (1)

Name Value
NAME MONTHLY ACCRUAL
DESCRIPTION 
During the compile performed in option Accrual (Monthly) [PRCB MONTHLY
ACCRUAL], entries are added to this file listing total unpaid and
unreconciled credit card order amounts by budget string "Fund/Beginning
Budget Fiscal Year/ Administration or Staff Office/Accounting
Classification Code/Cost Center/BOC" within month and station. This
file then serves as the source for generating the appropriate SV
documents to be sent to FMS.
