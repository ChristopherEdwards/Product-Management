RA ORDERHOLD (5446)    OPTION (19)

Name Value
NAME RA ORDERHOLD
MENU TEXT Hold a Request
UPPERCASE MENU TEXT HOLD A REQUEST
DESCRIPTION 
This option allows the user to place a 'requested' exam in the 'HOLD'
status. The user will be asked to select a reason from the 'Rad/Nuc Med
Reason' file (#75.2).  Only requests with a status of 'pending' or
'scheduled'  may be placed on 'HOLD'.
CREATOR USER,ONE
TYPE run routine
