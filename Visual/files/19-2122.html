DGPT MPCR INQUIRY (2122)    OPTION (19)

Name Value
NAME DGPT MPCR INQUIRY
MENU TEXT MPCR INQUIRY
UPPERCASE MENU TEXT MPCR INQUIRY
DESCRIPTION 
This option allows the user to view the MPCR information related
to a particular PTF record.  The information display is same
as the data shown on the 'MPCR' screen of the 'Load/Edit PTF Record'
option.
 
If the PTF record selected is not closed, then the PTF record is
automatically updated using the latest specialty and ward movement
information.
