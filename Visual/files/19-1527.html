ENBCNXRES (1527)    OPTION (19)

Name Value
NAME ENBCNXRES
MENU TEXT Restart Processing of Uploaded NX Inventory Data
UPPERCASE MENU TEXT RESTART PROCESSING OF UPLOADED
DESCRIPTION 
Used to resume processing of NX inventory that has been uploaded from
a portable bar code reader. User will need PROCESS ID and TIME STAMP
from failed process. If this information is unavailable, data upload
must be re-started from the beginning.
CREATOR POSTMASTER
TYPE run routine
ROUTINE RES^ENEQNX1
