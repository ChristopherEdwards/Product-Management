YSCL TEST LINK (8458)    OPTION (19)

Name Value
NAME YSCL TEST LINK
MENU TEXT Clozapine Test Link
UPPERCASE MENU TEXT CLOZAPINE TEST LINK
DESCRIPTION 
This option links the two tests required for Clozapine Roll-up. The first
is the White Blood Count.  The second is the Granulocyte (or Neutrophil)
percentage.  This option is intended for use by the LIM or their designee.
CREATOR USER,ONE
TYPE run routine
ROUTINE FLSET^YSCLTST2
