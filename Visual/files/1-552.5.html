CMOP COST STATS (552.5)    FILE (1)

Name Value
NAME CMOP COST STATS
DESCRIPTION 
This file is used by the Consolidated Mail Outpatient Pharmacy system to
perform drug cost analysis for the host facility.  Cost data is maintained 
by medical center/pharmacy division/drug and totals are stored daily and 
monthly. Data in this file is maintained by the Consolidated Mail
Outpatient Pharmacy software and must not be edited using VA FileMan.
