NAME ENPLMRV
MENU TEXT Validate Project Status Reports
UPPERCASE MENU TEXT VALIDATE PROJECT STATUS REPORT
DESCRIPTION 
This option validates selected construction projects. Only data which
pertains to the Progress Report (10-0051) is checked. Any validation
errors or warnings are listed in a detailed report.
CREATOR USER,ONE
TYPE action
PACKAGE ENGINEERING
