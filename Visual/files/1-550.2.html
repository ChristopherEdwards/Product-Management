CMOP TRANSMISSION (550.2)    FILE (1)

Name Value
NAME CMOP TRANSMISSION
DESCRIPTION 
This file is used by the Consolidated Mail Outpatient Pharmacy system
at the medical center to maintain summary information on all pharmacy
prescription data transmissions created and transmitted to the Consolidated 
Mail Outpatient Pharmacy host facility. Data in this file is maintained by 
the Consolidated Mail Outpatient Pharmacy software and must not be edited 
using VA FileMan.
