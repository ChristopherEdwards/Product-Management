SDAM APPT UPDATE PURGE (2270)    OPTION (19)

Name Value
NAME SDAM APPT UPDATE PURGE
MENU TEXT Purge Appointment Status Update Log File
UPPERCASE MENU TEXT PURGE APPOINTMENT STATUS UPDAT
DESCRIPTION 
This option allows the user to purge a date range from the Appointment
Status Update Log File.
 
Only dates later than the current plus one fiscal year will be purged.
 
At the end of the purge there will be a mailman message generated and 
sent to the user's IN mail box.
