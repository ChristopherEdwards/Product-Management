SDREVIEWDATE (2229)    OPTION (19)

Name Value
NAME SDREVIEWDATE
X ACTION PRESENT YES
MENU TEXT Enrollment Review Date Entry
UPPERCASE MENU TEXT ENROLLMENT REVIEW DATE ENTRY
DESCRIPTION 
THIS OPTION ALLOWS YOU TO ENTER A DATE WHEN AN NSC/OPT CLINIC ENROLLMENT
HAS BEEN REVIEWED.  THE EXISTENCE OF THIS DATE WILL PREVENT THE WARNING
FROM PRINTING ON THE APPOINTMENT LIST FOR A PERIOD OF 1 YEAR FROM THE
REVIEW DATE ENTERED.
CREATOR POSTMASTER
TYPE run routine
PACKAGE SCHEDULING
EXIT ACTION K DIC,X,Y,Z,%DT,DGA1,DGT,DGX,D0,D1,DQ
ROUTINE SDREV
