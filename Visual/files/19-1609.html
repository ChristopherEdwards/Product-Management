ENPLMAX (1609)    OPTION (19)

Name Value
NAME ENPLMAX
MENU TEXT Transmit Project Applications
UPPERCASE MENU TEXT TRANSMIT PROJECT APPLICATIONS
LOCK ENPLK003
DESCRIPTION 
This option transmits construction and lease project data to the Regional
Construction Database for selected projects. All data pertinent to a
project application is sent. Selected projects are automatically validated
prior to the actual transmission. The transmitter is a receipent of any
created mail messages that are sent to the Regional Contruction Database.
CREATOR POSTMASTER
TYPE run routine
