FEE BASIS MODIFIER TABLE INDEX (162.98)    FILE (1)

Name Value
NAME FEE BASIS MODIFIER TABLE INDEX
DESCRIPTION 
These entries contain percentage value to pay for a service.
After calculating the Medicare rate and the modifier is not
a separate entry in the fee schedule, this table stores by
calendar year a percentage multiplier which will adjust
the payment amount accordingly.
 
Per VHA Directive 10-93-142, this file definition should not be modified.
