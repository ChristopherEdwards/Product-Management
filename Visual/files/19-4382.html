DENT SERVICE TAPE (4382)    OPTION (19)

Name Value
NAME DENT SERVICE TAPE
MENU TEXT Generate Monthly Dental Service Tape
UPPERCASE MENU TEXT GENERATE MONTHLY DENTAL SERVIC
OUT OF ORDER MESSAGE DAS turned off
DESCRIPTION 
Generates a Dental Service tape from data stored in the ^UTILITY("DENTV"
global. Tape is in standard IBM format.  The tape is then mailed to
Austin, TX DPC.
CREATOR POSTMASTER
TYPE run routine
