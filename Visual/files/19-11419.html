XU PURGE PRINT QUEUE HFS FILES (11419)    OPTION (19)

Name Value
NAME XU PURGE PRINT QUEUE HFS FILES
MENU TEXT PURGE PRINT QUEUE HOST FILES
UPPERCASE MENU TEXT PURGE PRINT QUEUE HOST FILES
DESCRIPTION 
This option will purge old host files that were created for
printing.  It will also purge old entries in the PRINT QUEUE JOBS file
(#3.52). This option should be scheduled to run regularly.
