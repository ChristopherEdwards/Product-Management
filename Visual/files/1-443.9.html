IFCAP PENDING ARCHIVE (443.9)    FILE (1)

Name Value
NAME IFCAP PENDING ARCHIVE
DESCRIPTION 
This file is a local Archive/Purge file that documents what records are
pending archive.  Prior to initiating an archive, this file should be
empty.  This file is populated by the menu option Find Archivable IFCAP
Records [PRCG ARCHIVE FIND].
