ESP POLICE (225)    SECURITY KEY (19.1)

Name Value
NAME ESP POLICE
DESCRIPTIVE NAME ESP POLICE
PERSON LOOKUP LOOKUP
DESCRIPTION 
This Security Key functions as a screen that allows only currently
assigned VA Police Officers to be entered into the data records as
the Investigating Officer, Follow Up Officer, Issuing Officer, etc.
When an officer terminates his employment and this Security Key is
removed, his name can no longer be entered into any of these fields.
