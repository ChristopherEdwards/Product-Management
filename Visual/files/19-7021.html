ABSV DELETE SUSPENDED T.C. (7021)    OPTION (19)

Name Value
NAME ABSV DELETE SUSPENDED T.C.
MENU TEXT Delete Suspended Time Card
UPPERCASE MENU TEXT DELETE SUSPENDED TIME CARD
LOCK ABSV MGR
DESCRIPTION 
This option allows the user to delete from the system a time card which
had previously been marked as suspended.
