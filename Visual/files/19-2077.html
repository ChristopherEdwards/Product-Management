Name Value
NAME DG INPATIENT ROSTER
MENU TEXT Inpatient Roster
UPPERCASE MENU TEXT INPATIENT ROSTER
DESCRIPTION 
The Inpatient Roster report will provide a listing of patients 
sorted/selected by either Ward or Provider.  The report contains 
Name, Id, Age, Admission Date, LOS, Ward, Room-Bed, Primary 
Physician, Attending Physician, Treating Specialty, and Means Test 
Status.
It will also allow you to specify the number of copies to print.
