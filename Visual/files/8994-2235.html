Name Value
NAME DSIR SET LOCKED STATUS
TAG LOCKFOIA
ROUTINE DSIROI8
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
This RPC sets/resets the lock on the annual FOIA report add in numbers.

  1. INPUT PARAMETER:   EDDT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   99
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the internal number of the entry in 19620.3
  2. INPUT PARAMETER:   LSTAT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is the lock status 0 = Unlock, 1 = Lock
  3. INPUT PARAMETER:   IEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   3
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Internal Entry Number - Required
RETURN PARAMETER DESCRIPTION 
 IEN POINTER TO FILE 19620.3 PLUS THE END DATE AND DIVISION (IEN^EDDT^DIV)
 OR "-1^RECORD NOT FOUND" 
 OR "-2^IEN CANNOT BE BLANK"
