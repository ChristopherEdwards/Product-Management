VA-PRIMARY CARE TEAM (5)    REMINDER COMPUTED FINDINGS (811.4)

Name Value
NAME VA-PRIMARY CARE TEAM
ROUTINE PXRMPCIN
ENTRY POINT TEAM
PRINT NAME VA-PCMM Primary Care Team
DESCRIPTION 
This single occurrence computed finding returns the name of the
patient's PCMM primary care team as the value. If no team has been
assigned, the value will be null.
TYPE SINGLE
CLASS NATIONAL
