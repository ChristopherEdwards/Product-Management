SPNL REGISTRY DELETE (6988)    OPTION (19)

Name Value
NAME SPNL REGISTRY DELETE
MENU TEXT Delete Registry Record
UPPERCASE MENU TEXT DELETE REGISTRY RECORD
DESCRIPTION 
This option allows a user to delete a record from the SCD Registry file.
Once a record has been deleted it cannot be recovered!
