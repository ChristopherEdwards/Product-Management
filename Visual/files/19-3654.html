NAME PSJI DEVICE
MENU TEXT Change Report/Label Devices (IV)
UPPERCASE MENU TEXT CHANGE REPORT/LABEL DEVICES (I
DESCRIPTION 
This option simply allows the user to change output devices.
This option is 'Automatically' called when the user first signs
in to the IV PACKAGE.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
ENTRY ACTION D ^PSIVXU Q:$D(XQUIT) S PSIVZION=ION D ENCHK^PSIVSET,ENLD^PSIVSET,ENPD^PSIVSET S IOP=PSIVZION D ^%ZIS,SET^PSIVXU K IOP,PSIVZION,Z,%T D ENIVKV^PSGSETU
