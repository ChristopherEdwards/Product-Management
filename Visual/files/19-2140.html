DG ELIG RESET PATIENT (2140)    OPTION (19)

Name Value
NAME DG ELIG RESET PATIENT
MENU TEXT Reset All ID's for a Patient
UPPERCASE MENU TEXT RESET ALL ID'S FOR A PATIENT
DESCRIPTION 
The user is prompted for a patient.  Next, the corresponding ID'S for
all the patient's eligibilities are reset.
 
This utility can be used, if for some reason, a patient's ID's got
corrupted.
CREATOR POSTMASTER
TYPE run routine
PACKAGE REGISTRATION
ROUTINE 6^VADPT60
