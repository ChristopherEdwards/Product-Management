NAME XQHELPFIX
MENU TEXT Fix Help Frame File Pointers
UPPERCASE MENU TEXT FIX HELP FRAME FILE POINTERS
DESCRIPTION 
This option scans thru the help frame file for dangling pointers.
It deletes keywords which point to help frames that no longer exist.
