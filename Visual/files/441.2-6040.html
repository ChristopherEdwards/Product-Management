6040 (6040)    FEDERAL SUPPLY CLASSIFICATION (441.2)

Name Value
NUMBER 6040
ABBREVIATED DESCRIPTION FIBER OPTIC SENSORS
DESCRIPTION 
INCLUDED ARE FIBER OPTIC DEVICES WHICH DETECT THE PRESENCE OF A LIQUID
SOLID, AND/OR GAS AND PROVIDE AN ELECTRICAL OUTPUT TO MEASURE CHARACT-
ERISTICS SUCH AS MOVEMENT, DENSITY, SIZE, QUANTITY, AND/OR COLOR.
