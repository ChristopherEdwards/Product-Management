DENT SERVICE DELETE (4383)    OPTION (19)

Name Value
NAME DENT SERVICE DELETE
MENU TEXT Delete Dental Service (Temporary) Global
UPPERCASE MENU TEXT DELETE DENTAL SERVICE (TEMPORA
OUT OF ORDER MESSAGE DAS turned off
DESCRIPTION 
Delete the ^UTILITY("DENTV" global after the monthly Dental Service tape
has been produced.  A new global is then built as the dental application
coordinator releases Dental Service data during the month.  This option
must be run after a tape has been produced and before the dental
coordinator begins releasing data for the next month.
CREATOR POSTMASTER
HELP FRAME XQHELP-LINK
TYPE run routine
ROUTINE ATDEL^DENTE3
