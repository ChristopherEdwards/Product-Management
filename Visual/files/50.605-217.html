Name Value
CODE VT000
CLASSIFICATION VITAMINS
TYPE MAJOR
DESCRIPTION 
NOTE: The "vitamin B, other" subclassification (VT109) includes
       combinations containing only vitamin B complex.
       Combinations containing only vitamin D are included in
       subclassification (VT509). Subclassification (VT709)
       includes combinations of vitamin K only. The "vitamins,
       other" subclassification (VT809) includes any product in
       which a vitamin is found in combination with an ingredient
       which is neither a vitamin nor a mineral.
