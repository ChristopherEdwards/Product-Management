NAME VAQ - STATUS
DESCRIPTION 
This file defines all possible statuses of a PDX transaction.
 
 
Codes must be in the form XXXX-YYYYY where XXXX is the namespace of the
package (1-4 upper case characters) and YYYYY is 1-5 characters (upper or
lower case) of the package's choosing.  [Must pattern match 1.4U1"-"1.5A]
