Name Value
NAME PTF ARCHIVE/PURGE HISTORY FILE
DESCRIPTION 
This file contains a history of all Archive/Purge functions
performed on the PTF file(#45). The fields in this file should not be
updated through Fileman; they are updated by the PTF Archive/Purge
options throughout the Archive/Purge process.
