Name Value
NAME MH BATTERY USERS
DESCRIPTION 
This file contains a list of all interviews, surveys and tests associated 
with a entry in the MH BATTERIES file (#601.77).  The order of presentation of 
the interviews, surveys and tests within a battery and which battery belongs 
to which user in the NEW PERSON file (#200) is also specified.
