MAGV INACTIVATE INSTANCE FILE (3141)    REMOTE PROCEDURE (8994)

Name Value
NAME MAGV INACTIVATE INSTANCE FILE
TAG INFILE
ROUTINE MAGVRS31
RETURN VALUE TYPE ARRAY
AVAILABILITY RESTRICTED
DESCRIPTION 
This RPC inactivates an instance file entry.
INPUT PARAMETER
  1. INPUT PARAMETER:   FILEIEN
    PARAMETER TYPE:   LITERAL
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the IEN of the instance file entry to be inactivated.
  2. INPUT PARAMETER:   SOPIEN
    PARAMETER TYPE:   LITERAL
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is the IEN of the related SOP entry.
  3. INPUT PARAMETER:   OVERRIDE
    PARAMETER TYPE:   LITERAL
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    If set to 1 this flag prevents the checking of the parent procedure
reference prior to inactivation.
RETURN PARAMETER DESCRIPTION 
This parameter returns a 0 for success or a negative value and an error
message.
