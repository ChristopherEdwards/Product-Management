XOBW ASSOCIATE SORT (4436)    PROTOCOL (101)

Name Value
NAME XOBW ASSOCIATE SORT
ITEM TEXT Switch Sort
DESCRIPTION 
This protocol allows the user to switch the sorting ('key' or 'server')
of the list of web server lookup keys shown in the key manager utility.
 
The protocol act as a toggle. If list is currently sorted by 'key', it 
will switch to the sort to by 'server' and vice versa.
TYPE action
CREATOR USER,SEVENTEEN
PACKAGE VISTALINK
