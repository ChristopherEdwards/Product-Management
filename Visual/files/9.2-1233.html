NAME FSC U1001 VIEW CALLS
TEXT 
 
                              View Calls
 
The view screen allows multiple calls to be displayed.
 
The calls always begin with the line from the list and are separated by a
dashed line.
 
The calls can be edited by selecting calls again.  After editing you'll
return to the view screen.  It is sometimes easier to review calls for
specific information and then editing only those calls.
