PRCAC RC TRANSMISSION (8149)    OPTION (19)

Name Value
NAME PRCAC RC TRANSMISSION
MENU TEXT Transmission Log for RC
UPPERCASE MENU TEXT TRANSMISSION LOG FOR RC
DESCRIPTION 
This option displays the AR Transmissions sent via the Regional Counsel
Interface.  Users may View or Forward the mail messages in this
log.  They may also Delete or append Comments to existing AR RC
Transmission File entries.  Entries in this file will contian a
Purge Date set by the RC Parameters option.  Entries will be 
purge when the background option, Reconciliation Rollup Request
Background Task1 runs.  This file may grow to be very large.
