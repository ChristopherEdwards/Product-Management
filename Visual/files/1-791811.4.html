NAME REMOTE PROSTHETIC ITEM BRANDS
DESCRIPTION 
The Remote Prosthetic Item Brands file contains a list of Prosthetic
Brands available for items contained in the Remote Inventory Product File.
This file is distributed with data. Updates to the file will be made via
patch. The file should not be edited at the local facility.
