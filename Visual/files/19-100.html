XU-SPL-MAIL (100)    OPTION (19)

Name Value
NAME XU-SPL-MAIL
MENU TEXT Make spool document into a mail message
UPPERCASE MENU TEXT MAKE SPOOL DOCUMENT INTO A MAI
DESCRIPTION 
This option will take a spool document and post it as a mailman
message to the user's IN basket.  This doesn't move the data at all
but does decrease the number of lines charged to the user.
CREATOR USER,ONE
HELP FRAME XUDOC SPOOL USE5
TYPE run routine
ROUTINE MAIL^ZISPL
