GMRA PRINT A/AR NV (3451)    OPTION (19)

Name Value
NAME GMRA PRINT A/AR NV
MENU TEXT List by Location Not Verified Reactions
UPPERCASE MENU TEXT LIST BY LOCATION NOT VERIFIED
DESCRIPTION 
This option prints a list of patient reactions that have not been
verified. The output is sorted by location, patient and reaction. The IRM
can tasked this report to run on a daily basis by adding the GMRA TASK
A/AR NV option to the Option Scheduling file (#19.2).
CREATOR USER,ONE
TYPE run routine
ROUTINE EN1^GMRAPU
