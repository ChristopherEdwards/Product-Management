GMRC FEE PARAM (11420)    OPTION (19)

Name Value
NAME GMRC FEE PARAM
MENU TEXT Define Fee Services
UPPERCASE MENU TEXT DEFINE FEE SERVICES
DESCRIPTION 
This option allows review, adding, and removal of services listed in the 
GMRC FEE SERVICES system parameter. Services listed in this parameter are 
defined as Non-VA Care (Fee Basis) services.
