PRCSESCP (2632)    OPTION (19)

Name Value
NAME PRCSESCP
MENU TEXT Correct Sub-Control Point Amounts
UPPERCASE MENU TEXT CORRECT SUB-CONTROL POINT AMOU
DESCRIPTION 
This option allows you to edit/adjust the sub-control point 
dollar distribution for all except the 'ceiling' transactions.
The total sub-control point dollar distribution cannot exceed 
the original transaction amount.
CREATOR USER,ONE
TYPE run routine
ROUTINE PRCSED1
