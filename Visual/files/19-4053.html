PSJV PD/DD PRINT (4053)    OPTION (19)

Name Value
NAME PSJV PD/DD PRINT
MENU TEXT Primary Drug/Dispense Drug Report
UPPERCASE MENU TEXT PRIMARY DRUG/DISPENSE DRUG REP
DESCRIPTION 
  Allows the user to print primary drugs and the dispense drugs tied to
them.  The report can be sorted by primary drug or dispense drug.  The
report can also show exceptions - dispense drugs not yet tied to a primary
drug.
CREATOR POSTMASTER
TYPE run routine
PACKAGE PATIENT DATA EXCHANGE
