Name Value
NAME PRCSCPOQR
MENU TEXT Quarterly Report
UPPERCASE MENU TEXT QUARTERLY REPORT
DESCRIPTION 
This report displays all transactions for a quarter's activity for
a Control Point.  Data presented includes transaction number and
type, vendor, committed, obligated and actual costs and balances.
Totals are presented at the bottom of the report.
