NAME RA SYSEXROOM
MENU TEXT Camera/Equip/Rm Entry/Edit
UPPERCASE MENU TEXT CAMERA/EQUIP/RM ENTRY/EDIT
DESCRIPTION 
The 'Camera/Equip/Rm Enter/Edit' function allows the coordinator to enter
new cameras/equipment/rooms into the system or edit the descriptions and
locations of existing cameras/equip/rms.  These cameras/equip/rms can then
be associated with a location.  Also, unlike the relationship between
divisions and locations, cameras/equip/rms can be associated with more
than one location.
 
The entries in this file are used by the technologist when he/she
indicates with which camera/equip/rm the exam was performed.  Only
cameras/equipment/rooms associated with the location where the exam was
performed are valid choices.
 
Use the 'Location Parameter Set-Up' option to associate a camera/equip/rm
with a location.
