PRCHPC ITEM INACTIVATE (2695)    OPTION (19)

Name Value
NAME PRCHPC ITEM INACTIVATE
MENU TEXT Inactivate Item
UPPERCASE MENU TEXT INACTIVATE ITEM
DESCRIPTION 
This option allows A&MM to restrict access to an item in the Item Master
file. A&MM may specify a substitute item to be ordered.
