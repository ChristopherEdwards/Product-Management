XUTM PROBLEM DEVICES (8868)    OPTION (19)

Name Value
NAME XUTM PROBLEM DEVICES
MENU TEXT Problem Device report.
UPPERCASE MENU TEXT PROBLEM DEVICE REPORT.
DESCRIPTION 
This option will run the XUTMKA routine to produce a list of devices that
TASKMAN is having problems opening.  At the end of the report the problem
device list is cleared.
CREATOR USER,ONE
TYPE run routine
PACKAGE KERNEL
DELEGABLE YES
ROUTINE EN1^XUTMKA
