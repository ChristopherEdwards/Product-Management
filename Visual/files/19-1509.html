ENWOSTATUS-WARD (1509)    OPTION (19)

Name Value
NAME ENWOSTATUS-WARD
MENU TEXT Electronic Work Order Status Check
UPPERCASE MENU TEXT ELECTRONIC WORK ORDER STATUS C
DESCRIPTION 
Enables users outside of Engineering Service to view the status of work
orders, including those entered as Electronic Work Order Requests. This
option does not allow any changes to be made to the work order record.
CREATOR POSTMASTER
TYPE run routine
ROUTINE WRDCK^ENWARD
