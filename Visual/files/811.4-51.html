VA-AGENT ORANGE EXPOSURE (51)    REMINDER COMPUTED FINDINGS (811.4)

Name Value
NAME VA-AGENT ORANGE EXPOSURE
ROUTINE PXRMMSER
ENTRY POINT AORANGE
PRINT NAME VA-Agent Orange Exposure
DESCRIPTION 
This computed finding will be true if the patient has an agent
orange exposure registration date in the date range specified by
Beginning Date/Time and Ending Date/Time.

Subscripts that can be used in a Condition are:
 "LOCATION"

The default value is "LOCATION".
TYPE MULTIPLE
CLASS NATIONAL
