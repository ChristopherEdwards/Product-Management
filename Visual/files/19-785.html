NAME LRENTER
MENU TEXT Enter/verify/modify data (manual)
UPPERCASE MENU TEXT ENTER/VERIFY/MODIFY DATA (MANU
DESCRIPTION 
Data may be entered and validated (if LRVERIFY key is owned).  Data
which has been previously validated may be corrected.
Entering the "*" character for any entry will substitute the word
"canceled" as the result.
