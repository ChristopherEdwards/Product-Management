XPD VERIFY INTEGRITY (269)    OPTION (19)

Name Value
NAME XPD VERIFY INTEGRITY
MENU TEXT Verify Package Integrity
UPPERCASE MENU TEXT VERIFY PACKAGE INTEGRITY
DESCRIPTION 
This option will check the integrity of a package on your system.
It will compare the current checksums with the checksums sent with
the Distribution.
CREATOR USER,ONE
TYPE run routine
ROUTINE EN2^XPDDCS
