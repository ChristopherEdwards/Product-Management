RA AMISDUMP (5399)    OPTION (19)

Name Value
NAME RA AMISDUMP
MENU TEXT AMIS Code Dump by Patient
UPPERCASE MENU TEXT AMIS CODE DUMP BY PATIENT
DESCRIPTION 
This option allows the supervisor to list all patients
who have had an examination that is associated with a specific Amis
code within a specified period of time.
 
This option was created to help the supervisor investigate
discrepancies that may seem to be appearing in the Amis report.
CREATOR USER,ONE
TYPE run routine
DISPLAY OPTION? YES
ROUTINE RALIST
