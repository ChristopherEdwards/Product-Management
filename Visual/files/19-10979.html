PSA RET DRG BAT COMPLETE LIST (10979)    OPTION (19)

Name Value
NAME PSA RET DRG BAT COMPLETE LIST
MENU TEXT Batch Complete List
UPPERCASE MENU TEXT BATCH COMPLETE LIST
DESCRIPTION 
This option allows users to access and update batches with any status. 
Users can select batches with the following status:
        AP -  AWAITING PICKUP
        PU -  PICKED UP
        CO -  COMPLETED
        CA -  CANCELLED
       ALL -  ALL
CREATOR USER,SEVENTEEN
TYPE run routine
PACKAGE DRUG ACCOUNTABILITY
ROUTINE CBAT^PSARDCBL
