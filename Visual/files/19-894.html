LRUPT (894)    OPTION (19)

Name Value
NAME LRUPT
MENU TEXT Show list of accessions for a patient
UPPERCASE MENU TEXT SHOW LIST OF ACCESSIONS FOR A
DESCRIPTION 
If you need to find all the accession numbers (in one accession area)
for one patient, you may do so with this option.  The information is 
displayed on the screen only.  You cannot print the list with this option.
