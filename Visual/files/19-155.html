XQKEYALTODEL (155)    OPTION (19)

Name Value
NAME XQKEYALTODEL
MENU TEXT Change user's allocated keys to delegated keys
UPPERCASE MENU TEXT CHANGE USER'S ALLOCATED KEYS T
DESCRIPTION 
 
This option will prompt for a user and use %XY^%RCR to make all keys the user
holds delegated keys so that the user can give them to others.
