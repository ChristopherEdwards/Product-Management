VA-AGE (24)    REMINDER COMPUTED FINDINGS (811.4)

Name Value
NAME VA-AGE
ROUTINE PXRMPDEM
ENTRY POINT AGE
PRINT NAME VA-Patient Age
DESCRIPTION 
This is a single occurrence computed finding that returns the
patient's age in years. It can be used as the value in the Condition.
For example: I V>50.

If the patient is deceased the age will be their age on the date of
death and V("DECEASED")=1.
TYPE SINGLE
CLASS NATIONAL
