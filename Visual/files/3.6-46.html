PRC_IFCAP_CHANGE (46)    BULLETIN (3.6)

Name Value
NAME PRC_IFCAP_CHANGE
SUBJECT Notification from IFCAP of a CHANGE through an amendment
DESCRIPTION 
This bulletin will be used to notify users that there has been
a change to a P.O. to allow updating the inventory point.
   
This bulletin is called from 'PRCHAMYD' routine.
MESSAGE 
Purchase Order |1| for inventory point |2| has been
changed as follows:
PARAMETER
  1. PARAMETER:   1
    DESCRIPTION:   
    PURCHASE ORDER NUMBER.
  2. PARAMETER:   2
    DESCRIPTION:   
    INVENTORY POINT NAME FROM .01 FIELD OF FILE 445.
