DG ELIG RESET ALL (2141)    OPTION (19)

Name Value
NAME DG ELIG RESET ALL
MENU TEXT Reset ALL ID's for ALL Patients
UPPERCASE MENU TEXT RESET ALL ID'S FOR ALL PATIENT
DESCRIPTION 
This utility resets all ID's corresponding to each patient's
eligibilities.  This is done for ALL patients.
 
This option should be executed during non-peak hours.  When the job has
completed, a Mailman message will be sent to the user.
CREATOR POSTMASTER
TYPE run routine
