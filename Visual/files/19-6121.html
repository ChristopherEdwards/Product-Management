MENU TEXT Report of Cases Without Specimens
UPPERCASE MENU TEXT REPORT OF CASES WITHOUT SPECIM
DESCRIPTION 
This report lists all completed cases in which there were no specimens
taken from the operative site.  It can be printed for an individual
surgical specialty if desired.
