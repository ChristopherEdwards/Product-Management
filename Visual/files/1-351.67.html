TRANSFER PRICING INPT PROSTHETIC ITEMS (351.67)    FILE (1)

Name Value
NAME TRANSFER PRICING INPT PROSTHETIC ITEMS
DESCRIPTION 
This file stores the prosthetic devices that should be automatically
billed for inpatient devices issued.  Unless a device is in this file, it
will only be billed for outpatient services (automatically).
 
Per VHA Directive 10-93-142, this file definition should not be modified.
