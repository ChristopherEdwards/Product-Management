PRCHPC PO CANCEL (2692)    OPTION (19)

Name Value
NAME PRCHPC PO CANCEL
MENU TEXT Cancel an Unobligated Purchase Order
UPPERCASE MENU TEXT CANCEL AN UNOBLIGATED PURCHASE
DESCRIPTION 
This option allows the user to cancel an unobligated Purchase Order. The
total amount of the P.O. will be zero and all FPDS information will be
deleted.
