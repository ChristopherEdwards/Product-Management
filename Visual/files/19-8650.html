NAME PSO PNDPRI
MENU TEXT Delete Printed Rx's from Suspense
UPPERCASE MENU TEXT DELETE PRINTED RX'S FROM SUSPE
DESCRIPTION 
This option is used to delete printed Rx's from Suspense. Deletion can be
done for one Rx, all for a patient, all for a Date Range, or by batches.
