RA EXAMSTATUS (5346)    OPTION (19)

Name Value
NAME RA EXAMSTATUS
MENU TEXT Examination Status Entry/Edit
UPPERCASE MENU TEXT EXAMINATION STATUS ENTRY/EDIT
DESCRIPTION 
  This function allows the coordinator the option to perform
  the following:
 
      1) Add new examination statuses
      2) Edit existing examination statuses
      3) Modify requirement criteria for each status
      4) Modify which prompts appear when using the 'Status Tracking
         of Exams' function
      5) Indicate by status, which exams in each status to use
         in the compilations of the various workload reports
CREATOR USER,ONE
TYPE run routine
DISPLAY OPTION? YES
ROUTINE 1^RAMAIN1
