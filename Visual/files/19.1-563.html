PSDRPH (563)    SECURITY KEY (19.1)

Name Value
NAME PSDRPH
DESCRIPTIVE NAME Pharmacy Prescribing for CS
DESCRIPTION 
This key authorizes pharmacists to verify and dispense controlled substance
prescription(s). The PSDRPH security key should be given to registered 
Pharmacists working on controlled substances to honor Drug Enforcement 
Administration regulations, and should not be given to non-pharmacists 
except in cases where the package coordinator (ADPAC) is not a registered 
pharmacist.
