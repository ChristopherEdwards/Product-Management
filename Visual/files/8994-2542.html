DSIV GETS (2542)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIV GETS
TAG GETS
ROUTINE DSIVIC
RETURN VALUE TYPE ARRAY
DESCRIPTION 
Returns details of INSURANCE BUFFER file entry.
INPUT PARAMETER
  1. INPUT PARAMETER:   IENS
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    INSURANCE BUFFER file internal entry number or IENS.
  2. INPUT PARAMETER:   FIELDS
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   240
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Field list (optional).  Default is all top-level fields.
  3. INPUT PARAMETER:   FLAGS
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    Flags (optional).  Default is external format, omit empty fields.
RETURN PARAMETER DESCRIPTION 
Array of entry details in the format:
IEN^FIELD NUMBER^VALUE^FORMAT
