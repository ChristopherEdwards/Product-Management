DG BED CONTROL MOVEMENT EDIT (1962)    OPTION (19)

Name Value
NAME DG BED CONTROL MOVEMENT EDIT
MENU TEXT Edit Bed Control Movement Types
UPPERCASE MENU TEXT EDIT BED CONTROL MOVEMENT TYPE
DESCRIPTION 
This option allows you to edit all local admission, transfer, discharge,
check-in lodger, check-out lodger, or treating specialty movement types.
After editing all desired movement types, you will be asked if you want
to display standard default responses.  If you say yes, all MAS-distributed
values will be stored for all movement types.
