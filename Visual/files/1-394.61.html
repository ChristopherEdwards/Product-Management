VAQ - TRANSACTION (394.61)    FILE (1)

Name Value
NAME VAQ - TRANSACTION
DESCRIPTION 
This file holds information describing each PDX transaction.  A PDX
transaction is created when one of the following events occur.
 
  (1) A PDX Request is defined and sent
  (2) A PDX Request is received from a remote facility
  (3) An Unsolicited PDX is defined and sent
  (4) An Unsolicited PDX is received from a remote facility
 
Per VHA Directive 10-93-142, this file definition should not be modified.
