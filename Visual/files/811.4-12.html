VA-ETHNICITY (12)    REMINDER COMPUTED FINDINGS (811.4)

Name Value
NAME VA-ETHNICITY
ROUTINE PXRMPDEM
ENTRY POINT ETHNY
PRINT NAME VA-Patient's Ethnicity
DESCRIPTION 
This multiple occurrence computed finding will return the patient's
ethnicity values from the Patient file. If none are recorded then the
computed finding will be false. This computed finding does not support date
reversal.

The value of the computed finding that can be used in the Condition is
the patient's ethnicity.
TYPE MULTIPLE
CLASS NATIONAL
