PSO CHANGE PRINTER (8664)    OPTION (19)

Name Value
NAME PSO CHANGE PRINTER
X ACTION PRESENT YES
MENU TEXT Change Label Printer
UPPERCASE MENU TEXT CHANGE LABEL PRINTER
DESCRIPTION 
This option will be used to allow users to change their label printer
without logging completely out of the OP package.
CREATOR USER,ONE
TYPE action
PACKAGE OUTPATIENT PHARMACY
E ACTION PRESENT YES
EXIT ACTION K PSOCLBL,%ZIS("B")
ENTRY ACTION S PSOCLBL=1 D @$S('$D(PSOPAR):"^PSOLSET",1:"PLBL^PSOLSET") K PSOCLBL
