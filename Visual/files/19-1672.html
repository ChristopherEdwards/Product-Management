ENFA SEND SINGLE (1672)    OPTION (19)

Name Value
NAME ENFA SEND SINGLE
MENU TEXT Send a Single FA Document
UPPERCASE MENU TEXT SEND A SINGLE FA DOCUMENT
LOCK ENFACS
DESCRIPTION 
This option will send a FA Document (code sheet) to the Fixed Assets
package (FAP) in Austin for a selected equipment entry. Appropriate data
is extracted from the Equipment file and used to populate the code sheet.
FA Documents will not be created for equipment entries already established
in FAP.
CREATOR POSTMASTER
TYPE action
PACKAGE ENGINEERING
E ACTION PRESENT YES
ENTRY ACTION S ENBAT("XMT")=1 D EQUIP^ENFAXMTM K ENBAT("XMT")
