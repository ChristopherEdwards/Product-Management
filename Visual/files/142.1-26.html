PROGRESS NOTES BRIEF (26)    HEALTH SUMMARY COMPONENT (142.1)

Name Value
NAME PROGRESS NOTES BRIEF
PRINT ROUTINE MAIN;GMTSPNB
TIME LIMITS APPLICABLE yes
ABBREVIATION BPN
DESCRIPTION 
This component contains limited information from progress notes entered 
using the Text Integration Utility (TIU) or the Mental Health packages.
Time and maximum occurrence limits apply.  Data presented includes:  
Progress note date/time, title, author and last correction date/time.
    
Only those notes which have been signed with an electronic signature
or electronically marked signed on chart will be listed.
MAXIMUM OCCURRENCES APPLICABLE yes
DEFAULT HEADER NAME Brief Progress Notes
