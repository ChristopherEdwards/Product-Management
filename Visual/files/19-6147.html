SROKEY REMOVE (6147)    OPTION (19)

Name Value
NAME SROKEY REMOVE
X ACTION PRESENT YES
MENU TEXT Remove Restrictions on 'Person' Fields
UPPERCASE MENU TEXT REMOVE RESTRICTIONS ON 'PERSON
DESCRIPTION 
This option is used to delete keys that have been assigned to a field
to restrict entries.  Using this option you may delete one specific
key, or all keys.
