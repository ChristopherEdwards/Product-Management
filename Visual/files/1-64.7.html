CUMULATIVE (64.7)    FILE (1)

Name Value
NAME CUMULATIVE
DESCRIPTION 
 
This file stores temporary pages of the cumulative report.
 
The data in this file is maintained by the cumulative routines.  
Because patient care can be effected if the cumulative report has 
incomplete lab results, you should not edit this file or data contained
within it.  Contact your ISC for technical support if problems arise.
