LRRMM TASK MAIL LAB REPORTS (7663)    OPTION (19)

Name Value
NAME LRRMM TASK MAIL LAB REPORTS
MENU TEXT Create and Mail Lab Reports
UPPERCASE MENU TEXT CREATE AND MAIL LAB REPORTS
DESCRIPTION 
This option produces the interim reports for lab work from other hospitals
and sends it to a spool file. The reports are then taken from the spool file
and loaded into a mail message. Network mail transmits the reports to the 
referring hospital.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
ENTRY ACTION S LRRVDT="T-1" D ALLOC^LRRMM
SCHEDULING RECOMMENDED YES
