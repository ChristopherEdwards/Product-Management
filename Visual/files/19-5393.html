RA RPTPAT (5393)    OPTION (19)

Name Value
NAME RA RPTPAT
X ACTION PRESENT YES
MENU TEXT Select Report to Print by Patient
UPPERCASE MENU TEXT SELECT REPORT TO PRINT BY PATI
DESCRIPTION 
This function allows the user to print reports by patient name.
If the report has not been filed, then a warning message is displayed.
   
This function is often used to print a report that is lost or when a
duplicate is needed.
CREATOR USER,ONE
TYPE run routine
E ACTION PRESENT YES
EXIT ACTION K RANOSCRN,RAOPT("RPTPAT")
ENTRY ACTION S RANOSCRN="",RAOPT("RPTPAT")=""
ROUTINE 4^RART2
