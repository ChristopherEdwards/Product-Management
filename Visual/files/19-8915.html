IBCN ID DUP INSURANCE ENTRIES (8915)    OPTION (19)

Name Value
NAME IBCN ID DUP INSURANCE ENTRIES
MENU TEXT Generate Insurance Company Listings
UPPERCASE MENU TEXT GENERATE INSURANCE COMPANY LIS
DESCRIPTION 
This option allows users to search the Insurance Company (36#)
File to identify duplicate entries.  The file may be searched by
Insurance Company name, address, city, or state.  A listing
of Insurance Company names, address and phone number that meet
the search criteria will display.
