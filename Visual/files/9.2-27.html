NAME XQDIAGRAM
HEADER USING THE 'Diagram Menu' OPTION
TEXT 
This option will produce a 'tree-structure' diagram of the menu system.
You will be prompted for a USER or OPTION name.  If you enter a USER
name, that person's primary menu option will be used as the top of the
tree, otherwise the OPTION specified will be used.
