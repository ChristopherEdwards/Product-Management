SROP REQ (6040)    OPTION (19)

Name Value
NAME SROP REQ
X ACTION PRESENT YES
MENU TEXT Operation Requests for a Day
UPPERCASE MENU TEXT OPERATION REQUESTS FOR A DAY
LOCK SROREQ
DESCRIPTION 
This option will generate a list of requested operations for
a specific date.  There are two formats (long and short form)
in which the requests may be printed.
CREATOR POSTMASTER
TYPE run routine
E ACTION PRESENT YES
EXIT ACTION D EXIT^SROVAR
ENTRY ACTION D ^SROVAR
XQUIT EXECUTABLE Q
ROUTINE SROPREQ
