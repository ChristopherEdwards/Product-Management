FSC CLOSE CALL (10349)    OPTION (19)

Name Value
NAME FSC CLOSE CALL
MENU TEXT Close Call
UPPERCASE MENU TEXT CLOSE CALL
DESCRIPTION 
This option displays and closes a NOIS call using the Edit
Screen.  Only those calls that are not closed or cancelled are selectable.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
ENTRY ACTION D OPT^FSCOP("CLOSE")
INDEPENDENTLY INVOCABLE YES
