LEX CSV CPT QUERY (10940)    OPTION (19)

Name Value
NAME LEX CSV CPT QUERY
X ACTION PRESENT YES
MENU TEXT CPT/HCPCS Procedure Code Set Query
UPPERCASE MENU TEXT CPT/HCPCS PROCEDURE CODE SET Q
DESCRIPTION 
This option displays a single versioned entry from the CPT/HCPCS
file #81 based on a date provided by the user.  The date may
be a future date.
CREATOR USER,SEVENTEEN
TYPE run routine
E ACTION PRESENT YES
EXIT ACTION K ^TMP("LEXQCPO",$J)
ENTRY ACTION K ^TMP("LEXQCPO",$J)
ROUTINE EN^LEXQCP
