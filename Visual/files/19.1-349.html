PSO COPAY (349)    SECURITY KEY (19.1)

Name Value
NAME PSO COPAY
DESCRIPTIVE NAME Outpatient Pharmacy Copay Key
KEEP AT TERMINATE YES
DESCRIPTION 
This Key is given to Pharmacy users that may be responsible for resetting
the Copay status of prescriptions. Holders of this Key will receive mail
messages indicating that a prescription may need further action regarding
Copay status.
