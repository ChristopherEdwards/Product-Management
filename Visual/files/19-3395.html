PRCH OVERAGE (3395)    OPTION (19)

Name Value
NAME PRCH OVERAGE
MENU TEXT Overage Report
UPPERCASE MENU TEXT OVERAGE REPORT
DESCRIPTION 
Prints a report of purchase orders with overages, sorted by purchase
order date.  The report can be called for all stations or limited 
to one station.  If the station chosen has substations the report 
can be limited to one substation.
CREATOR USER,ONE
