MH QUESTIONS (601.72)    FILE (1)

Name Value
NAME MH QUESTIONS
DESCRIPTION 
This file contains a listing of all questions for all Psychological Interviews,
Surveys and Tests.  This allows usage of questions by multiple instruments. 
Each question can also be linked to an entry in the MH INTRODUCTIONS file 
(#601.73), the MH CHOICES file (#601.75), the MH CHOICETYPES file 
(#601.751) and the Hint field (#8) in the MH QUESTIONS file (#601.72).
