2210 Shipment of Bodies (2210)    BUDGET OBJECT CODE (420.2)

Name Value
NAME 2210 Shipment of Bodies
DESCRIPTION 
INCLUDES TRANSPORTATION OF BODIES OF BENFICIARIES WHEN PAID SEPARATELY, AND RAILROAD FARES WHEN SHIPMENT IS HANDLED BY CARRIER AS BAGGAGE. (TRANSPORTATION INCLUDED WITH OTHER BURIAL ITEMS SHOULD BE CHARGED TO SUB ACCOUNT 2597.)
