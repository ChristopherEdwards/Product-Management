RACE CODE FOR ONCOLOGY (164.46)    FILE (1)

Name Value
NAME RACE CODE FOR ONCOLOGY
DESCRIPTION 
This file contains codes used to record the patient's race in the RACE
Field (#8) of the ONCOLOGY PATIENT File (#160).  The codes are derived
from the ACOS Data Acquisition Manual.
 
This file comes with data, which overwrite the site's data.  It is not
recommended that sites modify this file.
