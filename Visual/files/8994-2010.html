ORWTIU CHKTXT (2010)    REMOTE PROCEDURE (8994)

Name Value
NAME ORWTIU CHKTXT
TAG CHKTXT
ROUTINE ORWTIU
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
Check for existence of text in TIU(8925,TIUDA, either in "TEXT" or "TEMP" 
nodes, before allowing signature.
INPUT PARAMETER
  1. INPUT PARAMETER:   ORTIUDA
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Pointer to file 8925.
RETURN PARAMETER DESCRIPTION 
1 = text present
0 = no text present
