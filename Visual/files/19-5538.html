Name Value
NAME ECTC CONS PROJ
MENU TEXT Construction Project Profiles
UPPERCASE MENU TEXT CONSTRUCTION PROJECT PROFILES
DESCRIPTION 
This option displays brief data about construction projects.  The
user may choose the 'sort by' field.  The following fields are
shown for each project: project number, title, category, funding
year, approval level, amount authorized for construction at the time 
the project is approved, status code, and construction method.
