XWB LOG VIEW (10238)    OPTION (19)

Name Value
NAME XWB LOG VIEW
MENU TEXT View XWB Log
UPPERCASE MENU TEXT VIEW XWB LOG
DESCRIPTION 
This option allow the user to view the debug log files that the Broker 
can set.  The parameter XWBDEBUG must be set for log files to be recorded.
CREATOR USER,ONE
TYPE run routine
PACKAGE RPC BROKER
DELEGABLE YES
ROUTINE VIEW^XWBDLOG
