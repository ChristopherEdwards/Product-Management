ECDEAL2 (11107)    OPTION (19)

Name Value
NAME ECDEAL2
MENU TEXT Remove Terminated Users Access to DSS Units
UPPERCASE MENU TEXT REMOVE TERMINATED USERS ACCESS
DESCRIPTION 
This option is run as a nightly task and used to deallocate DSS units 
from terminated users.  A listing of DSS UNITS and their terminated users 
will be delivered to members of the ECXMGR mailgroup on a nightly basis.
CREATOR USER,SEVENTEEN
TYPE run routine
PACKAGE EVENT CAPTURE
ROUTINE USER^ECDEAL2
