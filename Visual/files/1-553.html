CMOP INTERFACE (553)    FILE (1)

Name Value
NAME CMOP INTERFACE
DESCRIPTION 
This file is used by the Consolidated Mail Outpatient Pharmacy system and
contains information necessary for the operations and management of the
Consolidated Mail Outpatient Pharmacy interface to the non-DHCP automated
dispensing system. Data in this file is maintained by the Consolidated Mail 
Outpatient Pharmacy software and must not be edited using VA FileMan.
