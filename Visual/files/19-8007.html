NAME SCENI IEMM SUMMARY REPORT
MENU TEXT Summary Report - IEMM
UPPERCASE MENU TEXT SUMMARY REPORT - IEMM
DESCRIPTION 
This report prints a summary showing number of outpatient encounters,
and the number of those encounters that are incomplete, and the
percentage.  If there is one error that is occurring more than any 
other, it is displayed as well.  Formatted for 80 col. output.
