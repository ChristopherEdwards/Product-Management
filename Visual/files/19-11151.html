Name Value
NAME PXRM TAXONOMY CODE SEARCH
MENU TEXT Search Taxonomies for Codes
UPPERCASE MENU TEXT SEARCH TAXONOMIES FOR CODES
DESCRIPTION 
This option will allow a user to input a code (ICD
Operation/Procedure, ICD Diagnosis, or CPT) and then it will search
all taxonomies and list those where it is being used.
