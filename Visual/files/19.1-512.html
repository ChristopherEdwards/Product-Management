TIU MED MANUAL OVERRIDE (512)    SECURITY KEY (19.1)

Name Value
NAME TIU MED MANUAL OVERRIDE
DESCRIPTIVE NAME Allow Note Import On SSN Only
DESCRIPTION 
Operates similar to TIU MED MANUAL PATIENT except that only SSN needs to 
match. 
 
This key is only to be used by approved personnel when a need to import a 
record for a patient whose data was incorrectly entered when staff using 
MED were offsite.
