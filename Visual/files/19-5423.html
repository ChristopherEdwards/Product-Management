RA INCOMPLETE (5423)    OPTION (19)

Name Value
NAME RA INCOMPLETE
X ACTION PRESENT YES
MENU TEXT Incomplete Exam Report
UPPERCASE MENU TEXT INCOMPLETE EXAM REPORT
DESCRIPTION 
This option allows the supervisor to generate a list of
all exams that have not been completed.  This report is similar to the
'Delinquent Status' report, but unlike that report, this one lists any
exam that is not in the 'complete' status.
CREATOR USER,ONE
TYPE run routine
E ACTION PRESENT YES
