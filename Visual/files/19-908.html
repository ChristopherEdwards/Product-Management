LRAC LIST (908)    OPTION (19)

Name Value
NAME LRAC LIST
MENU TEXT List of patients by location for cumulative report
UPPERCASE MENU TEXT LIST OF PATIENTS BY LOCATION F
DESCRIPTION 
This function will list patients sorted by location that are in the 
queue to get a cumulative for a given day.  This option is useful
when a reprint of the cumulative is required.
CREATOR POSTMASTER
TYPE run routine
PACKAGE LAB SERVICE
ROUTINE LRACM2F
