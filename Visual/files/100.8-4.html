ALLERGY-CONTRAST MEDIA INTERACTION (4)    ORDER CHECKS (100.8)

Name Value
NAME ALLERGY-CONTRAST MEDIA INTERACTION
DESCRIPTION 
Triggered by selection of Imaging orderable items.  CPRS checks to see if 
the imaging orderable item uses a contrast media.  If so, and the
patient has an Allergy Tracking System entry indicating allergy to contrast 
medias, the order check occurs.
