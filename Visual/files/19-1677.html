NAME ENFAR8
MENU TEXT Accountable NX for Station
UPPERCASE MENU TEXT ACCOUNTABLE NX FOR STATION
DESCRIPTION 
This option generates a list of accountable equipment sorted by category
stock number and CMR for a specified station. Equipment must be on a CMR
in order to be included in this report. If desired, only capitalized
equipment can be listed on the output.
