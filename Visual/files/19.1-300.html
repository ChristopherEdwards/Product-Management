LRDATA (300)    SECURITY KEY (19.1)

Name Value
NAME LRDATA
DESCRIPTIVE NAME EDIT LEDI RESULTS
DESCRIPTION 
 This key is used to identify those persons allowed to edit/change results
that were done by another outside laboratory. Manual editing of host
laboratory results is not the first option. If the result was not
generated by an associated division, this key is required to change the
results.
 
Note: The laboratory responsible for the original result should send and
edit LEDI message to correct errors. This may not be possible in all cases
requiring this manual edit feature.
