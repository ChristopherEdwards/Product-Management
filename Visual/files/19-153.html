XQKEYDEL (153)    OPTION (19)

Name Value
NAME XQKEYDEL
MENU TEXT Delegate keys
UPPERCASE MENU TEXT DELEGATE KEYS
DESCRIPTION 
 
This option will delegate keys to a user so that that user can then give them
out to other users, and even allow other users to delegate them in return.
CREATOR USER,ONE
TYPE run routine
ROUTINE EN3^XQ6
