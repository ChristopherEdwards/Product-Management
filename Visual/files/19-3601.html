IBDF PRINT SAMPLE ALL CLINICS (3601)    OPTION (19)

Name Value
NAME IBDF PRINT SAMPLE ALL CLINICS
MENU TEXT Print Sample of all Forms
UPPERCASE MENU TEXT PRINT SAMPLE OF ALL FORMS
DESCRIPTION 
This option will print out one copy of each form currently assigned to a
clinic and that is in use.  Use this to prepare a package of materials
for review or sharing with other facilities.
CREATOR USER,ONE
TYPE run routine
