DVB HNET-ENABLE (4186)    OPTION (19)

Name Value
NAME DVB HNET-ENABLE
MENU TEXT Enable HINQ Network
UPPERCASE MENU TEXT ENABLE HINQ NETWORK
DESCRIPTION 
This option enables the HINQ options which allow the user to transmit
requests to the VBA computer, 'PROCESS THE HINQ SUSPENSE FILE' and
'GENERATE A HINQ REQUEST', and 'INDIVIDUAL HINQ REQUESTS'.
CREATOR POSTMASTER
TYPE action
E ACTION PRESENT YES
ENTRY ACTION D UP^DVBHQTM
