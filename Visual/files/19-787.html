NAME LRDIST
MENU TEXT Graph results
UPPERCASE MENU TEXT GRAPH RESULTS
DESCRIPTION 
This function plots data in a highly generalized fashion.  If a specimen
is specified, normal ranges are displayed if available.  Furthermore,
one may choose to plot the data centered about the normal range or
centered about the mean of the data being plotted.  If plot according to
normal ranges is selected, then any values that differ from the mean
of the normal ranges by more than three standard deviations are plotted
in the extreme left or right column of the display, as the case may be.
In addition to being plotted in the extreme left or right column, the
value is distinguished by a '*' rather than a 'X'.
