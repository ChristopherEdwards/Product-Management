DMSQ TS PTR PARENT FULL (8026)    OPTION (19)

Name Value
NAME DMSQ TS PTR PARENT FULL
MENU TEXT List Pointer and Parent Links (Full)
UPPERCASE MENU TEXT LIST POINTER AND PARENT LINKS
DESCRIPTION 
This option shows the pointer and subfile links that reach outward from
a selected table.  Regular pointers (foreign keys or FKs) as well as
subfile links out and up to parent files (parent foreign keys or PFKs)
are shown.
CREATOR USER,ONE
TYPE run routine
ROUTINE EN2^DMSQP
