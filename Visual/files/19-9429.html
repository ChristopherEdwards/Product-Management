IBCE EXTRACT STATUS (9429)    OPTION (19)

Name Value
NAME IBCE EXTRACT STATUS
MENU TEXT Extract Status Management
UPPERCASE MENU TEXT EXTRACT STATUS MANAGEMENT
DESCRIPTION 
This is a list manager screen that will display bills that are trapped in a
ready for extract status due to the EDI/MRA parameter being turned off. 
From here, the valid actions are cancel a bill, cancel/clone/authorize
a bill without user interaction, or print the report.
