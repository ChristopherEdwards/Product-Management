PSO CMOP CS RX DISPENSE REPORT (8939)    OPTION (19)

Name Value
NAME PSO CMOP CS RX DISPENSE REPORT
MENU TEXT CMOP Controlled Substance Rx Dispense Report
UPPERCASE MENU TEXT CMOP CONTROLLED SUBSTANCE RX D
DESCRIPTION 
This option prints the status of CMOP Controlled Substance
prescriptions that are of Schedule III-V drugs.
 
The user is prompted for:
 
       1) Date range of release
       2) Sort by 'All' or a 'Single' Division
       3) Sort by 'Release Date' or 'Drug'
 
Fields printed are Division, Release Date, Patient, SSN, Drug, Quantity,
Status, and CMOP Facility.
