NAME PRCFPC PO BLD LST
MENU TEXT Build List of POs to Reprint in Fiscal by Date
UPPERCASE MENU TEXT BUILD LIST OF POS TO REPRINT I
DESCRIPTION 
Operator enters a beginning and ending date and time.  The program builds
and prints a list of all Purchase Orders that were sent to the Fiscal
Printer from Supply, between those dates and times.
