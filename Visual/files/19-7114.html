ECDS CATEGORY DEAC (7114)    OPTION (19)

Name Value
NAME ECDS CATEGORY DEAC
X ACTION PRESENT YES
MENU TEXT Activate/Inactivate Local Category
UPPERCASE MENU TEXT ACTIVATE/INACTIVATE LOCAL CATE
OUT OF ORDER MESSAGE OPTION NO LONGER SUPPORTED. USE THE ECS GUI INTERFACE.
DESCRIPTION 
This option provides the mechanism to inactivate categories that exist
in the EVENT CAPTURE CATEGORY file (#726). You have the ability to
reactivate a category using this option.
