NAME CMOP RX QUEUE
DESCRIPTION 
This file is used by the Consolidated Mail Outpatient Pharmacy system
at the medical center to manage prescription data for transmission to the
host facility. Data in this file is entered, edited, maintained, and purged 
by background jobs triggered by the Consolidated Mail Outpatient Pharmacy 
software and must NOT be edited using VA FileMan.
