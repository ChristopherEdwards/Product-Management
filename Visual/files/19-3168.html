PRCAT FMS OP REGEN (3168)    OPTION (19)

Name Value
NAME PRCAT FMS OP REGEN
MENU TEXT Overpayment (OP) Document Regeneration
UPPERCASE MENU TEXT OVERPAYMENT (OP) DOCUMENT REGE
DESCRIPTION 
This option will allow a user to retransmit a "rejected" OP document.
It will only allow the retransmission of an OP document that has actually
been refunded in the AR package and has been rejected by FMS.
