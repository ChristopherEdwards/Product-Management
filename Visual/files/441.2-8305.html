8305 (8305)    FEDERAL SUPPLY CLASSIFICATION (441.2)

Name Value
NUMBER 8305
ABBREVIATED DESCRIPTION TEXTILE FABRICS
DESCRIPTION 
NOTE:  THIS CLASS INCLUDES ONLY TEXTILES IN BULK.  EXCLUDED FROM 
THIS CLASS ARE ITEMS THAT ARE PRECUT OR PREFORMED, THUS
LIMITING THEIR USEFULNESS TO SPECIFIC APPLICATIONS.  SUCH
ITEMS ARE CLASSIFIED IN THE SAME CLASS AS THE ITEM OR ITEMS
FOR WHICH DESIGNED, OR IN SUCH OTHER CLASS AS IS APPROPRIATE.
INCLUDES CANVAS MATERIALS; PLASTIC FABRICS; FELT MATERIALS; NETTING,
PLASTIC AND TEXTILE; OILCLOTH; TEXTILE SCRAP.
EXCLUDES CARPETING.
GROUP 83
COMMODITY CODE General
DLA ADDRESS DLA/DASS
