NAME PSJI BACKGROUND JOB
MENU TEXT Compile IV Costs in Background
UPPERCASE MENU TEXT COMPILE IV COSTS IN BACKGROUND
DESCRIPTION 
  Statistical data is stored in a "holding area" each time an IV label is
printed.  This option should be queued to run nightly to compile and merge
this information into the IV statistics file (50.8), and delete any data
which is over 100 days old.
