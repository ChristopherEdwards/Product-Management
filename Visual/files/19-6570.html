SOWKAAD (6570)    OPTION (19)

Name Value
NAME SOWKAAD
MENU TEXT All Reports By Division (Part 1 & Part 2)
UPPERCASE MENU TEXT ALL REPORTS BY DIVISION (PART
DESCRIPTION 
This option is used to print AMIS parts 1 and 2 by division,
if the hospital is multi-divisional.  If this is a one divisional hospital,
the reports will print for the complete service.
CREATOR POSTMASTER
TYPE action
PACKAGE SOCIAL WORK
E ACTION PRESENT YES
ENTRY ACTION K ZTSK S SOWKAB="ALL",SOWKSITE=1 D ^SOWKPAOQ S SOWKAB="" K DIC,ZTSK,B,SOWKFD,SOWKFB,Z
