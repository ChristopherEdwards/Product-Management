FBUC RETURN ADDRESS DIS/ED (5278)    OPTION (19)

Name Value
NAME FBUC RETURN ADDRESS DIS/ED
MENU TEXT Return Address Display/Edit
UPPERCASE MENU TEXT RETURN ADDRESS DISPLAY/EDIT
DESCRIPTION 
This option will display the return address which will appear on an
Unauthorized Claim letter, if letterhead is not used.  You can also edit
the return address using this option.
CREATOR POSTMASTER
TYPE run routine
DISPLAY OPTION? YES
ROUTINE RETURN^FBUCDIS
