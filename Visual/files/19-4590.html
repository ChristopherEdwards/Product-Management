NAME PRSA TP DAY LIST
MENU TEXT Daily T&L List
UPPERCASE MENU TEXT DAILY T&L LIST
DESCRIPTION 
The user is prompted to select an existing Time and Leave Unit (T&L)
and a posting date. The names of the employees in the T&L, their
scheduled Tour of Duty and any exceptions to the scheduled Tour of
Duty are displayed.
