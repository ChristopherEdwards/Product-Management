SDACS CGATLIST (2257)    OPTION (19)

Name Value
NAME SDACS CGATLIST
MENU TEXT Computer Generated Appointment Type Listing
UPPERCASE MENU TEXT COMPUTER GENERATED APPOINTMENT
DESCRIPTION 
Listing of visits with 'Computer Generated' Appointment Type.
MAS must update to a valid type.  Use 'Edit Computer Generated
Appointment Type' option to update.
   
This option is only needed if other packages (such as Radiology 4.0)
capture stop code workload.
CREATOR POSTMASTER
TYPE run routine
E ACTION PRESENT YES
ENTRY ACTION S SDX="ACG"
ROUTINE EN^SDACSCGP
