PRCA CO-PAY WAIVER REPORT (3163)    OPTION (19)

Name Value
NAME PRCA CO-PAY WAIVER REPORT
MENU TEXT Co-Pay Waiver Report
UPPERCASE MENU TEXT CO-PAY WAIVER REPORT
DESCRIPTION 
This option allows the user to enter data for lines 9-20 of the co-pay
waiver report.  After the user accepts this data entry, a background
job is queued to compile data for lines 1-8.  After compilation of the
data, the report is sent to G.PCWMCCR at FORUM (in string format).  The
report also is delivered to the senders in box in a printed format.
CREATOR POSTMASTER
