IMAGING STOP CODES (71.5)    FILE (1)

Name Value
NAME IMAGING STOP CODES
DESCRIPTION 
This file contains valid imaging stop codes.  It is not pointed to by
any other file.  The Rad/Nuc Med ADPAC should make sure this file is
updated.  Obsolete or unused entries should be deleted.  This file 
should contain all stop codes that can be assigned to imaging locations
at the facility.
