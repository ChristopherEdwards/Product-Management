RAD/NUC MED PROCEDURE MESSAGE (71.4)    FILE (1)

Name Value
NAME RAD/NUC MED PROCEDURE MESSAGE
DESCRIPTION 
This file contains messages concerning special requirements when ordering
a procedure.  One or more of these messages can be tied to a procedure
in the Rad/Nuc Med Procedures file (#71) so that they are displayed to a
requestor at the time an order is placed.
 
 Data Storage
 ------------
 The data for the 'RAD/NUC MED PROCEDURE MESSAGE' file is stored in the
^RAMIS(71.4, global.
 
 Templates
 ---------
 The package does not use any input, print or sort templates
associated with this file.
