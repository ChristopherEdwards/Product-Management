MST HISTORY (29.11)    FILE (1)

Name Value
NAME MST HISTORY
DESCRIPTION 
This file tracks the Military Sexual Trauma status for veterans as part
of the data collection requirement for VHA Directive 98-058, 
"Sexual Trauma Counseling Care and Services".  Data in this file SHOULD
NOT BE MODIFIED directly.  This is a history file which tracks the 
historical MST status for a veteran.  All entries should be done through
the associated List Manager application.
