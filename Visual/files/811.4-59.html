VA-UNKNOWN OEF/OIF SERVICE (59)    REMINDER COMPUTED FINDINGS (811.4)

Name Value
NAME VA-UNKNOWN OEF/OIF SERVICE
ROUTINE PXRMMSER
ENTRY POINT UNKOEIF
PRINT NAME VA-Unknown OEF/OIF Service
DESCRIPTION 
This multi-occurrence computed finding will search for periods of OEF/OIF
service with an unknown location in the date range specified by Beginning
Date/Time and Ending Date/Time.

Subscripts that can be used in a Condition are:
 "LOCATION"

The default value is "LOCATION".
TYPE MULTIPLE
CLASS NATIONAL
