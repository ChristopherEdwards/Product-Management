PSJ LM NEW ORDER FROM PROFILE (2559)    PROTOCOL (101)

Name Value
NAME PSJ LM NEW ORDER FROM PROFILE
ITEM TEXT New Order Entry
DESCRIPTION 
This is used to create new Inpatient Medication orders from the profile
list.
TYPE action
CREATOR USER,SEVENTEEN
PACKAGE INPATIENT MEDICATIONS
EXIT ACTION D KILL^PSJBCMA5(+$G(PSJSYSP)) I PSJLK S X=DFN_";DPT(" D ULK^ORX2 D INIT^PSJLMHED(PSJPROT) S VALMBCK="Q"
ENTRY ACTION K PSJSTARI,^TMP("PSJINTER",$J) D KILL^PSJBCMA5(+$G(PSJSYSP)) K PSJLK S X=DFN_";DPT(" D LK^ORX2 S PSJLK=Y I Y D LMNEW^PSJOE0(DFN,PSJPROT)
