PSJI CONTROL CODES (9466)    OPTION (19)

Name Value
NAME PSJI CONTROL CODES
MENU TEXT Test Control Codes (IV)
UPPERCASE MENU TEXT TEST CONTROL CODES (IV)
DESCRIPTION 
This option allows the user to print an IV label for the purpose of 
verifying their control code definition.  An entire label is not printed, 
simply a bar code and one line of underscores.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
ENTRY ACTION D ^PSIVXU Q:$D(XQUIT) D ^PSIVALNC,ENIVKV^PSGSETU K I,Z,ZISI
