PSO ADDRESS CHANGE REPORT (9565)    OPTION (19)

Name Value
NAME PSO ADDRESS CHANGE REPORT
MENU TEXT Patient Address Changes Report
UPPERCASE MENU TEXT PATIENT ADDRESS CHANGES REPORT
DESCRIPTION 
This option prints a report that displays address changes made in the 
PATIENT file (#2). It can either be run for one patient or for all 
patients over the specified date range.
CREATOR USER,ONE
TYPE run routine
PACKAGE OUTPATIENT PHARMACY
ROUTINE PSOADDR
