NAME SPNL REGISTRANT INJURY REPORT
MENU TEXT Registrant Injury Report
UPPERCASE MENU TEXT REGISTRANT INJURY REPORT
DESCRIPTION 
This option prints SCD injury information of a registrant from file 154.
Fields printed are:  PATIENT, SSN, DOB, SCI LEVEL, EXTENT OF SCI, INFO
SOURCE FOR SCD, ETIOLOGY, DATE OF ONSET, and TYPE OF INJURY.
