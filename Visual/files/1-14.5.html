VOLUME SET (14.5)    FILE (1)

Name Value
NAME VOLUME SET
DESCRIPTION 
This file describes the volume sets available in the current multiprocessor
network.  The information pertaining to each volume set is primarily
used by the Kernel, especially TaskMan.  The ucis that make up each
volume set can be determined by using the cross-references in the UCI
Association Table file.
APPLICATION GROUP
  1. APPLICATION GROUP:   ZT
