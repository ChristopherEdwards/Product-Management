PRSA PP OPEN (4556)    OPTION (19)

Name Value
NAME PRSA PP OPEN
MENU TEXT Open Next Pay Period
UPPERCASE MENU TEXT OPEN NEXT PAY PERIOD
DESCRIPTION 
This option allows Payroll to create a new pay period entry in the
TIME AND ATTENDANCE RECORDS file (#458) and creates a time card entry
for each employee within that pay period.
