XUSTIME (3)    BULLETIN (3.6)

Name Value
NAME XUSTIME
SUBJECT SIGN-ON TO A TERMINAL DURING PROHIBITED TIME
DESCRIPTION 
This bulletin is triggered by the Sign-on system if the device being
used has a "PROHIBITED SIGN-ON TIMES" value, and somebody has tried to
sign on during that prohibited time period.
the prohibited time frame.
MESSAGE 
"|2|" attempted to sign on to device |1| with code "|3|",
but the device is locked out during the time period |4|.
PARAMETER
  1. PARAMETER:   1
    DESCRIPTION:   
    Device name ($I)
  2. PARAMETER:   2
    DESCRIPTION:   
    User name
  3. PARAMETER:   3
    DESCRIPTION:   
    The access code used for sign-on
  4. PARAMETER:   4
    DESCRIPTION:   
    The time range during which sign-on is prohibited.
