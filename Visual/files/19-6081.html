MENU TEXT List Operation Requests
UPPERCASE MENU TEXT LIST OPERATION REQUESTS
DESCRIPTION 
This option is designed to list requested cases, including those patients
on the waiting list.  It will print all future requests, sorted by ward
location or surgical specialty.
