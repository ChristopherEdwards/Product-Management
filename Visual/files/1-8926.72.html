TIU LOINC SMD SYNONYMS (8926.72)    FILE (1)

Name Value
NAME TIU LOINC SMD SYNONYMS
DESCRIPTION 
This file supports and facilitates the mapping of local titles to LOINC 
Standard Titles, by simplifying the correct association between partial 
names and the LOINC Standard Subject Matter Domains. Unlike the TIU LOINC 
SUBJECT MATTER DOMAIN File (#8926.2), this file may be locally extended.
