ESP PRINT FAC CRIME REPORT (9038)    OPTION (19)

Name Value
NAME ESP PRINT FAC CRIME REPORT
X ACTION PRESENT YES
MENU TEXT Print Facility Crime Report
UPPERCASE MENU TEXT PRINT FACILITY CRIME REPORT
DESCRIPTION 
This option allows persons to print a crime report
for a specific time period and Facility.  You must
first run the Generate Crime Statistics option before you
can print a crime report.
