ORMTIME RUN (8910)    OPTION (19)

Name Value
NAME ORMTIME RUN
MENU TEXT ORMTIME Run
UPPERCASE MENU TEXT ORMTIME RUN
DESCRIPTION 
Option calls EN^ORMTIME to run the ORMTIME processing.  This option does
not appear on any menu but is designed to be called through an entry in
the OPTION SCHEDULING file which effectively tasks it.
CREATOR USER,ONE
TYPE run routine
ROUTINE EN^ORMTIME
