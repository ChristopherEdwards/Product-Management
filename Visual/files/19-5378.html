RA INACPRCLONG (5378)    OPTION (19)

Name Value
NAME RA INACPRCLONG
MENU TEXT Inactive Procedure List (Long)
UPPERCASE MENU TEXT INACTIVE PROCEDURE LIST (LONG)
DESCRIPTION 
This option allows the supervisor to generate a list of
'inactive' procedures. The list includes the following
information: CPT code, procedure, AMIS code, AMIS category, multiplier,
whether the procedure is 'bilateral' and if the procedure is CT Head/Body.
The list also includes the following: Type of procedure, required flash
card printer, required flash card format, staff review of reports required,
interpreting physician approval of request required.
CREATOR USER,ONE
TYPE run routine
