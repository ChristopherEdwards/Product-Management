QAOS RPT ENROLL DATES TALLY (6904)    OPTION (19)

Name Value
NAME QAOS RPT ENROLL DATES TALLY
MENU TEXT Enrollment Dates Tally
UPPERCASE MENU TEXT ENROLLMENT DATES TALLY
DESCRIPTION 
This option produces a report showing the dates on which auto enroll
ran or failed to run.  For the dates auto enroll ran the number of
patients auto enrolled and manually entered is displayed.
CREATOR POSTMASTER
TYPE run routine
PACKAGE OCCURRENCE SCREEN
