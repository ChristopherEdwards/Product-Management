MH INSTRUMENTRULES (601.83)    FILE (1)

Name Value
NAME MH INSTRUMENTRULES
DESCRIPTION 
This file ties together an instrument in the MH TESTS AND SURVEYS file 
(#601.71), a question in the MH QUESTIONS file (#601.72) and a rule in the 
MH RULES file (#601.82). This allows a rule to be used in multiple instruments 
and for different questions.
