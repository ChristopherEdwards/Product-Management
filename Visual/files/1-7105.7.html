NOIS LOCATION TYPE (7105.7)    FILE (1)

Name Value
NAME NOIS LOCATION TYPE
DESCRIPTION 
This file is used to specify the type of NOIS location.
For NOIS locations that are sites, these entries might be a category
like computer system (Alpha, 486, etc.).  For locations that are in a
hospital, these entries might be services (medicine, outpatient clinics,
etc.).
