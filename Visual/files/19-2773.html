PRCB MULTIPLE CREATE/POST (2773)    OPTION (19)

Name Value
NAME PRCB MULTIPLE CREATE/POST
MENU TEXT Create/Post Multiple Transaction
UPPERCASE MENU TEXT CREATE/POST MULTIPLE TRANSACTI
DESCRIPTION 
This option will allow you to enter/edit/review multiple transactions and
 post them in the Fund Distribution file for release.  For transaction
 with a TDA # the single transaction option should be used.
Transaction can be entered now and posted at a later date.
