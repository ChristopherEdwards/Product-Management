PRSA SUP CERT (4565)    OPTION (19)

Name Value
NAME PRSA SUP CERT
MENU TEXT Supervisory Approvals
UPPERCASE MENU TEXT SUPERVISORY APPROVALS
DESCRIPTION 
This option displays any actions that need a supervisor's approval
such as a change in an employee's Tour of Duty. The supervisor is
prompted to approve, disapprove or cancel the request.
