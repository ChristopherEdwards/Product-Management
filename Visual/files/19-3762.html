PSGW LOOKUP ITEM (3762)    OPTION (19)

Name Value
NAME PSGW LOOKUP ITEM
MENU TEXT Ward/AOU List for an Item (80 column)
UPPERCASE MENU TEXT WARD/AOU LIST FOR AN ITEM (80
DESCRIPTION 
This option prints a list of all the wards and their 
associated AOUs that stock a particular item. The
user must specify the item to be looked up.
CREATOR POSTMASTER
TYPE run routine
ROUTINE PSGWVW
