PERCENT (2)    DELTA CHECKS (62.1)

Name Value
NAME PERCENT
XECUTABLE CODE Q:(+X=0)!(+X1=0)!(+X2=0)!(X?.A) S Y=0,X3=(X-X1)/X1*100 S:X3<0 X3=-X3 I X3>X2 S Y=1 W " << DELTA CHECK >> ",$J(X3,2,0)
DESCRIPTION 
Similar to ABS VALUE delta check except the % change will be displayed.
If you use this the DELTA VALUE should also be a percentage.
