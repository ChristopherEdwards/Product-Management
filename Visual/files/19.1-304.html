SD GAF SCORE (304)    SECURITY KEY (19.1)

Name Value
NAME SD GAF SCORE
DESCRIPTIVE NAME SD GAF SCORE
DESCRIPTION 
This key determines whether the user can determine a Global Assessment
of Functionalilty (AXIS 5) score for a patient.  Holders of this key
will be able to be selected at the provider prompt in the Outpatient
GAF score entry.
