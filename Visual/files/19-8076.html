NAME LA7S EDIT 62.93
MENU TEXT Edit Shipping Condition
UPPERCASE MENU TEXT EDIT SHIPPING CONDITION
DESCRIPTION 
This option sets up the LAB SHIPPING CONDITIONS file (#62.93).
This option allows sites to define and configure shipping
conditions used by lab shipping in determining how to ship and
transport lab specimens.
