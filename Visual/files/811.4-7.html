VA-SERVICE BRANCH (7)    REMINDER COMPUTED FINDINGS (811.4)

Name Value
NAME VA-SERVICE BRANCH
ROUTINE PXRMMSER
ENTRY POINT SBRANCH
PRINT NAME VA-Service Branch
DESCRIPTION 
This multiple occurrence computed finding will return service branch
information for one or more service periods in the date range.
 
CSUB data that can be used in a Condition are:
 "BRANCH"
 "DISCHARGE TYPE"
 "ENTRY DATE"
 "SEPARATION DATE"
 "SERVICE COMPONENT"
 
The default value is "BRANCH".
TYPE MULTIPLE
CLASS NATIONAL
