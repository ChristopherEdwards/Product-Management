PRCEF PRINT FROM LIST (2875)    OPTION (19)

Name Value
NAME PRCEF PRINT FROM LIST
MENU TEXT Print/Reprint 1358s from List
UPPERCASE MENU TEXT PRINT/REPRINT 1358S FROM LIST
DESCRIPTION 
This option allows the operator to reprint in Fiscal, some or all
of the 1358s from a list created using the option 'Build List
of 1358s Printed in Fiscal by Date'.
CREATOR USER,ONE
TYPE run routine
ROUTINE EN2^PRCHRPT6
