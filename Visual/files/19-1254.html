LRUFCLS (1254)    OPTION (19)

Name Value
NAME LRUFCLS
MENU TEXT Print single future collection label
UPPERCASE MENU TEXT PRINT SINGLE FUTURE COLLECTION
LOCK LRLAB
DESCRIPTION 
 This option is a single label version of Print Future Collection option.
There are two major differences from the other option. 1) This option
requires the user to supply the Order Number. 2) It only prints a single
label at a time.
CREATOR POSTMASTER
TYPE run routine
PACKAGE LAB SERVICE
ROUTINE LRLABLDS
