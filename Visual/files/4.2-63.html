NOTES 
 
The Q-FMZ.DOMAIN.EXT domain will be used to transmit extracted
information (via HL7) from the SCD (Spinal Cord) REGISTRY (#154)
File and the OUTCOMES (#154.1) file to the Austin Information
Technology Center (formerly the Austin Automation Center). This new
domain's use begins with patch SPN*2.0*26.
