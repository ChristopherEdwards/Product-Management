LR LOINC MAP DEFAULT (9248)    OPTION (19)

Name Value
NAME LR LOINC MAP DEFAULT
MENU TEXT Map Lab Tests to Default LOINC Code
UPPERCASE MENU TEXT MAP LAB TESTS TO DEFAULT LOINC
DESCRIPTION 
This option allows you to map a lab test to a default LOINC code.
A default LOINC code is a LOINC code for a lab test without a
specimen defined.  If you perform this test and the specimen is 
not in the TOPOGRAPHY file, then the default code is used.
CREATOR USER,ONE
TYPE run routine
PACKAGE LAB SERVICE
