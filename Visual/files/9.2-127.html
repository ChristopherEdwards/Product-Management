HEADER %ZOSF GLOBAL REFERENCE
TEXT 
The ^%ZOSF global is used to hold operating system-dependent logic so
that programs may be written independently of any specific operating
system.  Most of the nodes contain logic that must be executed to return
a value, e.g. X ^%ZOSF("SS").  See the Kernel documentation for a
functional description of the nodes.
