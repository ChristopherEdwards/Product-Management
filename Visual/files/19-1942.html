DG INPATIENT INQUIRY EXTENDED (1942)    OPTION (19)

Name Value
NAME DG INPATIENT INQUIRY EXTENDED
MENU TEXT Detailed Inpatient Inquiry
UPPERCASE MENU TEXT DETAILED INPATIENT INQUIRY
DESCRIPTION 
This option is used to view all information about a specific patient
concerning an individual admission.  The user first selects the
patient which he/she desires to view and then the specific admission
date for which he/she wishes to view data.
CREATOR POSTMASTER
TYPE run routine
PACKAGE REGISTRATION
ROUTINE EN^DGPMEX
