MENU TEXT Employee Inquiry
UPPERCASE MENU TEXT EMPLOYEE INQUIRY
DESCRIPTION 
This option presents information about current VAMC employees.
Employee data comes from file 450 - Current Employee
and file 200 - New Person.  Data includes: service/section, mail
code, phone, address, and birth date.
