QAOS RPT TXSP CARE TYPE (6920)    OPTION (19)

Name Value
NAME QAOS RPT TXSP CARE TYPE
MENU TEXT Display Treating Specialty Care Types
UPPERCASE MENU TEXT DISPLAY TREATING SPECIALTY CAR
DESCRIPTION 
This option produces a report showing all the treating specialties in use
at your facility and their associated care type (acute, special,
intermediate, NHCU, psych.).  If a specialty is not associated with a care
type it will be flagged as "NOT SPECIFIED".  The report may be sorted by
care type or specialty.
