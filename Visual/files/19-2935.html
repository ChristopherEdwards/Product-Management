PRCU FPDS DATA EDIT (2935)    OPTION (19)

Name Value
NAME PRCU FPDS DATA EDIT
MENU TEXT FPDS Data Edit
UPPERCASE MENU TEXT FPDS DATA EDIT
DESCRIPTION 
This option allows the user to edit FPDS data on a purchase order at
any time after the P.O. has been signed by a purchasing agent.
Before it is signed, the FPDS data can be edited using the normal
Edit an Incomplete Purchase Order option.
