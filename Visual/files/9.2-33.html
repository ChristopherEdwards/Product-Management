NAME XQ-USER-OOO
HEADER OPTIONS WHICH ARE NOT AVAILABLE
TEXT 
Sometimes an option is displayed in the menu system with the words
'(NOT AVAILABLE)' following it.  This means that the site manager or
package developer has temporarily disabled this option.
 
If you attempt to select the option, there is sometimes additional
text explaining the out-of-order condition.
