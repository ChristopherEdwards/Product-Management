FBUC UPDATE DATE LETTER SENT (5244)    OPTION (19)

Name Value
NAME FBUC UPDATE DATE LETTER SENT
MENU TEXT Update Date Letter Sent
UPPERCASE MENU TEXT UPDATE DATE LETTER SENT
DESCRIPTION 
This option should be used if you are not generating your letters.
It will update the date the letter was sent.  If applicable, the
expiration date will also be updated.
