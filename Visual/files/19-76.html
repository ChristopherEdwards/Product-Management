XUFDEV (76)    OPTION (19)

Name Value
NAME XUFDEV
MENU TEXT Device Failed Access Attempts
UPPERCASE MENU TEXT DEVICE FAILED ACCESS ATTEMPTS
DESCRIPTION 
This display sorts on device then date/time of attempt.
Gives subtotal of attempts for each device and a total of all
those attempts requested in sort.  Prompts for a print device 
to generate a hard copy if desired.
CREATOR USER,ONE
TYPE print
DELEGABLE NO
DIC {DIP} %ZUA(3.05,
L. 0
FLDS [XUDFDISPLAY]
BY [XUDFDISPS]
