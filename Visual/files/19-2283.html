DGPW PATIENT WRISTBAND PRINT (2283)    OPTION (19)

Name Value
NAME DGPW PATIENT WRISTBAND PRINT
MENU TEXT Print Patient Wristband
UPPERCASE MENU TEXT PRINT PATIENT WRISTBAND
DESCRIPTION 
This option allows the user to print a Patient Wristband.  The user
selects the patient's name.  It then looks at the ward the patient is
on, and determines from the division designated for the ward whether
the wristband will be printed.

TYPE run routine
