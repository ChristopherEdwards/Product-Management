Name Value
NAME FBUC MILL ACT SUMMARY
MENU TEXT Millennium Act Emergency Care Summary Report
UPPERCASE MENU TEXT MILLENNIUM ACT EMERGENCY CARE
DESCRIPTION 
This option will generate summary statistics for 38 U.S.C. 1725 emergency
care unauthorized claims that were entered into the system within a user 
specified date range. The purpose of this report is to automate the 
development of emergency care claim summary statistics that sites must
report to the Chief Business Office (CBO) on a monthly basis.
CREATOR USER,ONE
TYPE run routine
ROUTINE FBUCMBS
