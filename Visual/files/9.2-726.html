LREPILK (726)    HELP FRAME (9.2)

Name Value
NAME LREPILK
HEADER LINKER HELP
TEXT 

  Allows a choice on how to link the ANTIMICROBIAL SUSCEPTIBILITY file (#62.06)
    with the WKLD CODE file (#64).

     Auto   -   Will attempt to link any entries that are not currently linked.

     Manual -   This option can create or edit links. Selection is by the
                the entry in the ANTIMICROBIAL SUSCEPTIBILITY file (#62.06)

    Semi-Auto - Looks for entries in that are not currently linked and prompts
                the user to select the corresponding entry in the WKLD CODE
                file.
               
  Should there be no entry in the WKLD CODE file (#64) that the entry in the
    ANTIMICROBIAL SUSCEPTIBILITY file (#62.06) can be linked to please use
    the 'Additional Workload and Workload Codes Suffixes Request Form'
    available in the Emerging Pathogen Documentation (EPI) Patch LR*5.2*132
