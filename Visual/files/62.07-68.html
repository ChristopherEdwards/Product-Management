DIFF EQUAL 100 (68)    EXECUTE CODE (62.07)

Name Value
NAME DIFF EQUAL 100
EXECUTE CODE S %X=0 X "F %=394:1:399 S:$D(LRSB(%)) %X=%X+LRSB(%)" I %X'=100 W !,"DIFF does not equal 100. count = ",%X
TYPE DATA
DESCRIPTION 
Adds up the entries of all tests (data names from 394 to 399) and displays
a warning if the total does not equal 100.  Could be entered on any single
test in the panel in the EXECUTE ON DATA REVIEW field in file 60.
