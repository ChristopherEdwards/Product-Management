TIU LOINC SETTING SYNONYMS (8926.74)    FILE (1)

Name Value
NAME TIU LOINC SETTING SYNONYMS
DESCRIPTION 
This file supports and facilitates the mapping of local titles to LOINC 
Standard Titles, by simplifying the correct association between partial 
names and the LOINC Standard Settings. Unlike the TIU LOINC SETTING File
(#8926.4), this file may be locally extended.
