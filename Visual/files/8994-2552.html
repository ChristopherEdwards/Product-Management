DSIV PURGE AUDIT (2552)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIV PURGE AUDIT
TAG PURGE
ROUTINE DSIVIC1
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY PUBLIC
DESCRIPTION 
Purge ICB AUDIT FILE entries through <date> or keeping <days>.
INPUT PARAMETER
  1. INPUT PARAMETER:   DSIV
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Either a FileMan internal date, specifying the last date to purge,
OR a number specifying how many days to keep.
RETURN PARAMETER DESCRIPTION 
0=OK or -1^Error message.
