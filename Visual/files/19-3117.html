PRCAC TR RE-ESTABLISH BILL (3117)    OPTION (19)

Name Value
NAME PRCAC TR RE-ESTABLISH BILL
MENU TEXT Re-Establish Bill
UPPERCASE MENU TEXT RE-ESTABLISH BILL
DESCRIPTION 
This option is used to re-activate a bill that is in the WRITE-OFF, 
SUSPENDED, CANCELLATION or COLLECTED/CLOSED status.  This will
create a Re-establish Transaction and change the status of the
bill to Active.  (not prepayment bill.)
     
If the bill has no balance, you must enter in the re-establish amount
of the bill which will be the principal balance of the bill.
