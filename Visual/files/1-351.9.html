CLAIMSMANAGER BILLS (351.9)    FILE (1)

Name Value
NAME CLAIMSMANAGER BILLS
DESCRIPTION 
This file contains information on bills that have been sent to the Ingenix
ClaimsManager.
 
The entries in this file have matching entries in the BILL/CLAIMS file
(399).  The internal number in file 399 is the same as the internal number
in the CLAIMSMANAGER BILLS file.
 
Per VHA Directive 10-93-142, this file definition should not be modified.
