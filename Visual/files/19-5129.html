FBAA CANCEL MEDICAL VOID (5129)    OPTION (19)

Name Value
NAME FBAA CANCEL MEDICAL VOID
X ACTION PRESENT YES
MENU TEXT Medical Delete Void Payment
UPPERCASE MENU TEXT MEDICAL DELETE VOID PAYMENT
DESCRIPTION 
This option will delete the void flag. The dollar amount for the payment
must be subtracted from the obligation using the appropriate
IFCAP option.
