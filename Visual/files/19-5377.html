RA PROCSHORT (5377)    OPTION (19)

Name Value
NAME RA PROCSHORT
MENU TEXT Active Procedure List (Short)
UPPERCASE MENU TEXT ACTIVE PROCEDURE LIST (SHORT)
DESCRIPTION 
This option allows the supervisor to generate a list of 'active'
procedures. The list includes the following information: CPT code, procedure,
AMIS code, multiplier, whether the procedure is 'bilateral', and if the
procedure is CT Head/Body.
CREATOR USER,ONE
TYPE run routine
