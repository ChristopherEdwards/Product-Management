PRCB BOC ADD/EDIT (2914)    OPTION (19)

Name Value
NAME PRCB BOC ADD/EDIT
MENU TEXT Add/Edit BOC
UPPERCASE MENU TEXT ADD/EDIT BOC
DESCRIPTION 
This option allows you to add a new budget object code or change the 
name of an existing budget object code.  Whenever the program offices 
in VA Central Office change the budget object codes that you use, you 
will have to use this option to make the necessary changes to the 
Budget Object Code File. You can use this option to create new entries 
or to change the name or number of existing entries.
CREATOR USER,ONE
HELP FRAME PRCB BOC ADD/EDIT
TYPE run routine
ROUTINE ADD^PRCBSA
