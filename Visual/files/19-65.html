XUTESTUSER (65)    OPTION (19)

Name Value
NAME XUTESTUSER
MENU TEXT Switch Identities
UPPERCASE MENU TEXT SWITCH IDENTITIES
DESCRIPTION 
This option will allow a user to simulate signing on as another user
and thus test out their menus.  It will assign all keys, but will not
allow for the execution of any options.
CREATOR USER,ONE
HELP FRAME XUSER-TEST
TYPE run routine
DELEGABLE NO
ROUTINE TESTM^XUS91
