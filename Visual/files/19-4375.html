DENTTREAT RE-RELEASE (4375)    OPTION (19)

Name Value
NAME DENTTREAT RE-RELEASE
MENU TEXT Enable Edit/Re-Release of Treatment Data (Full)
UPPERCASE MENU TEXT ENABLE EDIT/RE-RELEASE OF TREA
OUT OF ORDER MESSAGE DAS turned off
DESCRIPTION 
This option should be used to enable re-release of a Treatment Data 
entry that has been rejected by Austin.  This option merely enables the
re-release of a Treatment Data entry.  The next time Treatment Data is
released using the Review/Release option, this entry can be included in the
transmission to Austin.
CREATOR POSTMASTER
TYPE run routine
ROUTINE TSE1^DENTE2
