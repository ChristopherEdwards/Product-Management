PRCA FMS RSC CALCULATE (8155)    OPTION (19)

Name Value
NAME PRCA FMS RSC CALCULATE
MENU TEXT Calculate Revenue Source Code For A Bill
UPPERCASE MENU TEXT CALCULATE REVENUE SOURCE CODE
DESCRIPTION 
This option will show (calculate) the revenue source code for a selected
bill.  For non-accrued bills in fund 0160A1, the revenue source code is
entered by the user and is not calculated.  For accrued bills in fund
5287, the revenue source code will be calculated and displayed.
CREATOR USER,ONE
TYPE run routine
PACKAGE ACCOUNTS RECEIVABLE
ROUTINE SHOW^RCXFMSUR
