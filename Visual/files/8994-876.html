ORWTIU CANLINK (876)    REMOTE PROCEDURE (8994)

Name Value
NAME ORWTIU CANLINK
TAG CANLINK
ROUTINE ORWTIU
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
 Given a title, call CANLINK^TIULP to determine whether this title can use
 linked as an Interdisciplinary child note.
 
dbia #2322
INPUT PARAMETER
  1. INPUT PARAMETER:   TITLE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   YES
RETURN PARAMETER DESCRIPTION 
 1^      - document can be an ID child
 0^error - document cannot be an ID child, and reason.
