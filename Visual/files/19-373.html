XMMGR-RESPONSE-TIME-COMPILER (373)    OPTION (19)

Name Value
NAME XMMGR-RESPONSE-TIME-COMPILER
MENU TEXT Compile Response Time Statistics
UPPERCASE MENU TEXT COMPILE RESPONSE TIME STATISTI
DESCRIPTION 
This option will be used as a background job scheduled every day
to collect statistics on the previous day's response time and put
the data into file 4.2998 .  It will then KILL off the associated
%ZRTL global nodes.
