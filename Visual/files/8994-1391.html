OOPS GET WITNESSES (1391)    REMOTE PROCEDURE (8994)

Name Value
NAME OOPS GET WITNESSES
TAG WITR
ROUTINE OOPSGUI2
RETURN VALUE TYPE ARRAY
AVAILABILITY PUBLIC
WORD WRAP ON FALSE
DESCRIPTION 
This broker call with take and IEN as input and retrieve witness information
and associated comments from the ASISTS 2260 file.
INPUT PARAMETER
  1. INPUT PARAMETER:   IEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    DESCRIPTION:   
    The only input is the IEN of the ASISTS case number you wish to retrieve 
Witness information from.
RETURN PARAMETER DESCRIPTION 
The Return parameter is an Array containing the Witness information 
(sometimes multiple entries ) and associated comments.
