QAOS EDIT SCREEN (6887)    OPTION (19)

Name Value
NAME QAOS EDIT SCREEN
MENU TEXT VAMC-Specific Screens
UPPERCASE MENU TEXT VAMC-SPECIFIC SCREENS
DESCRIPTION 
This option allows the user to enter site specific screens in the range
201-999.99.  This option also allows the user to set the screen status
to either local or inactive.  Exceptions to screens may also be entered.
