ONCO FOLL-ATTEMPT FOLLOWUP (6253)    OPTION (19)

Name Value
NAME ONCO FOLL-ATTEMPT FOLLOWUP
MENU TEXT Attempt a Follow-up
UPPERCASE MENU TEXT ATTEMPT A FOLLOW-UP
DESCRIPTION 
Allows for the tracking of follow-up attempts, successful or non-successful.
Select date, contact, type of contact - the letter status is set to
"pending" automatically. Selection of letter to send is linked to the
type of follow-up selected. Successful contacts lead into posting of
follow-up. This process then allows for not only tracking the follow-up
history, but the attempt history as well.
CREATOR USER,ONE
TYPE run routine
ROUTINE ATM^ONCOFUL
