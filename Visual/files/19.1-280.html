PRPF DEFERRAL OVERRIDE (280)    SECURITY KEY (19.1)

Name Value
NAME PRPF DEFERRAL OVERRIDE
DESCRIPTION 
This key is used to authorize an individual to override a deferral
date on an account and to disperse funds.  It does not allow
an individual to overdraw the account.
 
To assure backward compatibility, If the key has NOT been assigned to
anyone, all clerks may override a deferral.  However if even one person
has been assigned the key, ALL persons needing to override a transaction
will require the key.
 
This key should be assigned to a lead clerk, supervisor, assistant chief
or chief.
