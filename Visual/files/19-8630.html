ORW HEALTH SUMMARY TYPES (8630)    OPTION (19)

Name Value
NAME ORW HEALTH SUMMARY TYPES
MENU TEXT GUI Health Summary Types
UPPERCASE MENU TEXT GUI HEALTH SUMMARY TYPES
DESCRIPTION 
This option allows you to set up a list of health summary types to be
displayed on the GUI Reports tab. Don't add health summaries that require
additional prompting (e.g., for date ranges & occurrence limits) 
since the GUI cannot support this type of prompting.
CREATOR USER,ONE
TYPE action
Short Menu Text Health Summary Types
E ACTION PRESENT YES
ENTRY ACTION D EDITPAR^XPAREDIT("ORWRP HEALTH SUMMARY TYPE LIST")
