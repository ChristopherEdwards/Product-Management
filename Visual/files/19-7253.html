LRAR ARCHIVE NP CONVERSION (7253)    OPTION (19)

Name Value
NAME LRAR ARCHIVE NP CONVERSION
MENU TEXT Convert archived data to use New Person file
UPPERCASE MENU TEXT CONVERT ARCHIVED DATA TO USE N
DESCRIPTION 
After previously archived data is restored to the ^LAR global, this option
should be run to convert any records that were not converted during the
install of LAB V5.2.  Do this before moving the data back into the ^LR
global.  Records previously converted will be unaffected.
CREATOR USER,ONE
TYPE run routine
ROUTINE LRARNPX
