FISCAL NOTIFICATION (128)    MAIL GROUP (3.8)

Name Value
NAME FISCAL NOTIFICATION
DESCRIPTION 
Members of this group are notified whenever an obligation
amount is decreased through use of the "Obligate Data" option.
 
This mail group is also used to notify Fiscal when the
delivery date has been changed in a Purchase Order by
A&MM Service.
 
This mail group is used to notify Fiscal when an amendment has been
Pending Fiscal Action for 4 or more days.
TYPE public
ORGANIZER USER,ONE
