2 (2)    LOCAL PROCUREMENT REASON CODES (443.8)

Name Value
CODE 2
CODE EXPANSION PERISHABLE GOODS
DESCRIPTION 
Not to be used for commodities with normal shelf life restrictions or
expiration dates.  To be used for items which will spoil if not 
refrigerated, frozen or otherwise provided special handling and storage.
e.g. organs for transplant, blood, biologicals shipped in dry ice, etc.
