PRCSPITEMH (2601)    OPTION (19)

Name Value
NAME PRCSPITEMH
MENU TEXT Item History
UPPERCASE MENU TEXT ITEM HISTORY
DESCRIPTION 
This option prints/displays the history of an item in the Item file, using
the last five purchase orders on which the item appeared.
