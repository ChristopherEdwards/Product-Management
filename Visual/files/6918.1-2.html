SCRTY (2)    UTILITY SYSTEMS (6918.1)

Name Value
NAME SCRTY
FULL NAME SECURITY SYSTEMS
DESCRIPTION 
Systems that provide monitoring or control of space or people beyond
mechanical locking systems, such as:
 
       Passkey Systems
       Access Control
       Intrusion Detection
       Traffic Monitoring
       Gates, Doors, Vaults
       Employee Client Identification and Verification
       Patient Tracking Systems
       Video Monitoring and Time Stamping
       Building and Security Time Stamps
       Time Card and Check-in Systems
       Communications to Security Agencies
