DENTFEE RE-RELEASE (4376)    OPTION (19)

Name Value
NAME DENTFEE RE-RELEASE
MENU TEXT Applications and Fee (type 5) Edit/Re-Release
UPPERCASE MENU TEXT APPLICATIONS AND FEE (TYPE 5)
OUT OF ORDER MESSAGE DAS turned off
DESCRIPTION 
This option should be used to enable the re-release of Applications and
Dental Fee entries that have been rejected by Austin.  This option merely
enables the re-release of an entry.  The next time Applications and Dental
Fee data is released using the Review/Release option, this entry can be
included in the transmission to Austin.
CREATOR POSTMASTER
TYPE run routine
ROUTINE FBRR^DENTE2
