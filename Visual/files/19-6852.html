QAM PURGE HISTORY FILE (6852)    OPTION (19)

Name Value
NAME QAM PURGE HISTORY FILE
MENU TEXT History File Purge
UPPERCASE MENU TEXT HISTORY FILE PURGE
DESCRIPTION 
Purge History File:  Prompts the user for the monitor or a group of 
monitors that the user wants to purge and a date range.  There is a 
warning message to the user before the data is purged.
