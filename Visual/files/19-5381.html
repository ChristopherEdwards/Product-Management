RA PROCSERIES (5381)    OPTION (19)

Name Value
NAME RA PROCSERIES
MENU TEXT Series of Procedures List
UPPERCASE MENU TEXT SERIES OF PROCEDURES LIST
DESCRIPTION 
This option allows the supervisor to generate a list
of procedures which are classified as 'SERIES'.  The
following information is included: CPT code, procedure, AMIS code,
multiplier, bilateral, CT Head/Body, type of procedure, required
flash card printer, required flash card format, and staff review of
reports required.
CREATOR USER,ONE
TYPE run routine
