SD EDIT LOCAL STOP CODE NAME (11217)    OPTION (19)

Name Value
NAME SD EDIT LOCAL STOP CODE NAME
MENU TEXT Edit Clinic Stop Code Name - Local Entries Only
UPPERCASE MENU TEXT EDIT CLINIC STOP CODE NAME - L
LOCK SD SUPERVISOR
DESCRIPTION 
This option will allow the user to edit only the  NAME field for the 
CLINIC STOP (#40.7) file.  Only those entries identified as "local" may be
edited.
 
Any entry associated with the AMIS REPORTING STOP CODE VALUES of 451-456, 
458-473, 475-479, and 482-485 is considered local and may be edited.
CREATOR USER,SEVENTEEN
TYPE run routine
ROUTINE UPDLOCAL^SDUTL3
