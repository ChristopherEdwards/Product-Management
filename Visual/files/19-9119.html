SPN PRINT ADMISSIONS (9119)    OPTION (19)

Name Value
NAME SPN PRINT ADMISSIONS
MENU TEXT SCI/SCD Admissions
UPPERCASE MENU TEXT SCI/SCD ADMISSIONS
DESCRIPTION 
This report provides a list of SCD patients who have been admitted within
a user-specified date range.  The list consists of admitted patients who
are either in the SCD Registry or who have been marked as SCI in the
Patient file (i.e., field 57.4, "SPINAL CORD INJURY", has been populated).
