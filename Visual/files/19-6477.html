NAME QAC ISSUE TOT
MENU TEXT Issue Totals for All, Male, or Female
UPPERCASE MENU TEXT ISSUE TOTALS FOR ALL, MALE, OR
DESCRIPTION 
This option prints the total number of issues for a date range.  The print
can be for all records or only for records according to gender.  The user
selects a date range and counts for All, Male, or Female.
