XUTM BACKGROUND PRINT (201)    OPTION (19)

Name Value
NAME XUTM BACKGROUND PRINT
MENU TEXT Print Options that are Scheduled to run
UPPERCASE MENU TEXT PRINT OPTIONS THAT ARE SCHEDUL
DESCRIPTION 
This option prints a list of options from the option schedule file that
have data in one of the background task fields:
 1. Queued to run at what time.
 2. Device for queued job output.
 3. Rescheduling frequency
 4. Special queueing
CREATOR USER,ONE
TYPE print
DIC {DIP} DIC(19.2,
L. 0
FLDS [XQ-BACKGROUND SCHEDULE]
BY [XQ-BACKGROUND SCHEDULE]
