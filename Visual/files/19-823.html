LRNONCOM (823)    OPTION (19)

Name Value
NAME LRNONCOM
MENU TEXT Special test accessioning
UPPERCASE MENU TEXT SPECIAL TEST ACCESSIONING
DESCRIPTION 
Several additional questions are asked in this option.  The key
question is which accession area the test(s) will be put under.
 
If the order number is known and entered, the accessioning is the
same as the [LROE] option with the above noted exception.
This option may be used for accessioning employee health patient
and other non-standard patients; e.g., research animals.
CREATOR POSTMASTER
TYPE run routine
PACKAGE LAB SERVICE
ROUTINE LROEN^LROE
