FSC QUERY CALLS (10343)    OPTION (19)

Name Value
NAME FSC QUERY CALLS
MENU TEXT Query Calls
UPPERCASE MENU TEXT QUERY CALLS
DESCRIPTION 
This option selects calls meeting a specific criteria for display
on the List Screen.  Queries are made by specifying a field, condition,
and value for a call.  This is useful for finding calls that are similar
to a problem being reported.  You may also use queries within the List
Screen to add calls to a list or to remove or select calls from a list.
