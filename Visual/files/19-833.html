DESCRIPTION 
Data from an automated instrument can be reviewed/edited individually
by accession, by worklist sequence, by tray-cup.  If the LRVERIFY
key is owned, the data may also be verified.  Once verified, the data are
not available for viewing with this option.  Likewise, if test results
have been previously verified for a given accession, even new data
from the automated instrument can neither be viewed or verified.
Entering the "*" character for any entry will substitute the word
"canceled" as the result.
