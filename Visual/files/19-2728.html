PRCHPM CS MODIFY BATCH PRI. (2728)    OPTION (19)

Name Value
NAME PRCHPM CS MODIFY BATCH PRI.
MENU TEXT Modify Batch Priority (LOG/GSA/DLA)
UPPERCASE MENU TEXT MODIFY BATCH PRIORITY (LOG/GSA
DESCRIPTION 
This option allows a PPM person to assign a higher (2) or
lower (4) batch priority to a code sheet.  This can be used to insure 
that a certain code sheet will be transmitted ahead of or after others.
CREATOR USER,ONE
TYPE run routine
ROUTINE MBP^PRCFALOG
