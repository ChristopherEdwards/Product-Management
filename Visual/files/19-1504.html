ENWODISAP (1504)    OPTION (19)

Name Value
NAME ENWODISAP
MENU TEXT Disapprove Work Order
UPPERCASE MENU TEXT DISAPPROVE WORK ORDER
DESCRIPTION 
Use this option to record disapproval action on a work request. It is
anticipated that this option will be most useful at facilities which allow
entry of work requests into AEMS/MERS by personnel outside of Engineering
Service.
CREATOR POSTMASTER
TYPE run routine
ROUTINE DISAP^ENWO2
