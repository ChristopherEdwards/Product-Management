FSC FILE SETUP (10351)    OPTION (19)

Name Value
NAME FSC FILE SETUP
MENU TEXT File Setup
UPPERCASE MENU TEXT FILE SETUP
DESCRIPTION 
This option uses VA FileMan to print, inquire, and search NOIS
files.  NOIS Supervisors may also edit these files (with the exception
of the NOIS Call file which is only editable using the NOIS options).
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
ENTRY ACTION D OPT^FSCOP("FILES")
INDEPENDENTLY INVOCABLE YES
