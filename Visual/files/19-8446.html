XPAR EDIT PARAMETER (8446)    OPTION (19)

Name Value
NAME XPAR EDIT PARAMETER
MENU TEXT Edit Parameter Values
UPPERCASE MENU TEXT EDIT PARAMETER VALUES
DESCRIPTION 
The option calls the low level parameter editor which allows you to edit
the values for every parameter.  Normally packages supply other means of
editing parameters.
CREATOR USER,ONE
TYPE action
Short Menu Text Edit Parameter Values
E ACTION PRESENT YES
ENTRY ACTION D ^XPAREDIT
