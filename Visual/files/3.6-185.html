HDIS XML MSG PROCESS ERROR (185)    BULLETIN (3.6)

Name Value
NAME HDIS XML MSG PROCESS ERROR
SUBJECT ERROR(S) DURING MESSAGE PROCESSING
MAIL GROUP
  1. MAIL GROUP:   HDIS ERRORS
DESCRIPTION 
Bulletin is generated when a problem processing a received XML encoded 
message occurs.
MESSAGE 
An error occurred when attempting to process an XML message that was sent
to an HDIS MailMan Server.  The message was not deleted and left in the 
server basket.

Message Sender : |2|
Server Option  : |3|
