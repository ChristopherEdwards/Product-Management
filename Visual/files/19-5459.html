RA COMMON PROCEDURE (5459)    OPTION (19)

Name Value
NAME RA COMMON PROCEDURE
MENU TEXT Common Procedure Enter/Edit
UPPERCASE MENU TEXT COMMON PROCEDURE ENTER/EDIT
DESCRIPTION 
This menu option may be used to add and or remove procedures from
the display used to enter requests. Procedures should
not be deleted from this file but just inactivated. When you
enter 'YES' to the 'Inactive' field it will automaticly remove
it's sequence number.
 
You may enter as many procedures into the Common Procedure file as you
wish, however, you may only assign sequence numbers from 1 to 40.  Only
procedures with sequence numbers are displayed, therefore the maximum
number of common procedures that can be displayed is 40.
 
Any procedures entered into this file (with or without a sequence number)
can be converted into an Order Entry/Results Reporting protocol, thus
making the procedure an orderable item that can be selected by anyone
requesting an exam in OE/RR.
