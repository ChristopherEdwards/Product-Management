MISSING LAB TESTS FOR ANGIOGRAM PROCEDURE (22)    ORDER CHECKS (100.8)

Name Value
NAME MISSING LAB TESTS FOR ANGIOGRAM PROCEDURE
DESCRIPTION 
Triggered at conclusion of ordering session. If the Imaging
order is linked to CPRS national term ANGIOGRAM (PERIPHERAL) and required 
concurrent lab tests prothrombin time and thromboplastin were not ordered in
this session, the order check occurs.
