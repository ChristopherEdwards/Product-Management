Name Value
NAME LAB MONTHLY WORKLOADS
DESCRIPTION 
  This file is used to collect workload data in preparation for
transmission to the National Data Base. Each months LMIP reportable
statistics is compiled into this file. The file is reviewed by the
Laboratory Approving Official, the a mail message is prepared and
forwarded to the user. The user in turn forwards the mail message to
the appropriate Domain.
  This file should be archived before purging the data.
 
  THIS FILE SHOULD NOT BE LOCALLY EDITED.
