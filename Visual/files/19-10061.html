RCDPE ACTIVE WITH EEOB REPORT (10061)    OPTION (19)

Name Value
NAME RCDPE ACTIVE WITH EEOB REPORT
MENU TEXT Active Bills With EEOB Report
UPPERCASE MENU TEXT ACTIVE BILLS WITH EEOB REPORT
DESCRIPTION 
This option produces the ACTIVE BILLS WITH EEOBs report.  This report
contains a list of bills by insurance company that have EEOBs on file
and also have a balance greater than 0.
