DESCRIPTION 
This option allows the user to edit a completed 7078/authorization.
The following fields may be editted using this option:
         - Admitting Authority
         - To Date
         - Discharge Type
         - Bedsection/Treating Specialty
         - Purpose of Visit
         - Authorization Remarks
         - Primary Service Area
