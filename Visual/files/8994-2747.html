MAG DICOM GET RAD CPT MOD (2747)    REMOTE PROCEDURE (8994)

Name Value
NAME MAG DICOM GET RAD CPT MOD
TAG CPTMOD
ROUTINE MAGDRPCB
RETURN VALUE TYPE ARRAY
AVAILABILITY RESTRICTED
INACTIVE ACTIVE
VERSION 3
DESCRIPTION 
This RPC is used to return a valid entry in the CPT MODIFIER file (#81.3).
INPUT PARAMETER
  1. INPUT PARAMETER:   CPTMOD
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the name of the CPT modifier.
  2. INPUT PARAMETER:   MAGDT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is the date used during the lookup process for screening
'active' CPT modifiers.
RETURN PARAMETER DESCRIPTION 
RESULTS(1)= -1 (entry not found) or 1 (entry found)
RESULTS(2)= ien^CPT name
