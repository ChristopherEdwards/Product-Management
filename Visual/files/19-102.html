NAME XQSMD SHOW
MENU TEXT Show a Delegate's Options
UPPERCASE MENU TEXT SHOW A DELEGATE'S OPTIONS
DESCRIPTION 
This option will show you who created a delegate, when, and what
delegation level they are; as well as all of the options delegated
to that person, when each was delegated and by whom.
