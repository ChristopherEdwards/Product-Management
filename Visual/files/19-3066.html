PRCAC TR TERM-FISCAL (3066)    OPTION (19)

Name Value
NAME PRCAC TR TERM-FISCAL
MENU TEXT Fiscal Officer Terminated
UPPERCASE MENU TEXT FISCAL OFFICER TERMINATED
DESCRIPTION 
Use this option to process the account terminated by fiscal officer.
This changes the account's status to 'Write-Off'.
CREATOR USER,ONE
TYPE run routine
PACKAGE ACCOUNTS RECEIVABLE
ROUTINE 8^RCWROFF
