NAME ENPM6
MENU TEXT Enter/Edit PM Procedure
UPPERCASE MENU TEXT ENTER/EDIT PM PROCEDURE
DESCRIPTION 
Enter procedures to be followed in performing PMI's. The procedure
identifier will be printed on PM worksheets as a reminder to the tech,
and will become a part of the equipment history whenever a scheduled
PMI is recorded. It is recommended that the full text of PM procedures
be stored in this file if time and disk space permit.
