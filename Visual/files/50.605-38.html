Name Value
CODE AS000
CLASSIFICATION ANTISEPTICS/DISINFECTANTS
TYPE MAJOR
DESCRIPTION 
NOTE: Includes products used only for the disinfection of
       inanimate objects and surfaces (e.g.,benzalkonium
       chloride).
      Excludes products used for the cleansing or disinfection
       of animate objects (e.g.,hexachlorophene (DE400)) and
       products used for the cleansing or disinfection of both
       animate and inanimate objects (e.g.,povidone iodine
       (DE101)).
