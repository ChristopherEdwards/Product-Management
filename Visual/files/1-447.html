INVENTORY LOCK MANAGEMENT (447)    FILE (1)

Name Value
NAME INVENTORY LOCK MANAGEMENT
DESCRIPTION 
This file is used to manage the locking of files by user in the inventory
package.  When a user locks a file or inventory item, an entry is made
in this file.  Other users which try to access the locked file or item
will be displayed a message showing the user and option which is locking
the file or item.
