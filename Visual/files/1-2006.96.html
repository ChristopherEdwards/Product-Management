IMAGE INDEX CONVERSION (2006.96)    FILE (1)

Name Value
NAME IMAGE INDEX CONVERSION
DESCRIPTION 
This file records the internal entry numbers of IMAGE File (#2005) 
entries whose index values in Fields 40-45 have been automatically 
generated using the functionality exported in Patches 17 and 25.
 
This file is to be set ONLY by image index conversion routines.  It may 
NOT be edited by sites.
