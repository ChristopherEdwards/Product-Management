QAOS RPT RELIABILITY ASSMT (6919)    OPTION (19)

Name Value
NAME QAOS RPT RELIABILITY ASSMT
MENU TEXT Reliability Assessment Worksheets
UPPERCASE MENU TEXT RELIABILITY ASSESSMENT WORKSHE
DESCRIPTION 
This option randomly selects occurrences, within user specified
parameters, and prints worksheets for the assessment of inter-reviewer
reliability.  The user may select a date range, the type of screens to
include (National, Local, Inactive) and, the number of occurrences to
capture.
