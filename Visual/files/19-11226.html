PXRM ORDER CHECK TESTER (11226)    OPTION (19)

Name Value
NAME PXRM ORDER CHECK TESTER
MENU TEXT Reminder Order Check Test
UPPERCASE MENU TEXT REMINDER ORDER CHECK TEST
DESCRIPTION 
This option allows the user to run a test against a patient and an orderable
item. This allows them to evaluate the results of a reminder order check
rule before turning it on in CPRS.
