FBCH DELETE INVOICE (5217)    OPTION (19)

Name Value
NAME FBCH DELETE INVOICE
MENU TEXT Delete Inpatient Invoice
UPPERCASE MENU TEXT DELETE INPATIENT INVOICE
DESCRIPTION 
This option will allow a user to delete an invoice entered in error.
The invoice must be in a batch that has not been released for payment.
