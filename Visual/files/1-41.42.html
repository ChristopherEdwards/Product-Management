PRE-REGISTRATION CALL LIST (41.42)    FILE (1)

Name Value
NAME PRE-REGISTRATION CALL LIST
DESCRIPTION 
This file contains a list of patients to be called as part of the 
Pre-Registration process.
 
*** NOTE ***
This file SHOULD NOT be edited directly.  Field values need to be
entered by the background job, or the Add to Call List Option.
Editing this file directly can result in erroneous data being displayed
in the Call List.
