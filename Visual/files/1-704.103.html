TERM_QUALIFIER_PAIR (704.103)    FILE (1)

Name Value
NAME TERM_QUALIFIER_PAIR
DESCRIPTION 
NOTICE: This file is maintained exclusively by the CP Terminology group 
and must not be modified or edited in any way. Subsequent releases of CP 
Terminology may create lasting errors in the CliO data store if this file 
is changed.
 
This file is a join table for the Clinical Data Model that matches 
observations to their allowable qualifiers.  A sample could be the 
observation equating to "weight" and the paired qualifier equating to 
"lift scale".
