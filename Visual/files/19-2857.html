PRCT ENTER/EDIT/VIEW (2857)    OPTION (19)

Name Value
NAME PRCT ENTER/EDIT/VIEW
MENU TEXT Enter/Edit/View
UPPERCASE MENU TEXT ENTER/EDIT/VIEW
DESCRIPTION 
This option will allow the viewing of what data user uploaded from
a barcode reader and the status of the barcode data.  This option
will allow the entering/viewing/editing of barcode data.
