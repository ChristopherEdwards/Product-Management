ECTP T&L INDIVIDUAL (5529)    OPTION (19)

Name Value
NAME ECTP T&L INDIVIDUAL
MENU TEXT Individual PAID Inquiry
UPPERCASE MENU TEXT INDIVIDUAL PAID INQUIRY
DESCRIPTION 
This option produces a report for a selected employee.  The user may
choose the pay period/date range.  For the selected time-frame,
the following PAID data will be displayed: annual leave, sick leave,
leave without pay, authorized absence, earned comp time, used comp
time, and over time.
