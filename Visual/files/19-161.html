XUFILEPRINT (161)    OPTION (19)

Name Value
NAME XUFILEPRINT
X ACTION PRESENT YES
MENU TEXT Print Users Files
UPPERCASE MENU TEXT PRINT USERS FILES
OUT OF ORDER MESSAGE ACCESS DISABLED
DESCRIPTION 
This option lists, by user, each file the user has access to and what
that access is.  Users who have no access are not listed.
CREATOR USER,ONE
TYPE print
EXIT ACTION K %ZISI,B,P,DIJ,DISYS
DIC {DIP} VA(200,
L. 0
FLDS [XUFILEOFILES]
DIS(0) I '$P(^VA(200,D0,0),U,11),$P(^(0),U,4)="@"!($N(^("FOF",0))>0)
