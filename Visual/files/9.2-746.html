NAME GMRV-PO2 RATE HELP
HEADER Pulse Oximetry
TEXT 
This field stores the patient's oxygen saturation (SaO2) measured by pulse
oximetry.  The value is interpreted as a percent.
  Normal values are usually 95-99% but this might vary according to the
type of presenting disease, e.g. there are many patients that tolerate a
pulse oximetry of 70-80.
