NAME SDAM RPT WAITING TIME
MENU TEXT Appointment Waiting Time Report
UPPERCASE MENU TEXT APPOINTMENT WAITING TIME REPOR
DESCRIPTION 
This report will list the waiting time for patients with scheduled
appointments.  The patient has to be checked-in and check-out.
 
It will give the user the opportunity to print a full report with patient's
names or a report of totals only.
 
Both types of reports will furnish averages for clinics/stop codes, division
and total for the hospital.  It will print in the format of:
 
Patient - ID - (Clinic) - CI Date/Time - Elapsed Minutes - Appt. Date/Time
 - CO Date/Time - Elapsed Minutes - Total Minutes
