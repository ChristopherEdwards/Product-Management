TEXT 
The ultimate goal of Library Management is to provide an integrated
automated library system for VALNET.  The first portion available is the
Serials Control Module which includes check-in, routing, bibliographic
data, holdings information, and purchasing data.  Access to the Serials
Management (LSM) option is restricted to the Chief, Library Service or
other locally designated professional librarian in order to maintain the
integrity of data in key parts of the file.  Access to the Serials 
Control (SER) option is not as restricted, since this option has functions
which can be routinely updated by any library staff member.
