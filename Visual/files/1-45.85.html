CENSUS WORKFILE (45.85)    FILE (1)

Name Value
NAME CENSUS WORKFILE
DESCRIPTION 
This file is a work file of patients who were inpatients as
of 11:59pm on acensus date.  File entries are created by the 
Regenerate Census Workfile option of the Census module.
 
The file is used to produce the Census Status Report.  Without this
workfile, it would be necessary to search all admissions in order
to compile this report.  This file eliminates that processing.
