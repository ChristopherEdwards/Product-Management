SDSC PURGE NSC ENC (10768)    OPTION (19)

Name Value
NAME SDSC PURGE NSC ENC
MENU TEXT Purge ASCD NSC Encounters
UPPERCASE MENU TEXT PURGE ASCD NSC ENCOUNTERS
LOCK SDSC SUPER
DESCRIPTION 
This option will purge encounters with a status of NEW where the 
Encounter SC value equals the ASCD value of "NO" for a specified
division(s) within a user defined date range. Users must have the SDSC
SUPER key to run this option.
CREATOR USER,ONE
TYPE run routine
PACKAGE SCHEDULING
ROUTINE EN^SDSCNSCP
