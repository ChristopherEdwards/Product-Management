QAM SITE PARAMETERS EDIT (6851)    OPTION (19)

Name Value
NAME QAM SITE PARAMETERS EDIT
X ACTION PRESENT YES
MENU TEXT Site Parameters Edit
UPPERCASE MENU TEXT SITE PARAMETERS EDIT
DESCRIPTION 
Site Parameters Edit:  The user configures the program to the site 
through this option.  Edits include 1) the date the weekly time frame 
begins, 2) use of the "contains" operator (Yes or No) in the group edit, 
fields that control the manual run of auto enroll, and the 
device that auto enroll will print to.
CREATOR POSTMASTER
TYPE edit
E ACTION PRESENT YES
EXIT ACTION K %X,%Y,DI,DLAYGO,DQ
ENTRY ACTION S DLAYGO=740
DIC {DIC} QA(740,
DIC(0) AELMNQ
DIE QA(740,
DR {DIE} [QAQA QAM SITE PARAMETERS]
