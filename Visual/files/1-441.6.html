TYPE OF REQUISITION AMENDMENT (441.6)    FILE (1)

Name Value
NAME TYPE OF REQUISITION AMENDMENT
DESCRIPTION 
  This file contains a list of the allowable types of amendments that
can be made to a Requisition after it has been Obligated. The file is used
during the Amendment process, both to allow the user to select the type of
amendment, and to direct the programs to the proper entry point for
processing the type of amendment selected.
