OOPS RELEASE RECORD LOCK (1427)    REMOTE PROCEDURE (8994)

Name Value
NAME OOPS RELEASE RECORD LOCK
TAG CLRLCK
ROUTINE OOPSGUI3
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
This broker call will release the previously locked ASISTS record.
INPUT PARAMETER
  1. INPUT PARAMETER:   IEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This input parameter contains the ASISTS Internal Record Number that will be
used to attempt to unlock a previously locked record.
RETURN PARAMETER DESCRIPTION 
The return parameter will contain a status message relating to clearing the
lock.
