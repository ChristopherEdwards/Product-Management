PSJU PLATCS (3648)    OPTION (19)

Name Value
NAME PSJU PLATCS
MENU TEXT Send Pick List to ATC
UPPERCASE MENU TEXT SEND PICK LIST TO ATC
LOCK PSJU PL
DESCRIPTION 
  Allows the user to have the appropriate meds from a selected Pick List
dispensed through the ATC machine.  PLEASE NOTE that only those meds on
the pick list that have the appropriate ATC data (canister number and
mnemonic) will dispense through the ATC.
CREATOR USER,ONE
TYPE run routine
ROUTINE EN^PSGTAP
