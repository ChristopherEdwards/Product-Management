SD WL SERVICE/SPECIALTY (409.31)    FILE (1)

Name Value
NAME SD WL SERVICE/SPECIALTY
DESCRIPTION 
This file contains the list of DSS Identifiers that can be used by an 
institution in the Wait List (Sch/PCMM) package as a Service/Specialty. 
Each DSS ID is assigned an institution, date activated and date 
inactivated. The Service/Specialty represents a collection of clinics 
that have this DSS ID as the primary stop code that a patient is waiting 
for an appointment.
