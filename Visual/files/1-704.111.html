OBS_VIEW (704.111)    FILE (1)

Name Value
NAME OBS_VIEW
DESCRIPTION 
This file maintains the views displayed within a flowsheet via the 
Clinical Flowsheets application.  A view is a group of terms displayed 
together.
A sample view could display "ICU VITALS,INPUTS,OUTPUTS, and ICU 
HEMODYNAMICS" information together in a section of a flowsheet 
displayed via the Clinical Flowsheets application.
