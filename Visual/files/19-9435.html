Name Value
NAME IBCE MESSAGE SCREEN TEXT
MENU TEXT EDI Message Text to Screen Maintenance
UPPERCASE MENU TEXT EDI MESSAGE TEXT TO SCREEN MAI
DESCRIPTION 
This option allows for the display of a list of words or phrases
that if found in the text of an informational status message, will 
either always require the message to be reviewed or will auto-file
the message and flag it as not needing a review.
