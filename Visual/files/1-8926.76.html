TIU LOINC DOCUMENT TYPE SYNONYMS (8926.76)    FILE (1)

Name Value
NAME TIU LOINC DOCUMENT TYPE SYNONYMS
DESCRIPTION 
This file supports and facilitates the mapping of local titles to LOINC 
Standard Titles, by simplifying the correct association between partial 
names and the LOINC Standard Document Types. Unlike the TIU LOINC 
DOCUMENT TYPE File (#8926.6), this file may be locally extended.
