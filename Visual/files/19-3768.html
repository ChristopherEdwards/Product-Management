PSGW PRINT AMIS WORKSHEET (3768)    OPTION (19)

Name Value
NAME PSGW PRINT AMIS WORKSHEET
MENU TEXT Print AMIS Worksheet (80 column)
UPPERCASE MENU TEXT PRINT AMIS WORKSHEET (80 COLUM
DESCRIPTION 
This option prints a worksheet to be used in classifying AR/WS
drugs for AMIS.  The print order is by "type", and within type
the drug listing is alphabetical.
