NAME RCTOP BILL REPORT
MENU TEXT TOP Bill Report
UPPERCASE MENU TEXT TOP BILL REPORT
DESCRIPTION 
This report lists the bills for the individual debtor that have been
forwarded to TOP.  It shows a status code, the total amount of the bill,
the principle,  interest, admin, marshall fee and court cost for each
bill.
