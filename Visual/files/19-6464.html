NAME QAC ARCHIVE
MENU TEXT Archive Report
UPPERCASE MENU TEXT ARCHIVE REPORT
DESCRIPTION 
This report captures the Contact Date, Contact Number, Patient Name, SSN,
Issue Codes and Resolution Comments.  No records are purged using this
option.  At the device prompt, the report may be sent to a file, captured
for a word processing package, or sent to a printer for a hard copy.
