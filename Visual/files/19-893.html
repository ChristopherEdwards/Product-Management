NAME LRCKPTR
MENU TEXT Check patient and lab data cross pointers
UPPERCASE MENU TEXT CHECK PATIENT AND LAB DATA CRO
LOCK LRLIASON
DESCRIPTION 
This routine will look at all patients and check that the
lab data entry pointed to points back to it.
The routine then looks at all entries in the lab data file
and checks that the pointed to file points back.
