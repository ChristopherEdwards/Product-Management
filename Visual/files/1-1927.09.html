NAME NUPA SAVED NOTES
DESCRIPTION 
This is the file that contains notes that are saved by the nurses.  Do 
not edit this file via Fileman.  To purge this file, be sure option NUPA
PURGE SAVED NOTES (Purge Saved Notes Over 5 Days Old) is queued to run
nightly.
