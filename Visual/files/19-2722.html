PRCHPM REQN REMOVE 2237 (2722)    OPTION (19)

Name Value
NAME PRCHPM REQN REMOVE 2237
MENU TEXT Remove 2237 from Requisition
UPPERCASE MENU TEXT REMOVE 2237 FROM REQUISITION
DESCRIPTION 
This option will remove the line items associated with a 2237 from a
Requisition. The line items can then be transferred onto another
Requisition.
CREATOR USER,ONE
TYPE run routine
ROUTINE EN0^PRCHEB
