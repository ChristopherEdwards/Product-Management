LEX CSV ICD QUERY (10938)    OPTION (19)

Name Value
NAME LEX CSV ICD QUERY
X ACTION PRESENT YES
MENU TEXT ICD-9 Diagnosis Code Set Query
UPPERCASE MENU TEXT ICD-9 DIAGNOSIS CODE SET QUERY
DESCRIPTION 
This option displays a single versioned entry from the ICD 
Diagnosis file #80 based on a date provided by the user.  
The date may be a future date.
CREATOR USER,SEVENTEEN
TYPE run routine
E ACTION PRESENT YES
EXIT ACTION K ^TMP("LEXQID",$J),^TMP("LEXQIDA",$J),^TMP("LEXQIDO",$J)
ENTRY ACTION K ^TMP("LEXQID",$J),^TMP("LEXQIDA",$J),^TMP("LEXQIDO",$J)
ROUTINE EN^LEXQID
