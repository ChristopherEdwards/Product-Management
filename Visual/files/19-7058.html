ABSV MEAL PRINT (7058)    OPTION (19)

Name Value
NAME ABSV MEAL PRINT
MENU TEXT Print/Display Meal List
UPPERCASE MENU TEXT PRINT/DISPLAY MEAL LIST
DESCRIPTION 
This option allows the user to print or display a listing of the volunteers
on the meal list for a specific date.  It DOES NOT print the meal list FORMS,
rather only an alphabetical listing of volunteers.
