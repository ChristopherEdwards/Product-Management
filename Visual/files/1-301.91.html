IVM REASONS FOR NOT UPLOADING INSURANCE (301.91)    FILE (1)

Name Value
NAME IVM REASONS FOR NOT UPLOADING INSURANCE
DESCRIPTION 
 
This file contains reasons why the facility did not upload insurance data
sent from the IVM Center.
 
New entries should not be added to this file.  The addition of new
entries must be coordinated with the IVM Center.
 
Per VHA Directive 10-93-142, this file definition should not be modified.
