DESCRIPTION
This RPC allows for Significant Findings updates to be added to a Consult.
 
Cloned from ORQQCN SIG FIND

    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1

findings update to.

    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    1
    REQUIRED:
    YES

'Y' - Significant Finding
'N' - No Significant Finding
'U' - Unknown Significant Finding

    PARAMETER TYPE:
    LIST
    MAXIMUM DATA LENGTH:
    80
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    3
    DESCRIPTION:

Significant Findings update.

'N' - NO
