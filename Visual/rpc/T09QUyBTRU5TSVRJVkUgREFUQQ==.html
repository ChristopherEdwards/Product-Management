OOPS SENSITIVE DATA (1571)

Name Value
NAMEOOPS SENSITIVE DATA
TAGSENSDATA
ROUTINEOOPSGUI7
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYPUBLIC
DESCRIPTION
This Broker call is used to pass data to the bulletin routine for supervisors
accessing sensitive data and not creating an ASISTS case.
INPUT PARAMETER
  1. INPUT PARAMETER:
    SDUZ
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    DESCRIPTION:
    This is the DUZ of the supervisor accessing the Employee's data.
  2. INPUT PARAMETER:
    EMP
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    DESCRIPTION:
    This is the Employee's name for which the sensitive date has been viewed.
