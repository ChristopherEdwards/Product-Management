DSIR KILL BILL (2080)

Name Value
NAMEDSIR KILL BILL
TAGKILLBILL
ROUTINEDSIRBIL0
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
This RPC will delete a bill and all related entries in the DSIR BILL 
HISTORY and DSIR BILL TRACKING files.
INPUT PARAMETER
  1. INPUT PARAMETER:
    BILL IEN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    9999
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    Internal Number to file 19620.2 DISR BILLS.
  2. INPUT PARAMETER:
    ADMIN OVERRIDE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    1
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is a set of codes 0 = No, 1 = Yes to override the checking for any 
payments and delete the bill and all payments.
RETURN PARAMETER DESCRIPTION
Return Values
 
        1 - Successful deletion
        -1^Missing Bill Number!
        -2^No Bill on file for Bill IEN!
        -3^Payment(s) received for $  nnn.nn. Unable to delete!
