DESCRIPTION
Provides a list of Active Background Processes.
INPUT PARAMETER
  1. INPUT PARAMETER:
    WORKSTAT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This value is the actual workstation name as specified by
the network operating system.  It is identical to the field
in the VISTA workstation file (#2006.8) named WRKS COMPUTER
NAME (#50).
  2. INPUT PARAMETER:
    PROCESS
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    8
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This value is by design either "B-PROC" or "Purge" and is used to rename
the VISTA Job name on DSM systems.  This value has the last two IP octet
concatenated to it for easy identification for system management purposes.
It is also displayed in the title bar of the workstation process form.
of the workstation process form.
