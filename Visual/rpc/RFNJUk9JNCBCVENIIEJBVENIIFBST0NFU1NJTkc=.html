DSIROI4 BTCH BATCH PROCESSING (3000)
Name Value
NAMEDSIROI4 BTCH BATCH PROCESSING
TAGBTCH
ROUTINEDSIROI4
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
This RPC clones requests in a batch manner.
INPUT PARAMETER
  1. INPUT PARAMETER:
    TYPE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    1
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    The type of cloning (1=multi patient, 2=multi requestors)
  2. INPUT PARAMETER:
    IEN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    The IEN of the request being cloned from 19620.
  3. INPUT PARAMETER:
    SELLIST
    PARAMETER TYPE:
    LIST
    MAXIMUM DATA LENGTH:
    60
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    A list of requestors or patients as follows:
   REQUESTOR IEN^REQUESTOR ADDRESS IEN
   or
   PATIENT IEN^PATIENT ADDRESS IEN^PATIENT ADDRESS IEN^ADDRESS TYPE
           (PATIENT IEN MUST BE FULLY QUALIFIED)
RETURN PARAMETER DESCRIPTION
RETURN VALUE
   ONE BATCH REQUEST HAS BEEN CREATED
   or
   ## BATCH REQUESTS HAVE BEEN CREATED
   or
   -1^error message
