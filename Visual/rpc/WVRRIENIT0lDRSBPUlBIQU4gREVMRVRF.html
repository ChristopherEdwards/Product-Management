DESCRIPTION
This API will remove all choices (601.75) that are not part of a 
CHOICETYPE (601.751)
 
It is suggested that API YTQ CHOICETYPE ORPHAN DELETE will be run before 
the API.

        OUTPUT list of choices deleted
