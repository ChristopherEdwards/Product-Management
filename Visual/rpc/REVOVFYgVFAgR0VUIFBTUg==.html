DENTV TP GET PSR (1901)
Name Value
NAMEDENTV TP GET PSR
TAGPSR
ROUTINEDENTVTP3
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYAGREEMENT
WORD WRAP ONTRUE
DESCRIPTION
Returns all of the PSR type records from the Treatment Plan 
Transaction/Exam file (#228.2) for a particular patient.
INPUT PARAMETER
  1. INPUT PARAMETER:
    DFN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    20
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is a pointer to the PATIENT file.
RETURN PARAMETER DESCRIPTION
Returns a global array List[n] := PSR data, or if problems returns
List[1] : -1^error message.
The format of the PSR data will be:
  List[n] := p1^p2^p3^p4  where
       p1 := transaction id
       p2 := exam date (external format)
       p3 := provider name
       p4 := PSR score string
