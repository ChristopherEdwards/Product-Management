DSIF CNSLTS ADDCMT (3226)
Name Value
NAMEDSIF CNSLTS ADDCMT
TAGADDCMT
ROUTINEDSIFCNS1
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYRESTRICTED
DESCRIPTION
This RPC allows for the adding of a Comment to a Consult.
 
Cloned from ORQQCN ADDCMT
INPUT PARAMETER
  1. INPUT PARAMETER:
    DSIFIEN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the Consult IEN in which a comment needs to be added to.
  2. INPUT PARAMETER:
    DSIFCOM
    PARAMETER TYPE:
    LIST
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is the array of comments that need to be added to the Consult.
  3. INPUT PARAMETER:
    DSIFALRT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    1
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    This is the indiciation that alerts should be sent to someone.
 
1 means yes.
  4. INPUT PARAMETER:
    DSIFALTO
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    4
    DESCRIPTION:
    Contains pointers to NEW PERSON FILE (200) for those to receive alert for 
this addition of a comment.
  5. INPUT PARAMETER:
    DSIFORDT
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    5
    DESCRIPTION:
    This is the date of the Comment is added to the Consult. If not passed 
in, defaults to NOW^XLFDT.
RETURN PARAMETER DESCRIPTION
Returns:
1^Successful
-1^Error Message
