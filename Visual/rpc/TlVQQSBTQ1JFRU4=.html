NUPA SCREEN (2944)
Name Value
NAMENUPA SCREEN
TAGSCREEN
ROUTINENUPABCL
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
Allows M code to be executed from inside a Delphi program.
INPUT PARAMETER
  1. INPUT PARAMETER:
    CODE
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    Code to execute.  If you set variable NUPa to a result, it will be pulled 
back into the assessment GUI.
 
Code can be:
 
S NUPA=$$D(^DPT(DFN))
 
D TAG^ROUTINE
RETURN PARAMETER DESCRIPTION
Varies depending on the code xecuted.
