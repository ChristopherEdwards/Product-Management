TIU GET ADDITIONAL SIGNERS (156)

Name Value
NAMETIU GET ADDITIONAL SIGNERS
TAGGETSIGNR
ROUTINETIULX
RETURN VALUE TYPEARRAY
AVAILABILITYSUBSCRIPTION
DESCRIPTION
Returns the list of additional signers currently identified for a given
TIU document.
INPUT PARAMETER
  1. INPUT PARAMETER:
    TIUDA
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the record # of the TIU Document in file 8925.
RETURN PARAMETER DESCRIPTION
This is an array of persons whose signature is expected on the document.
e.g.,
 
       TIUY(1)=213^CASEY,BEN
       TIUY(2)=73425^HOWSER,DOOGEY
       TIUY(3)=172^WELBY,MARCUS
 
Note that the array elements are returned as DUZ^Username.
