XWB DEFERRED GETDATA (436)
Name Value
NAMEXWB DEFERRED GETDATA
TAGRTNDATA
ROUTINEXWBDRPC
RETURN VALUE TYPEARRAY
AVAILABILITYPUBLIC
WORD WRAP ONTRUE
VERSION1
DESCRIPTION
This RPC is used to return the data from the XWB DEFERRED RPC call.
INPUT PARAMETER
  1. INPUT PARAMETER:
    HANDLE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    The HANDLE from the XWB DEFERRED RPC. It is used to link the call to the
data.
RETURN PARAMETER DESCRIPTION
 The return value is the array of data.  In the event of an error
condition, the first node of the array is equal to a string with the
syntax "-1^error text".
