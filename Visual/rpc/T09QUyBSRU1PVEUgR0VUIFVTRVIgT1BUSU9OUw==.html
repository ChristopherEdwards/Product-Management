OOPS REMOTE GET USER OPTIONS (1380)
Name Value
NAMEOOPS REMOTE GET USER OPTIONS
TAGOPT
ROUTINEOOPSGUI1
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
This call returns what ASISTS GUI menu options the user has assigned to them.
RETURN PARAMETER DESCRIPTION
In a single value ^ delimited array of seven pieces: P1 =  if DUZ
defined, P2 = 1 if Emp Health menu available, P3 = 1 if Emp menu, P4 = 1 if
Supervisor menu, P5 = 1 if Safety Officer menu, P6 = 1 if Union, and P7 = 1 if
Workers' Comp menu, otherwise piece = 0.
