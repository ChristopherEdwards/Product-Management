DSIF CNSLTS LEX (3235)

Name Value
NAMEDSIF CNSLTS LEX
TAGLEX
ROUTINEDSIFCNS1
RETURN VALUE TYPEARRAY
DESCRIPTION
This RPC performs a Lexicon search based upon what is passed in.
 
Cloned from LEX^ORWPCE
INPUT PARAMETER
  1. INPUT PARAMETER:
    DSIFVAL
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the text or code you wish to search Lexicon for.
  2. INPUT PARAMETER:
    DSIFAPP
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is the type of search you want to perform:
 
CPT or ICD. Null searches all of Lexicon
  3. INPUT PARAMETER:
    DSIFDT
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    This is the date you want to do the Lexicon search for. If nothing is 
passed, it uses the current date.
RETURN PARAMETER DESCRIPTION
Returns a list of Lexicon codes in the following format:
 
IEN^Short Description (Code System Code)
 
or
 
-1^Error.
