DENTV FEE BASIS LIST (1891)
Name Value
NAMEDENTV FEE BASIS LIST
TAGLIST
ROUTINEDENTVFB
RETURN VALUE TYPEARRAY
AVAILABILITYAGREEMENT
DESCRIPTION
This RPC gets a list of DENTAL FEE BASIS EXTRACT (#228.5) records for
a specified patient.  The input is a single value, patient id (DFN).
INPUT PARAMETER
  1. INPUT PARAMETER:
    INPUT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    The input value is a single patient id (DFN).
RETURN PARAMETER DESCRIPTION
Returns an array containing the internal entry number (IEN) appended to 
the beginning of the zero node of the DENTAL FEE BASIS EXTRACT file, in 
the format:  ien;txnUID^create dt^create person^patient^facility^
             category^date authorized^cost^^^^
