DSIR GET SENSITIVE PROVIDERS (2231)
Name Value
NAMEDSIR GET SENSITIVE PROVIDERS
TAGPROVLIST
ROUTINEDSIROI8
RETURN VALUE TYPEGLOBAL ARRAY
WORD WRAP ONTRUE
DESCRIPTION
This RPC returns a list of all persons flagged as sensitive in the ROI 
package.
RETURN PARAMETER DESCRIPTION
Array '^' delimited
       IEN - Pointer to file 200
       NAME - Provider Name
       TITLE - Title from field 8 in file 200
or
       -1 ^ No Provider Found!
