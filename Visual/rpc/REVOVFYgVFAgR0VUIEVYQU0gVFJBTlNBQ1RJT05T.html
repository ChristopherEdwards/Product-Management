DENTV TP GET EXAM TRANSACTIONS (3019)
Name Value
NAMEDENTV TP GET EXAM TRANSACTIONS
TAGGETFEX
ROUTINEDENTVTP7
RETURN VALUE TYPEARRAY
AVAILABILITYAGREEMENT
INACTIVEACTIVE
DESCRIPTION
This will return a complete list of exam transactions for a specified 
patient. The array is organized by node, not by date. For example all of 
the "OHA" Oral Health Assessment data is returned together. See 
GETFEX^DENTVTP7 comments for details.
INPUT PARAMETER
  1. INPUT PARAMETER:
    DPAT
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This value should be the Patient IEN.
RETURN PARAMETER DESCRIPTION
The return will group all return values by type (ex. "OHA" data will be 
returned in one group, then "OCC" in another group). If there is no data 
in a specific group to be returned, there will be no header ($$START^OHA 
for example).
