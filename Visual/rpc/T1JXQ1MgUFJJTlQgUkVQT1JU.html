ORWCS PRINT REPORT (272)

Name Value
NAMEORWCS PRINT REPORT
TAGPRINT
ROUTINEORWCSP
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYSUBSCRIPTION
DESCRIPTION
This rpc is used to print a consult report on the Consult tab
in CPRS.
INPUT PARAMETER
  1. INPUT PARAMETER:
    DEVICE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    100
    REQUIRED:
    YES
    DESCRIPTION:
    Name of the device where the report should print.

Format: <device name> ; <right margin> ; <page length>

Right margin and page length are optional.
  2. INPUT PARAMETER:
    DFN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    20
    REQUIRED:
    YES
    DESCRIPTION:
    Internal entry number of entry in the Patient file.
  3. INPUT PARAMETER:
    CONSULT ID
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    20
    REQUIRED:
    YES
    DESCRIPTION:
    Identification number of the consult.
RETURN PARAMETER DESCRIPTION
If the print request was successfully queued then the
Task manager task number is return. Otherwise, and error
code and error description are returned.

Error Code Table:
  
     Code            Text
     ----            ----
       0             <Task Number>

       1             No device selected
       2             No exam specified
       6             Patient specified is not valid
