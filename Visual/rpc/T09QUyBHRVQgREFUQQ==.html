OOPS GET DATA (1399)
Name Value
NAMEOOPS GET DATA
TAGGETDATA
ROUTINEOOPSGUI3
RETURN VALUE TYPEARRAY
DESCRIPTION
This broker call returns the data in the ASISTS Accident Reporting File (#2260)
for fields that are 1) Set of Codes, 2) Set of Codes Multiple, 3) Pointer to
a file, and 4) Pointer to a file multiple.
INPUT PARAMETER
  1. INPUT PARAMETER:
    INPUT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    256
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This parameter contains the File #, the Field #, and the IEN of the data to be
extracted.
RETURN PARAMETER DESCRIPTION
The RESULTS array will contain the data that was requested.
