OOPS CHANGE CASE STATUS (1396)
Name Value
NAMEOOPS CHANGE CASE STATUS
TAGCHGCASE
ROUTINEOOPSGUI6
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
This broker call passes in the ASISTS IEN, new case status and if the case
status is deleted the reason for deletion.
INPUT PARAMETER
  1. INPUT PARAMETER:
    INPUT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    256
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    The INPUT parameter has the format: IEN^CASE STATUS where IEN is the ASIST 
record IEN and CASE STATUS is the case status the claim should be changed to.
  2. INPUT PARAMETER:
    FLD58
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    80
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    The parameter FLD58 is the text field Reason for Deletion if the case status
has been changed to deleted.
RETURN PARAMETER DESCRIPTION
The return parameter, RESULTS, contains a message indicating the new case
number, if changed.  The message is: Case Status has been changed 
to: 'case #'.
