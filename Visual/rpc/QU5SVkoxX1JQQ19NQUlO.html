ANRVJ1_RPC_MAIN (2241)

Name Value
NAMEANRVJ1_RPC_MAIN
TAGRPC
ROUTINEANRVJ1
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYRESTRICTED
APP PROXY ALLOWEDYes
DESCRIPTION
This is the main entry point for calling line tags in routine ANRVJ1.  
The format is:
        Parameter 1 = OPTION (name of the line tag to call)
        Parameter 2 = DFN (value being passed in)
        Parameter 3 = DATA (any additional values to be passed in)
INPUT PARAMETER
  1. INPUT PARAMETER:
    OPTION
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    10
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    Routine Line tag you wish to call.
  2. INPUT PARAMETER:
    DFN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    Parameter variable to be passed into routine tag.
  3. INPUT PARAMETER:
    DATA
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    Any additional parameters to be passed into the routine line tag.
RETURN PARAMETER DESCRIPTION
Results are a global array.
