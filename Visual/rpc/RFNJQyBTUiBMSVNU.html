DSIC SR LIST (1612)
Name Value
NAMEDSIC SR LIST
TAGLISTSR
ROUTINEDSICSRPT
RETURN VALUE TYPEARRAY
AVAILABILITYAGREEMENT
DESCRIPTION
Return a list of actual surgical cases for a patient for a date range
INPUT PARAMETER
  1. INPUT PARAMETER:
    DFN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    20
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is patient pointer from file 2 for which to retrieve cases.
  2. INPUT PARAMETER:
    ALPHA
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    14
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is the starting date.time in Fileman format for retrieving surgical 
cases.  Default value is 1/1/2000
  3. INPUT PARAMETER:
    OMEGA
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    14
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    This is a Fileman date.time  It is the ending date for surgical cases.  
The default value is TODAY
RETURN PARAMETER DESCRIPTION
On error or problems, return List[1] := -1^message
 
Else return:
   DSICX(#) = p1^p2^p3^...^p10
         p1 = ien to file 130
         p2 = principal procedure name
         p3 = Fileman date.time of operation
         p4 = external date.time or operation
         p5 = provider DUZ number
         p6 = provider name
         p7 = TIU IEN of operation report
         p8 = TIU IEN of anesthesia report
         p9 = TIU IEN of nurse's report
        p10 = TIU IEN of procedure report (NON-OR)
