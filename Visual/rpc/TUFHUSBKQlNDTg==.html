MAGQ JBSCN (2904)

Name Value
NAMEMAGQ JBSCN
TAGCNP2
ROUTINEMAGQBPG1
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYRESTRICTED
DESCRIPTION
This function returns the next image file entry to be evaluated by the
BP Verifier function.
INPUT PARAMETER
  1. INPUT PARAMETER:
    IEN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    256
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This value is the initial $Order value applied to the Image file.
  2. INPUT PARAMETER:
    START
    PARAMETER TYPE:
    LIST
    MAXIMUM DATA LENGTH:
    32
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is the starting IEN for the BP Verifier session.
  3. INPUT PARAMETER:
    STOP
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    32
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    This is the ending IEN for the BP Verifier session.
  4. INPUT PARAMETER:
    AUTO
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    1
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    4
    DESCRIPTION:
    When set this will allow the BP Verifier session to terminate automatically.
RETURN PARAMETER DESCRIPTION
A string containing each of the network and Jukebox internal entry
numbers (IENs) where the VistA Imaging file references exists.
