TIU DELETE RECORD (98)
Name Value
NAMETIU DELETE RECORD
TAGDELETE
ROUTINETIUSRVP
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYSUBSCRIPTION
DESCRIPTION
Deletes TIU Document records...Evaluates authorization.
INPUT PARAMETER
  1. INPUT PARAMETER:
    TIUDA
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    DESCRIPTION:
    Record number of TIU Document to be deleted.
  2. INPUT PARAMETER:
    TIURSN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This optional parameter specifies the reason for deletion (i.e., Privacy
Act, or Administrative Action). It only needs to be passed if the document 
has already been signed, and the user is still authorized to delete the
record.
RETURN PARAMETER DESCRIPTION
Returns error message with ERR=1^Explanation text if the user is NOT
authorized to delete the named record (e.g., it's his, but signed; or it's
not his, and he better keep his paws off it).
