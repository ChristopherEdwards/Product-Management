ORQQCN GET CONSULT (486)

Name Value
NAMEORQQCN GET CONSULT
TAGGETCSLT
ROUTINEORQQCN1
RETURN VALUE TYPEARRAY
AVAILABILITYSUBSCRIPTION
DESCRIPTION
Given a Consult ID from file 123, return the zero node to the client for
loading into a consult record in RESULTS[0].  If the consult has any
associated TIU records (completion, addenda) these will be returned in
RESULTS[i..j].
INPUT PARAMETER
  1. INPUT PARAMETER:
    Consult ID
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    REQUIRED:
    YES
    DESCRIPTION:
    Consult IEN from file #123, REQUEST/CONSULT.
RETURN PARAMETER DESCRIPTION
'-1^error message' if unsuccessful
Zero node from ^GMR(123 if successful as RESULTS[0], and any associated
TIU records in RESULTS[i..j]
