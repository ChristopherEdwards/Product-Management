OOPS GET PRT ACC STATUS RPT (1422)

Name Value
NAMEOOPS GET PRT ACC STATUS RPT
TAGACCID
ROUTINEOOPSGUIT
RETURN VALUE TYPEGLOBAL ARRAY
WORD WRAP ONTRUE
DESCRIPTION
This broker call retrieves the data required to generate the PRINT ACCIDENT
REPORT STATUS Report.
INPUT PARAMETER
  1. INPUT PARAMETER:
    INPUT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    256
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This input parameter consists of the Start Date, End Date, and Station that 
the report should be run.  The Date/Time of Incident (field #4) is used in the
date criteria and the Station (field #13) is used for the Station.  The format 
is STARTDATE^ENDDATE^STATION IEN.
  2. INPUT PARAMETER:
    CALL
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    10
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This input parameter contains the letter of the calling menu.  This will 
enable the name and SSN to be screened if the calling menu is the Union.
RETURN PARAMETER DESCRIPTION
This broker call returns the ASISTS data to produce the PRINT ACCIDENT STATUS
Report.
