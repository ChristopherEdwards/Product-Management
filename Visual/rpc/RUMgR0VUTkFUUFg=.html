EC GETNATPX (921)

Name Value
NAMEEC GETNATPX
TAGECNATPX
ROUTINEECUMRPC
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYAGREEMENT
WORD WRAP ONTRUE
DESCRIPTION
Returns an array of active, inactive or both of Event Capture national and 
local Procedures from file #725.
INPUT PARAMETER
  1. INPUT PARAMETER:
    ECARY
    PARAMETER TYPE:
    LITERAL
    DESCRIPTION:
    Input variable ECARY contains the following subscripted elements
   ECPX - Procedures to output, L- local, N- National, B- Both
   STAT - Active or inactive EC Nat Codes
          A-ctive (default), I-nactive, B-oth

If not valued is passed in defaults to Local and Active.
RETURN PARAMETER DESCRIPTION
Returns an array of EC local/national procedures with the following pieces of 
dat separated by a up-arrow.
  1.  File #725 IEN
  2.  Name National Number
  3.  Inactive Date
  4.  Synonym
  5.  CPT ien
  6.  CPT code
  7.  CPT Short Name
