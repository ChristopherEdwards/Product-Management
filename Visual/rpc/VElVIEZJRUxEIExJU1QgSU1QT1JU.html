TIU FIELD LIST IMPORT (1108)

Name Value
NAMETIU FIELD LIST IMPORT
TAGLIMPORT
ROUTINETIUSRVF1
RETURN VALUE TYPEARRAY
DESCRIPTION
Calls the import process for a pre-loaded (into ^TMP) list of template
fields.
RETURN PARAMETER DESCRIPTION
Returns a list of field names imported in the format NAME^RCODE^RENAME, 
where NAME is the desired field name, RCODE is 1 or 0 if the field
was added/same or renamed (respectively), and RENAME is the actual
saved name of the field if RCODE is 0.
