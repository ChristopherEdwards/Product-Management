ORPRF TRIGGER POPUP (2170)
Name Value
NAMEORPRF TRIGGER POPUP
TAGTRIGRPOP
ROUTINEORPRF
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
Returns 1 if popup flag display should be triggered for given patient upon
patient selection. If not, returns 0. Does not require clean-up after
calling it since it does not set arrays or globals.
INPUT PARAMETER
  1. INPUT PARAMETER:
    PTDFN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    DESCRIPTION:
    IEN in Patient file of given patient.
RETURN PARAMETER DESCRIPTION
Returns 1 if popup flag display should be triggered for given patient upon
patient selection. If not, returns 0.
