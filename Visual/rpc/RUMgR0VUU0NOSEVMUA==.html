Name Value

RETURN VALUE TYPEGLOBAL ARRAY

DESCRIPTION

INPUT PARAMETER
  1. INPUT PARAMETER:
    ECARY
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    DESCRIPTION:
    Input variable ECARY contains the Help Frame Name.
RETURN PARAMETER DESCRIPTION
Returns an array with help text from the HELP FRAM File (#9.2).
