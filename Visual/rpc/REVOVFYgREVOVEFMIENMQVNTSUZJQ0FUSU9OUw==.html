Name Value

ROUTINEDENTVRP1
RETURN VALUE TYPEARRAY
AVAILABILITYAGREEMENT
DESCRIPTION
RPC to return dental classifications from file 220.2
RETURN PARAMETER DESCRIPTION
This returns the dental classifications from file 220.2 in an array in the
following format:
   array(#) = p1^p2^p3  where
    p1 := ien to file 220.2
    p2 := dental classification name
    p3 := I or O
          where I indicates an inpatient classification
                O indicates an outpatient classification
