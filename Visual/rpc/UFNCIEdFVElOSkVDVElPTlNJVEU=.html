PSB GETINJECTIONSITE (3029)

Name Value
NAMEPSB GETINJECTIONSITE
TAGRPC
ROUTINEPSBINJEC
RETURN VALUE TYPEARRAY
AVAILABILITYPUBLIC
INACTIVEACTIVE
WORD WRAP ONTRUE
VERSION2
DESCRIPTION
Get the last nn injections site info from the BCMA MEDICATION LOG file 
#53.9.   Get only of specific Orderable Items per patient in reverse
chronology date/time order or All Orderable Items per patient in 
reverse chronology date/time order.
INPUT PARAMETER
  1. INPUT PARAMETER:
    DFN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    Patient IEN number.
  2. INPUT PARAMETER:
    TIME
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    A time range, in hours, to search backwards through the BCMA MEDICATION 
LOG file #53.79 for injection site data.
  3. INPUT PARAMETER:
    MAX
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    4
    DESCRIPTION:
    The maximum number of injection site items to return within the Time 
Range specified.
  4. INPUT PARAMETER:
    PSBOI
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    Orderable Item IEN.
RETURN PARAMETER DESCRIPTION
 Results[0] = Number of Nodes.
 Results[n]= ^1  Action Date/Time (Fileman format)
             ^2  Orderable Item IEN
             ^3  Orderable Item Name
             ^4  Injection Site
 
 Results[1]= ^1  -1
             ^2  Error text message, i.e. <<No data to display>>
