OOPS SIGN CA7 (1971)

Name Value
NAMEOOPS SIGN CA7
TAGSIGNCA7
ROUTINEOOPSGUIS
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
This broker call accepts an encrypted electronic signature, validates the 
entry and files the signature fields in the ASISTS COMPENSATION CLAIM (CA7) 
File (#2264).
INPUT PARAMETER
  1. INPUT PARAMETER:
    INPUT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This parameter contains the file number, field number and internal record
number for verifying and filing the electronic signature.  The expected format
is:  FILE^FIELD^IEN.  An Example being 2264^84^5.
  2. INPUT PARAMETER:
    SIGN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This parameter contains the encrypted electronic signature code.
RETURN PARAMETER DESCRIPTION
The return parameter, contains a message indicating the success of filing
the electronic signature fields.
