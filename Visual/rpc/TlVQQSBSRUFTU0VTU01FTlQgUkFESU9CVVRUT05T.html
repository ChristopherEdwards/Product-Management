NUPA REASSESSMENT RADIOBUTTONS (2967)
Name Value
NAMENUPA REASSESSMENT RADIOBUTTONS
TAGRAOK
ROUTINENUPABCL2
RETURN VALUE TYPESINGLE VALUE
WORD WRAP ONTRUE
DESCRIPTION
Determines if the nurse can do a partial reassessment.  If response is 
true, these radiobuttons appear on the initial reassessment screen.
 
1. Medical/Surgical update reassessment (full reassessment completed
previously on current shift)
 
2. Critical Care update reassessment (full reassessment completed
previously on current shift
 
3. Mental Health update reassessment (full reassessment completed
previously on current shift)

    1
    DESCRIPTION:
    Patient's DFN.
RETURN PARAMETER DESCRIPTION
True or False.
