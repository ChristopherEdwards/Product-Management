DSIROIC WPFILER WP FILER (2977)

Name Value
NAMEDSIROIC WPFILER WP FILER
TAGWPFILER
ROUTINEDSIROIC
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
This RPC is used to update comments is several places withing the Release 
of Information Record Manager (ROI/DSIR)
INPUT PARAMETER
  1. INPUT PARAMETER:
    IEN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    20
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    The IEN (Internal Entry Number) is the location within the file where 
the data is to be filed.
  2. INPUT PARAMETER:
    TYPE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    1
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
     TYPE    - Must be "C", "I", "A", OR "R" as follows:
           Regular comments (19620,.31)(C)
           Internal Comments (19620,.32)(I) 
           Annotations (19620.98,100)(A)
           Released Document Comment (19620.1,1)(R)
  3. INPUT PARAMETER:
    STRLIST
    PARAMETER TYPE:
    LIST
    MAXIMUM DATA LENGTH:
    9999
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    STRLIST - List (ARRAY) of lines to be filed
RETURN PARAMETER DESCRIPTION
Retuns
  IEN if successful
  -1^error message if not successful
