ORQQCN ADDCMT (485)
Name Value
NAMEORQQCN ADDCMT
TAGCMT
ROUTINEORQQCN2
RETURN VALUE TYPESINGLE VALUE

DESCRIPTION
Allows addition of a comment to a consult request/consult without changing
its status. Optionally, allows sending of an alert to the requesting
provider and others.
INPUT PARAMETER
  1. INPUT PARAMETER:
    Consult ID
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    Internal entry number of the consult in file 123, REQUEST/CONSULT.
  2. INPUT PARAMETER:
    Comments
    PARAMETER TYPE:
    LIST
    MAXIMUM DATA LENGTH:
    80
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    Array of comments in the form ARRAY(0)..ARRAY(n).
  3. INPUT PARAMETER:
    Alert
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    5
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    If '1', send an alert to the requesting provider.  Otherwise, don't.
  4. INPUT PARAMETER:
    Alert to
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    80
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    4
    DESCRIPTION:
    Contains pointers to NEW PERSON FILE (200) for those to receive alert for
this addition of a comment.
RETURN PARAMETER DESCRIPTION
0 if successful, '1^error message' if unsuccessful
