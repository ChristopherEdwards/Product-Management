MAGV CREATE WORK ITEM (3101)

Name Value
NAMEMAGV CREATE WORK ITEM
TAGCRTITEM
ROUTINEMAGVIM01
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYRESTRICTED
DESCRIPTION
This RPC creates work item entries in the MAG WORK ITEM file (#2006.941).
INPUT PARAMETER
  1. INPUT PARAMETER:
    TYPE
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the type of WORK ITEM created (i.e. DICOM Correct).
  2. INPUT PARAMETER:
    SUBTYPE
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is the subtype of a work item.
  3. INPUT PARAMETER:
    STATUS
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    This is the initial status given to a work item.
  4. INPUT PARAMETER:
    PLACEID
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    4
    DESCRIPTION:
    This is the place ID number of a work item.
  5. INPUT PARAMETER:
    PRIORITY
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    5
    DESCRIPTION:
    This is the numeric priority given to a work item.
  6. INPUT PARAMETER:
    MSGTAGS
    PARAMETER TYPE:
    LIST
    SEQUENCE NUMBER:
    6
    DESCRIPTION:
    This is an array of tag names and values assigned to a work item. Tag
names and values may be used to look up a work item.
  7. INPUT PARAMETER:
    CRTUSER
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    7
    DESCRIPTION:
    This is the user that created the work item.
  8. INPUT PARAMETER:
    CRTAPP
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    8
    DESCRIPTION:
    This is the application that created the work item.
RETURN PARAMETER DESCRIPTION
The return parameter returns a "0" for success. n < 0 followed by an error message is returned if an error has occurred.
