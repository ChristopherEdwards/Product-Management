YTRP INSTRUMENT REPORT (1226)

Name Value
NAMEYTRP INSTRUMENT REPORT
TAGINTRMNT
ROUTINEYTRPWRP
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYPUBLIC
WORD WRAP ONTRUE
DESCRIPTION
This remote procedure allows staff to print out psychological tests and
interview reports.
INPUT PARAMETER
  1. INPUT PARAMETER:
    YSDFN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    YSDFN is a pointer to the Patient file #2.
  2. INPUT PARAMETER:
    YSXT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    60
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This varible contains "Test Completion Date,Name of the test", for example
3000721,223.
The Name of the test and Completion Date are field 0.1 and field 1 of the
Psych Instrument Patient file #601.2.
RETURN PARAMETER DESCRIPTION
RESULT returns the test text report for a selected patient and a given
test completion date.
