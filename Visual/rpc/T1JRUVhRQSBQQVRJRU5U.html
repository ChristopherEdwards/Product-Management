ORQQXQA PATIENT (169)

Name Value
NAMEORQQXQA PATIENT
TAGPATIENT
ROUTINEORQQXQA
RETURN VALUE TYPEARRAY
AVAILABILITYSUBSCRIPTION
DESCRIPTION
Function returns a list of notifications for a patient for the current user.
INPUT PARAMETER
  1. INPUT PARAMETER:
    PATIENT ID
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    REQUIRED:
    YES
    DESCRIPTION:
    The record number of the patient from the Patient File (#2).
  2. INPUT PARAMETER:
    START DATE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    DESCRIPTION:
    Start date for retrieving patient notifications in Fileman format.
  3. INPUT PARAMETER:
    STOP DATE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    DESCRIPTION:
    Stop date for retrieving patient notifications in Fileman format.
RETURN PARAMETER DESCRIPTION
Array of patient notifications delimited by "^".  Returned data includes
alert id (XQAID)^alert message^patient id (DFN).
