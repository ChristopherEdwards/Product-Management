DSIR GET CLERK REQUESTS (2882)
Name Value
NAMEDSIR GET CLERK REQUESTS
TAGGETTODAY
ROUTINEDSIROI
RETURN VALUE TYPEGLOBAL ARRAY
WORD WRAP ONTRUE
DESCRIPTION
This RPC returns all open/pending requests for a given clerk.  This is 
used to populate the today screen.
INPUT PARAMETER
  1. INPUT PARAMETER:
    CLERK
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    The IEN for file 2 which points at the desired ROI clerk.
  2. INPUT PARAMETER:
    BILLING
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    1
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    BILLING - The billing system is on (1) or off(0)
RETURN PARAMETER DESCRIPTION
 ; Return: Global Array of:
 ;  PATFOIA^RECEIVED^STATCODE^SSN^IEN^DFN
 ;
 ;  PATFOIA  - Patient Name/FOIA
 ;  RECEIVED - External Date Entered
 ;  STATCODE - Status code (numeric)
 ;  SSN
 ;  IEN      - Instance IEN
 ;  DFN      - Patient/FOIA DFN
 ;  INTCMNTS - INTERNAL COMMENTS 0=FALSE 1=TRUE
 ;  DISCALRT - Disclosure Alert
 ;  BILLFLAG - The requestor has outstanding bills
