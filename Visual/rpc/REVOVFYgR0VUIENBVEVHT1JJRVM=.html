DENTV GET CATEGORIES (1083)

Name Value
NAMEDENTV GET CATEGORIES
TAGCAT
ROUTINEDENTVRP1
RETURN VALUE TYPEARRAY
AVAILABILITYAGREEMENT
DESCRIPTION
This rpc will return the different categories (fields 5 & 6) in file 228
INPUT PARAMETER
  1. INPUT PARAMETER:
    TYP
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is optional.  If passed it should be either 1, 2, or 12
If it is not passed, then this call will default to 12
 For file 228,
  If TYP=1, then only return the subcategory-1 names (field 5)
  If TYP=2, then only return the subcategory-2 names (field 6)
  If TYP=12, then return both subcategories' names
RETURN PARAMETER DESCRIPTION
An array is returned (in alphabetical order) of the various subcategory
values found in file 228 (fields 5 and 6)
   array(#) = subcategory name
