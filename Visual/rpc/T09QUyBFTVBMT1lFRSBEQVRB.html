OOPS EMPLOYEE DATA (1386)

Name Value
NAMEOOPS EMPLOYEE DATA
TAGPAID
ROUTINEOOPSGUI4
RETURN VALUE TYPEARRAY
DESCRIPTION
This call returns a listing of individuals in the PAID file #450 that match
the Name or partial name entered.  It returns PAID fields 6, 10, 13, 16, 31,
32, 38, 186.1, 186.3, 186, 186.4, 458, 604.
INPUT PARAMETER
  1. INPUT PARAMETER:
    NAME
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    256
    DESCRIPTION:
    This parameter contains the name or partial name of the employee being looked
up.
RETURN PARAMETER DESCRIPTION
This return parameter, RESULTS, contains fields from the PAID Employee File
(#450) that are needed in the ASISTS Accident Report File (#2260).  An example
of this data is as follows:

AM (CO^13^NURSING ASSISTANT^0621^M^JUN 27, 1963^4^^^VA^^80120045^^ANNUAL^45000
^OTHER^WG^NURSING ASSISTANT^INFORMATION RESOURCES MGMT
