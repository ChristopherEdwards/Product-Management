OOPS DELETE WITNESS (1408)

Name Value
NAMEOOPS DELETE WITNESS
TAGDELWITN
ROUTINEOOPSGUI2
RETURN VALUE TYPEARRAY
AVAILABILITYPUBLIC
WORD WRAP ONFALSE
DESCRIPTION
This broker call will delete witness information from the "CA1W" node in 
file 2260 based on the IEN and sub file id passed in.
INPUT PARAMETER
  1. INPUT PARAMETER:
    INPUT
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    DESCRIPTION:
    This required input parameter must contain the IEN and record number of the
subfile for witness information to be deleted. The input format is IEN^SUB.
  2. INPUT PARAMETER:
    OOPS CHANGE CAS
RETURN PARAMETER DESCRIPTION
The return array will contain a status message for the deletion process and
whether it was successful (SUCCESSFULLY DELETED) or not (DELETION FAILED)
sucessful.
