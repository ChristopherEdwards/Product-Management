XWB DEFERRED CLEAR (439)
Name Value
NAMEXWB DEFERRED CLEAR
TAGCLEAR
ROUTINEXWBDRPC
RETURN VALUE TYPEARRAY
AVAILABILITYPUBLIC
VERSION1
DESCRIPTION
This RPC is used to CLEAR the data under a handle in the ^XTMP global.
INPUT PARAMETER
  1. INPUT PARAMETER:
    HANDLE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the HANDLE from the XWB DEFERRED RPC.
RETURN PARAMETER DESCRIPTION
The return value is always an array.  The first node of the array is equal
to 1.
