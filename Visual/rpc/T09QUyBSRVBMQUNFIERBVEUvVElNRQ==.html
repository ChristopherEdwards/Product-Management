OOPS REPLACE DATE/TIME (1389)
Name Value
NAMEOOPS REPLACE DATE/TIME
TAGDTFC
ROUTINEOOPSGUI8
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
This call assures the date and/or time is a valid and reformats it.
INPUT PARAMETER
  1. INPUT PARAMETER:
    DATE
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the date to be validated and reformatted.
  2. INPUT PARAMETER:
    FLAG
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is a control flag for assuring a valid date.
RETURN PARAMETER DESCRIPTION
The return parameter, RESULTS, either returns a data in a valid Fileman
format or returns an error message indicating that the date passed in the 
broker call was invalid and there was a "DATE ERROR".
