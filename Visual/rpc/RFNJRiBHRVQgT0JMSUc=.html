DSIF GET OBLIG (2587)

Name Value
NAMEDSIF GET OBLIG
TAGGETOBLIG
ROUTINEDSIFBAT5
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYAGREEMENT
WORD WRAP ONTRUE
VERSION3.2
DESCRIPTION
Used to return a list of obligations based upon Control point and number 
of years past to return.  Example, control point 038 with 2 years, will 
return all obligations from control point 038 from today - two years 
to today
 
on error returns -1^error message

  1. INPUT PARAMETER:
    FUND CONTROL POINT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    99
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    Pass in the control point used to return current obligations.
  2. INPUT PARAMETER:
    FBYEAR
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    4
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    Years to search for existing obligations, default is current year and 
last year.  If you input 3 it will display current year and go back 3 
years.
