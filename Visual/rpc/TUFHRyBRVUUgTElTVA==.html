MAGG QUE LIST (981)
Name Value
NAMEMAGG QUE LIST
TAGQUELIST
ROUTINEMAGGTU7
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYRESTRICTED
DESCRIPTION
Queues a list of images to be copied from jukebox to harddrive.
INPUT PARAMETER
  1. INPUT PARAMETER:
    CODE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    5
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
      A Code containing A | F | B
     if [ A  then Abstracts will be queued.
     if [ F  then Full Resolution image will be queued.
     if [ B  then BIG file ( Xray 2k x 2k ) will be queued.
  2. INPUT PARAMETER:
    LIST
    PARAMETER TYPE:
    LIST
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    The list of Image IEN's to be queued.
RETURN PARAMETER DESCRIPTION
Standard Imaging return variable  1 = success  0 = error
 1^message 
 0^error message
