DSIR GET REQUESTORS (2429)

Name Value
NAMEDSIR GET REQUESTORS
TAGGTREQSTR
ROUTINEDSIROI3
RETURN VALUE TYPEGLOBAL ARRAY
WORD WRAP ONTRUE
DESCRIPTION
THIS RPC WILL RETURN ALL REQUESTORS THAT MEET ONE OF THE FOLLOWING 
CRITERIA:
 
THE REQUESTORS LAST NAME STARTS WITH THE CHARACTERS IN THE FIRST INPUT 
PARAMETER AND THE SECOND PARAMETER IS AN "L"
 
OR 
 
THE REQUESTORS CORPORATE NAME STARTS WITH THE CHARACTERS IN THE FIRST
INPUT PARAMETER AND THE SECOND PARAMETER IS A "C"L"
INPUT PARAMETER
  1. INPUT PARAMETER:
    PREF
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    PREF - FIRST PART OF THE REQUESTOR NAME(S) BEING SEARCHED FOR
  2. INPUT PARAMETER:
    STYP
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    1
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    STYP - "L" - USE THE "AC" INDEX FOR LAST NAME  
       "C" - USE THE "AD" INDEX FOR CORPORATE NAME
RETURN PARAMETER DESCRIPTION
RETURNS:
 -1^NO DATA FOUND 
 OR 
 FULL_NAME^CORPORATE_NAME^CITY^STATE^PHONE^IEN^PATIENT POINTER
