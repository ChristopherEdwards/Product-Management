ORWRA DEFAULT EXAM SETTINGS (654)
Name Value
NAMEORWRA DEFAULT EXAM SETTINGS
TAGGETDEF
ROUTINEORWRA
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYRESTRICTED
DESCRIPTION
This RPC returns the default settings for the display of imaging exams on
the reports tab.
RETURN PARAMETER DESCRIPTION
The format of the returned string is as follows (pieces separated by '^')
 
  Piece         Description
  -----         -----------
    1           Beginning date for search
    2           End date for search
    3           Maximum number of exams
