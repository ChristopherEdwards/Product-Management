OOPS LIST CAS (1968)
Name Value
NAMEOOPS LIST CAS
TAGLISTCA
ROUTINEOOPSGUIC
RETURN VALUE TYPEGLOBAL ARRAY
DESCRIPTION
This broker call returns a listing of all the valid CA's (either 1 or 2) that
can be selected by this user to create a new Request for Compensation (CA-7)
claim.
INPUT PARAMETER
  1. INPUT PARAMETER:
    INPUT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    60
    REQUIRED:
    YES
    DESCRIPTION:
    This parameter passes in 3 pieces of information:  the name or SSN to be used
in building the selection list, the file number to be used for looking up, and
the calling menu.
RETURN PARAMETER DESCRIPTION
This return parameter returns a list of valid CA-1's and CA-2's containing
the claim number, name and date of incident.  Additional fields that are
pulled back to be used as default values are:  SSN, PAY AMOUNT, PAY PERIOD,
GRADE, STEP, RETIREMENT, HEALTH BENEFIT CODE, LIFE INSURANCE.
