MAGV INACTIVATE SOP (3145)
Name Value
NAMEMAGV INACTIVATE SOP
TAGINSOP
ROUTINEMAGVRS24
RETURN VALUE TYPEARRAY
AVAILABILITYRESTRICTED
DESCRIPTION
This RPC inactivates a SOP INSTANCE entry in the IMAGE SOP INSTANCE file (#2005.64).
INPUT PARAMETER
  1. INPUT PARAMETER:
    SOPIEN
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    The IEN of the SOP entry to be inactivated.
  2. INPUT PARAMETER:
    SERIEN
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    The IEN of the related series IEN.
  3. INPUT PARAMETER:
    OVERRIDE
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    If set to 1 this flag prevents the checking of the parent procedure
reference prior to inactivation.
RETURN PARAMETER DESCRIPTION
This parameter returns a 0 for success or a negative value and an error
message.
