RPC Broker entry point for classification help.
INPUT PARAMETER
  1. INPUT PARAMETER:
    ECARY
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    DESCRIPTION:
    Input key ECARY consist of the following pieces:-
ECDFN  - Patient DFN from file (#2)
ECKY   - Key to provide help on.  Example ECKY='SC' provides help on service
         connection.

Array of help text for classification.
