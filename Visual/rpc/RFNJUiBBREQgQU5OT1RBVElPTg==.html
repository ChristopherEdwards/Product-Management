DSIR ADD ANNOTATION (2081)

Name Value
NAMEDSIR ADD ANNOTATION
TAGADDANNO
ROUTINEDSIROI
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
This RPC adds entries into file 19620.98 which is for internal 
annotations on patients pertinent to the ROI department.
INPUT PARAMETER
  1. INPUT PARAMETER:
    PATIENT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    999
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the pointer to either the Patient File #2 or the DSIR 
Non-Computerized Patient file #19620.96.
 
Example values:
 
        2345;DPT(
        23111;DSIR(19620.96,
RETURN PARAMETER DESCRIPTION
-1^Missing/Invalid Patient Pointer!
-2^Unable to add record!
NNNN - File Entry Number of added record in 19620.98
