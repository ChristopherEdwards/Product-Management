NUPA DLOOK (2945)

Name Value
NAMENUPA DLOOK
TAGDLOOK
ROUTINENUPABCL
RETURN VALUE TYPEGLOBAL ARRAY
WORD WRAP ONTRUE
DESCRIPTION
Does a lookup on the B Xref of a file, and does not convert dates or
pointers.
INPUT PARAMETER
  1. INPUT PARAMETER:
    FILE
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    File number to do the lookup on.
  2. INPUT PARAMETER:
    VAL
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    Value to lookup.
  3. INPUT PARAMETER:
    TYPE
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    Type of file to lookup - variable pointer or date.
  4. INPUT PARAMETER:
    SCR
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    4
    DESCRIPTION:
    Fileman screen for the lookup.
  5. INPUT PARAMETER:
    IND
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    5
    DESCRIPTION:
    Index to lookup the data on.
RETURN PARAMETER DESCRIPTION
Output is returned in ^TMP($J) and varies depending on the fields 
selected.
