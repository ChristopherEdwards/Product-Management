EC GETPRODEFS (934)
Name Value
NAMEEC GETPRODEFS
TAGPRDEFS
ROUTINEECUERPC
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYRESTRICTED
WORD WRAP ONTRUE
DESCRIPTION
This broker entry point returns the defaults for procedure data entry.
INPUT PARAMETER
  1. INPUT PARAMETER:
    ECARY
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    DESCRIPTION:
    Input value, ECARY, contains the following values separated by "^"
     ECL  - Location IEN
     ECD  - DSS Unit IEN
     ECC  - Category IEN
     ECP  - Procedure IEN
RETURN PARAMETER DESCRIPTION
Output value, RESULTS, contain the data pieces as follows:-
    1  - Associated Clinic IEN
    2  - Associated Clinic
    3  - Medical Speciality IEN
    4  - Medical Speciality
