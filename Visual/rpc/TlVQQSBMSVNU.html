NUPA LIST (2946)
Name Value
NAMENUPA LIST
TAGLIST
ROUTINENUPABCL
RETURN VALUE TYPEGLOBAL ARRAY
WORD WRAP ONTRUE
DESCRIPTION
Returns a list of all entries in a file (.01 field only).  You can specify
a screen, in regular Fileman format.
INPUT PARAMETER
  1. INPUT PARAMETER:
    FILE
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    File number to list data from.
  2. INPUT PARAMETER:
    SCREEN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    Fileman screen for the lookup.
  3. INPUT PARAMETER:
    MULT
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    Subscript of a multiple to do the lookup on.
RETURN PARAMETER DESCRIPTION
Returns .01 filed, plus the IEN.  This iexample is from file 1927.2.
