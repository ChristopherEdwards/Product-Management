GMV GET VITAL TYPE IEN (1575)
Name Value
NAMEGMV GET VITAL TYPE IEN
TAGTYPE
ROUTINEGMVUTL8
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYSUBSCRIPTION
INACTIVEACTIVE
DESCRIPTION
Returns the IEN if the value is found in the GMRV VITAL TYPE (#120.51)
file.
 
This remote procedure call is documented in Integration Agreement 4357.
INPUT PARAMETER
  1. INPUT PARAMETER:
    GMVTYPE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    55
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    GMVTYPE = Name of Vital Type (from FILE 120.51) (e.g., WEIGHT)
RETURN PARAMETER DESCRIPTION
Returns the IEN if GMVTYPE exists in FILE 120.51.
 
Example:
 > S GMVTYPE="WEIGHT"
 > D TYPE^GMVUTL8(.RESULT,GMVTYPE) ZW RESULT
 > RESULT=9
