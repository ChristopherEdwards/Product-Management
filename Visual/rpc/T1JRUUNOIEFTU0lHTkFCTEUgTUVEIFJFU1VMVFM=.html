ORQQCN ASSIGNABLE MED RESULTS (775)

Name Value
NAMEORQQCN ASSIGNABLE MED RESULTS
TAGGETMED
ROUTINEORQQCN3
RETURN VALUE TYPEARRAY
DESCRIPTION
Returns a list of medicine results that can be attached to a procedure.
INPUT PARAMETER
  1. INPUT PARAMETER:
    CONSULT IEN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    REQUIRED:
    YES
    DESCRIPTION:
    Consult (procedure) IEN in file 123.
RETURN PARAMETER DESCRIPTION
List of medicine results in the format:
 
19;MCAR(691.5,^ELECTROCARDIOGRAM^JUN 30,1999^ABNORMAL
 
where piece 1 = variable pointer to result
            2 = naem of result
            3 = date of result
            4 = interpretation of result
