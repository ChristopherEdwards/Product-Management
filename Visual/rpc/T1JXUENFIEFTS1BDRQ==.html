ORWPCE ASKPCE (702)
Name Value
NAMEORWPCE ASKPCE
TAGASKPCE
ROUTINEORWPCE2
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYRESTRICTED
WORD WRAP ONTRUE
DESCRIPTION
Returns the value of the ORWPCE ASK ENCOUNTER UPDATE parameter.
INPUT PARAMETER
  1. INPUT PARAMETER:
    USER
    PARAMETER TYPE:
    LITERAL
    DESCRIPTION:
    The user to get the parameter for.
  2. INPUT PARAMETER:
    LOC
    PARAMETER TYPE:
    LITERAL
    DESCRIPTION:
    The Location to use when getting the parameter.
RETURN PARAMETER DESCRIPTION
The 0-3 value stored in the ORWPCE ASK ENCOUNTER UPDATE parameter.

 0 = User is the Primary Encounter Provider, and Encounter Data is Needed
 1 = User is the Primary Encounter Provider
 2 = Encounter Data is Needed
 3 = Always
