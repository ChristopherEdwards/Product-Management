DENTV GET PROVIDER SPECIALTIES (1928)
Name Value
NAMEDENTV GET PROVIDER SPECIALTIES
TAGSPEC
ROUTINEDENTVRP5
RETURN VALUE TYPEARRAY
AVAILABILITYAGREEMENT
DESCRIPTION
Gets a list of all active Provider Specialties from file 220.52.
RETURN PARAMETER DESCRIPTION
Returns an array List[n] := Provider Specialty info.
Returns List[1] := -1^error message if there are no active specialties.
The format of the return array is List[n] := p1^p2 where
  p1:=Specialty name
  p2:=Code (numeric)
