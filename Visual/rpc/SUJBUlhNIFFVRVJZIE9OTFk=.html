IBARXM QUERY ONLY (769)
Name Value
NAMEIBARXM QUERY ONLY
TAGRQUERY
ROUTINEIBARXMR
RETURN VALUE TYPEARRAY
AVAILABILITYRESTRICTED
INACTIVEACTIVE
DESCRIPTION
This RPC is used to query only the information for pharmacy co-payment
billing that has happened for the given month/year.
INPUT PARAMETER
  1. INPUT PARAMETER:
    IBICN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    40
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the patient's ICN for reference to any VistA database
  2. INPUT PARAMETER:
    IBM
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    7
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is in fileman format the month/year for which the request is being made.
RETURN PARAMETER DESCRIPTION
The return array is in file 354.71 format.
