DSIC PX PRIMARY (1635)
Name Value
NAMEDSIC PX PRIMARY
TAGPCE
ROUTINEDSICPX2
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYAGREEMENT
DESCRIPTION
For a VISIT ien this will return the primary PCE diagnosis and the 
primary PCE provider.
INPUT PARAMETER
  1. INPUT PARAMETER:
    IEN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the ien to the VISIT file (#9000010).
RETURN PARAMETER DESCRIPTION
Return p1^p2^p3^p4  where
  p1 = primary pce diagnosis short description
  p2 = primary pce diagnosis code
  p3 = primary pce provider duz
  p4 = primary pce provider name (first m last format)
 
If problems, return -1^message
 
See DSICPX2 routine, this can also be called as an extrinsic function.  
In that case the return value is (all values are for primary pce:
  icd9 ien ^ icd9 code ^ icd9 short desc ^ duz ^ name
