EC GETPATCLASTAT (924)

Name Value
NAMEEC GETPATCLASTAT
TAGPATCLAST
ROUTINEECUERPC1
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYRESTRICTED
WORD WRAP ONTRUE
DESCRIPTION
Returns a patient in/out status and classification. Classifications are:
Agent Orange, Ionizing Radiation, SC Condition, Environmental Contaminants and
Military Sexual Trauma.
INPUT PARAMETER
  1. INPUT PARAMETER:
    ECARY
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    DESCRIPTION:
    Input variable ECARY contains the following values separated by up-arrow.
  1. ECDFN - Patient ien (#2)
  2. ECD   - DSS Unit ien (#724)
  3. ECDT  - Procedure date and time (fileman format)
RETURN PARAMETER DESCRIPTION
Returns a single line with values for a patient status and classifications.
Data is delimited by (^) and '~'. Values after the '~' refers to those 
classifications that must be asked by Delphi application when the answer to 
SC=No. Pieces delimited by '^' are:-
  1.   Patient Status: I for inpatient or O for outpatient
  2-6. Classification: 2- Agent Orange, 3- Ionizing Radiation
                       4- SC Condition, 5- Environmental Contaminants
                       6- Military Sexual Trauma
  Data delimited by '~' follows those of '^'. Pieces as follows:-
  1.   Agent Orange    2. Ionizing Radiation
  3.   Environmental Contaminants
