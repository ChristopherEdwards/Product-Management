DSIF VIEW INVOICE (2665)
Name Value
NAMEDSIF VIEW INVOICE
TAGINDISP
ROUTINEDSIFPAY7
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYAGREEMENT
WORD WRAP ONTRUE
VERSION3.2
DESCRIPTION
An RPC used to display all fields of file 162.5
INPUT PARAMETER
  1. INPUT PARAMETER:
    IEN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
RETURN PARAMETER DESCRIPTION
Returns a list of file 162.5 in the format:
 
field number^internal value;external value
Please note - fields 18,58 and 59 are multiples
Multiple format is: 
    field number^multiple^sub-field number^internal value;external value
            - for displaying ICDs and POAs, we use the following format
    field number^internal ICD;external ICD^internal POA;external POA
            - for displaying line items, we use the subscript "LI"
    field number^line number^line item
