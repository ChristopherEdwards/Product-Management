DENTV DD SECURITY KEY (1074)
Name Value
NAMEDENTV DD SECURITY KEY
TAGKEY
ROUTINEDENTVRP3
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYAGREEMENT
DESCRIPTION
This rpc will check to see if user holds the DENTV EDIT FILE security
keys.
INPUT PARAMETER
  1. INPUT PARAMETER:
    KEY
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    1
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    Optional if KEY="L" (or KEY not passed at all), then check to see if the
user owns the VEJDWPD EDIT LOCAL security key.  If they do, then they can
add cpt codes to file 19600 in the local number space.
 
If KEY="M", then check to see if the user owns the VEJDWPD EDIT FILE
security key.  If so, then they are a VACO representative, and are allowed
to add cpt procedures to file 19600 with iens <100,000
RETURN PARAMETER DESCRIPTION
Returns 1 if user holds the security key
        0 otherwise
