OOPS GET UNION (1400)
Name Value
NAMEOOPS GET UNION
TAGUNIGET
ROUTINEOOPSGUI6
RETURN VALUE TYPEARRAY
DESCRIPTION
This call retrieves a listing of all Unions.
RETURN PARAMETER DESCRIPTION
The 0 node of the ASISTS UNION INFORMATION File (#2263.7) is returned in the 
results array.  This node contains the Union Name, Acronym, and Union 
Representative.

The format of the return parameter is:
       AMERICAN FED OF CIVIL LIBERTIES UNION^AFLCIO^2440^HOWELL,LYNN^2
