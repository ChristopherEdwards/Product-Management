    MAGFROM
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    Source reader (a pointer to NEW PERSON file (#200)) to be cloned.
  2. INPUT PARAMETER:
    MAGTO
    PARAMETER TYPE:
    LIST
    REQUIRED:
    YES
    SEQUENCE NUMBER:

MAGRY = An array with results per cloned reader.
if error

if success
 MAGRY(0) = 1 ^ Number of processed readers
 MAGRY(1..n) = 0|1 ^ Message
               0 failure
               1 success
