Name Value

TAGENC
ROUTINEDENTVRH

DESCRIPTION
This will return the dental encounter data for a given record in file
228.1
INPUT PARAMETER
  1. INPUT PARAMETER:
    IEN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    9
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the internal entry number for file 228.1
RETURN PARAMETER DESCRIPTION
This will return two lines of data:
  RET(1) = p1^p2^p3^...^p15           RET(2) = q1^q2^q3^q4^q5
   p1 = date record created               q1 = DAS record number (date)
   p2 = creator                           q2 = DAS patient category
   p3 = provider                          q3 = DAS bed section
   p4 = encounter date/time               q4 = DAS division
   p5 = location                          q5 = DAS disposition
   p6 = primary pce diagnosis short description
   p7 = primary pce diagnosis code
   p8 = duz of pce primary provider
   p9 = name (1st m last) of pce primary provider
  p10 = service connected (y)
  p11 = agent orange related (y)
  p12 = ionizing radiation (y)
  p13 = environmental contamination related (y)
  p14 = military sexual trauma (y)
  p15 = head & neck cancer (y)
