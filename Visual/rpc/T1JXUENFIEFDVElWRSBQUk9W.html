ORWPCE ACTIVE PROV (521)
Name Value
NAMEORWPCE ACTIVE PROV
TAGACTIVPRV
ROUTINEORWPCE2
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
This calls the PCE API $$ACTIVPRV^PXAPI(provider ien, encounter d/t) to
see if the provider can be stored by PCE.   Returns a 1 if provider is 
good and 0 if the provider is not active or does not have an active 
person class.
INPUT PARAMETER
  1. INPUT PARAMETER:
    PROVIDER IEN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
  2. INPUT PARAMETER:
    ENCOUNTER DATE/TIME
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
RETURN PARAMETER DESCRIPTION
1 if provider is good and 0 if the provider is not active or does not have
an active person class.
