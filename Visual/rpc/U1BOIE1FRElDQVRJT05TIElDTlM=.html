SPN MEDICATIONS ICNS (2828)
Name Value
NAMESPN MEDICATIONS ICNS
TAGCOL
ROUTINESPNJRPDC
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYAGREEMENT
INACTIVEACTIVE
APP PROXY ALLOWEDYes
DESCRIPTION
Returns list of ICNs of all patients prescribed a specific VA Drug Class 
during a given date range.
INPUT PARAMETER
  1. INPUT PARAMETER:
    FDATE
    PARAMETER TYPE:
    LITERAL
    DESCRIPTION:
    Date to begin Medications search
  2. INPUT PARAMETER:
    TDATE
    PARAMETER TYPE:
    LITERAL
    DESCRIPTION:
    Date to end Medications search
  3. INPUT PARAMETER:
    VADC
    PARAMETER TYPE:
    LITERAL
    DESCRIPTION:
    VA Drug Class list (array format)
  4. INPUT PARAMETER:
    ICNLST
    PARAMETER TYPE:
    LITERAL
    DESCRIPTION:
    List of ICNs to search from
