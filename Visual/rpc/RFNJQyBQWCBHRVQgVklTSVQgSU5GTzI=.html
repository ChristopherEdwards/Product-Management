DSIC PX GET VISIT INFO2 (1954)
Name Value
NAMEDSIC PX GET VISIT INFO2
TAGLOOK
ROUTINEDSICPX3
RETURN VALUE TYPEARRAY
AVAILABILITYAGREEMENT
DESCRIPTION
Return all field values for a visit in internal or external format or 
both.
INPUT PARAMETER
  1. INPUT PARAMETER:
    DATA
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    20
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    DATA = p1^p2   where  p1 = visit ien or visit's ID -  required
       p2 = return format
            I := internal values
            E := external values
            B := both internal and external formats - default
RETURN PARAMETER DESCRIPTION
RETURN: DSIC(#) = p1^p2^p3^p4  where
                  p1 = field # from file 9000010
                  p2 = code representing type of data
                  p3 = internal value for field
                  p4 = external value for field
 
If problems return DSIC(1) = -1^message
