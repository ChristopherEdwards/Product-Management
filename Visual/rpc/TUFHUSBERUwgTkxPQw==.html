MAGQ DEL NLOC (2898)
Name Value
NAMEMAGQ DEL NLOC
TAGRLOCA
ROUTINEMAGQBU6A
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYSUBSCRIPTION
DESCRIPTION
This is used to remove Network Location entries
and requires the user to have the MAG SYS key.
INPUT PARAMETER
  1. INPUT PARAMETER:
    Network location file entry
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the Network Location file entry number.
RETURN PARAMETER DESCRIPTION
The value of 0 means the entry could not be deleted.  The value of 1 means the entry was deleted.
