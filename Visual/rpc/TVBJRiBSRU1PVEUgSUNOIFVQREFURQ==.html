MPIF REMOTE ICN UPDATE (1251)
Name Value
NAMEMPIF REMOTE ICN UPDATE
TAGUPDATE
ROUTINEMPIFRPC2
RETURN VALUE TYPESINGLE VALUE
VERSION1
DESCRIPTION
This remote procedure call (RPC) allows the remote update of the 
INTEGRATION CONTROL NUMBER (#991.01), ICN CHECKSUM (#991.02), and 
COORDINATING MASTER OF RECORD (#991.03) fields in the PATIENT (#2) file 
at a specified site. The patient is found based upon social security 
number.
INPUT PARAMETER
  1. INPUT PARAMETER:
    SSN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    9
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    Social Security Number of the patient to be updated.
  2. INPUT PARAMETER:
    ICN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    ICN - without the checksum to be updated in field 991.01
  3. INPUT PARAMETER:
    CHECK
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    6
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    4
    DESCRIPTION:
     CHECKSUM for the patient to be stored in field 991.02
  4. INPUT PARAMETER:
    CMOR
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    3
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    5
    DESCRIPTION:
     Station Number to be stored as the CMOR in field 991.03.
  5. INPUT PARAMETER:
    A08
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    1
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    6
    DESCRIPTION:
    If an A08 message needs to be sent this flag should be set to 1.
RETURN PARAMETER DESCRIPTION
 1 for successful update
 -1^error message if not successfully updated.
