DSIRRPTR CRPT CHECK REPORTS (2982)
Name Value
NAMEDSIRRPTR CRPT CHECK REPORTS
TAGCRPT
ROUTINEDSIRRPTR
RETURN VALUE TYPEGLOBAL ARRAY
WORD WRAP ONTRUE
DESCRIPTION
Retrieve all report information from the DSIR SCHEDULED REPORTS file for 
a given DUZ.
INPUT PARAMETER8994.02, , {}
RETURN PARAMETER DESCRIPTION
 
RETURNS in external format:
 
   IEN^
   CLERK^
   REPORT NAME^
   REPORT STATUS^
   SCHEDULE TIME^
   RUN TIME^
   COMPLETION TIME^
   KILL TIME^
   TASK #
    NOTE ALL TIMES ARE IN DATE TIME FORMAT
  OR
 -1^error message
