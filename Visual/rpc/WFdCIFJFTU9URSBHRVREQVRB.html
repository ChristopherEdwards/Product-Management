XWB REMOTE GETDATA (433)

Name Value
NAMEXWB REMOTE GETDATA
TAGRTNDATA
ROUTINEXWBDRPC
RETURN VALUE TYPEARRAY
AVAILABILITYSUBSCRIPTION
INACTIVEACTIVE
WORD WRAP ONTRUE
VERSION1
DESCRIPTION
This RPC will return an ARRAY with what ever data has been sent back from
the remote site.
INPUT PARAMETER
  1. INPUT PARAMETER:
    HANDLE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    The HANDLE from the XWB REMOTE RPC call. It is used to link the call to
the data.
RETURN PARAMETER DESCRIPTION
 The return value is the array of data.  In the event of an error
condition, the first node of the array is equal to a string with the
syntax "-1^error text".
