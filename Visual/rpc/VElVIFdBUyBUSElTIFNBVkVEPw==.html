TIU WAS THIS SAVED? (428)

Name Value
NAMETIU WAS THIS SAVED?
TAGSAVED
ROUTINETIUSRVP1
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYSUBSCRIPTION
DESCRIPTION
This Boolean Remote Procedure will evaluate whether a given document was
committed to the database, or whether the user who last edited it was
disconnected.
INPUT PARAMETER
  1. INPUT PARAMETER:
    TIUDA
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the Internal Entry Number of the Document in the TIU Document File
(#8925).
RETURN PARAMETER DESCRIPTION
This Boolean result will be returned TRUE (1) if the document was
committed to the database successfully. Otherwise, it will return FALSE,
with an explanatory message (e.g., "0^You appear to have been
disconnected...")
