MAGV GET IMAGE FILE (3121)
Name Value
NAMEMAGV GET IMAGE FILE
TAGGETFILE
ROUTINEMAGVRS31
RETURN VALUE TYPEARRAY
AVAILABILITYRESTRICTED
DESCRIPTION
Returns image attributes given the IEN of the IMAGE INSTANCE FILE (#2005.65).
INPUT PARAMETER
  1. INPUT PARAMETER:
    FILEIEN
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    The IEN of the IMAGE INSTANCE FILE (#2005.65) entry to be retrieved.
  2. INPUT PARAMETER:
    SOPIEN
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    The IEN of the parent IMAGE SOP INSTANCE file (#2005.64) entry.
  3. INPUT PARAMETER:
    OVERRIDE
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    If the OVERRIDE flag is set to 1 the RPC will not check the validity of the
PARENT IEN.
RETURN PARAMETER DESCRIPTION
This RPC returns an array of name-value pairs of patient record
attributes.
