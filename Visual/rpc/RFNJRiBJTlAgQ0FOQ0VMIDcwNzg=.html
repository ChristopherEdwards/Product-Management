DSIF INP CANCEL 7078 (2634)

Name Value
NAMEDSIF INP CANCEL 7078
TAGCAN7078
ROUTINEDSIFNOT2
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
This RPC will mark the 7078 cancelled and if possible remove the Non-VA 
PTF record and make modifications to the 1358.  Logic cloned from 
existing routine FBCHC78.
INPUT PARAMETER
  1. INPUT PARAMETER:
    IEN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    99
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    Pointer to file 162.4 VA FORM 10-7078 (Required)
RETURN PARAMETER DESCRIPTION
Return Values
        -1 ^ Invalid Input!
        -1 ^ FEE BASIS Site Parameters not set!
(0 in first piece means that the 7078 was set to a cancel state but PTF or
1358 problems occured)
        0 ^ Unable to delete Non-VA PTF Record.
        0 ^ Unable to locate reference number on 1358.
        0 ^ 1358 Not available for posting.
        0 ^ No data string passed
        0 ^ Data element in string missing
        0 ^ Invalid Transaction or Transaction previously closed
        0 ^ Authorization amount can't be adjusted.
        0 ^ Unauthorized control point user
        0 ^ Insufficent obligation funds to adjust authorization to post
            bill
        0 ^ Credit bill amount will exceed total bill amount
        0 ^ Unable to create Authorization Line Item
        1 = Everything went fine
