MAGG QUE IMAGE GROUP (979)

Name Value
NAMEMAGG QUE IMAGE GROUP
TAGQUEGROUP
ROUTINEMAGGTU7
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYRESTRICTED
DESCRIPTION
RPC Call to queue all images of an
  Image group for a copy from JukeBox.
  This can also be a PREFETCH, in which case the Images being
  Queued have a lower priority than other copies from JukeBox
INPUT PARAMETER
  1. INPUT PARAMETER:
    MAGIEN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is the IEN of the Image Group in MAG(2005 file.
  2. INPUT PARAMETER:
    CODE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    10
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
            ; CODE is a string code for which images to Queue
        ;     ["A" Abatract
        ;     ["F" Full Resolution
        ;     ["B" Big File
  3. INPUT PARAMETER:
    QCODE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    10
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
     QCODE is a QUEUE code.  If = 1 then this is a prefetch
RETURN PARAMETER DESCRIPTION
'^' delimited string.  Standard Imaging return format 
   i.e. '1 | 0 ^ message '
 - - - - 
 Count of Images Queued ^ message
