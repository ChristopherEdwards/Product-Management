NUPA ORDERS (2957)

Name Value
NAMENUPA ORDERS
TAGAPO
ROUTINENUPABCL2
RETURN VALUE TYPEGLOBAL ARRAY
WORD WRAP ONTRUE
DESCRIPTION
Provides a list of Active/pending orders for the patient.
INPUT PARAMETER
  1. INPUT PARAMETER:
    DFN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    The patient's DFN.
RETURN PARAMETER DESCRIPTION
List of active & pending orders in ^TMP($J).
 
Pieces are:
 
1. Order type
2. Item ordered
3. Date ordered
4. Ordered by
 
^TMP(562303312,"ACTIVITY","AUG 20, 2009",89018868,4)=ACTIVITY^Up in 
Chair^AUG 20, 2009^DOCTOR,ELEVEN
 
^TMP(562303312,"ACTIVITY","AUG 20, 2009",89018869,1)=ACTIVITY^HOB up^AUG 
20, 200 9^DOCTOR,ELEVEN
