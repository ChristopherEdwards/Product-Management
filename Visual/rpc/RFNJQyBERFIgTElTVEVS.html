DSIC DDR LISTER (1587)

Name Value
NAMEDSIC DDR LISTER
TAGLIST
ROUTINEDSICDDR0
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYAGREEMENT
WORD WRAP ONTRUE
DESCRIPTION
This calls the Fileman lister, LIST^DIC to get a list of records matching 
the input value

    REQUIRED:
    YES
    SEQUENCE NUMBER:
