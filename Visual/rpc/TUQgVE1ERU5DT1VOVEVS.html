DESCRIPTION
This remote procedure will return the existing data in an encounter.
INPUT PARAMETER
  1. INPUT PARAMETER:
    STUDY
    PARAMETER TYPE:
    REFERENCE
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the CP Study internal entry number.
RETURN PARAMETER DESCRIPTION
