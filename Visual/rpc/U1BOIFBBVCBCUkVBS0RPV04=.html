SPN PAT BREAKDOWN (2801)
Name Value
NAMESPN PAT BREAKDOWN
TAGCOL
ROUTINESPNJRPC5
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYAGREEMENT
INACTIVEACTIVE
APP PROXY ALLOWEDYes
DESCRIPTION
Returns breakdown of patient statistics for a list of given patients 
(based on the ICN) based on a date range, if provided.
INPUT PARAMETER
  1. INPUT PARAMETER:
    ICNLIST
    PARAMETER TYPE:
    LIST
    DESCRIPTION:
    Patient's ICN in string format (ICN^ICN^ICN^ICN)
  2. INPUT PARAMETER:
    FDATE
    PARAMETER TYPE:
    LITERAL
    DESCRIPTION:
    (Optional) Date to start search from.
  3. INPUT PARAMETER:
    TDATE
    PARAMETER TYPE:
    LITERAL
    DESCRIPTION:
    (Optional) Date to end search at.
RETURN PARAMETER DESCRIPTION
 
Returns data that displays in the SPN Patient Breakdown report.
