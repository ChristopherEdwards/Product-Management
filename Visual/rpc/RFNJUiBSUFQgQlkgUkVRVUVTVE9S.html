DSIR RPT BY REQUESTOR (1661)

Name Value
NAMEDSIR RPT BY REQUESTOR
TAGBYREQ
ROUTINEDSIROIR
RETURN VALUE TYPEGLOBAL ARRAY
WORD WRAP ONTRUE
VERSION5.2
DESCRIPTION
Retrieves ROI requests for a given date range for use in GUI reports.
INPUT PARAMETER
  1. INPUT PARAMETER:
    FROM DATE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    7
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    The FileMan Date to start retrieval process.
  2. INPUT PARAMETER:
    TO DATE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    7
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    FileMan date to end retrieval process.
  3. INPUT PARAMETER:
    STATUS
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    7
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    "A"  : Get only the appealed requests  (-RV,-PR)
"O"  : Get only the open
"C"  : "          " closed  (-D,-G,-P)
"P"  : "          " pending
"N"  : "          "other nondisclosured (-NR,-RF,-ND,-NV,-RC)
"E"  : "          " entered in error
"X"  : "          " cancelled
"ALL": Gets all statuses
 
This parameter may be any combination of the codes delimited by the '^' 
or the word 'ALL'.
  4. INPUT PARAMETER:
    CLERK(S)
    PARAMETER TYPE:
    LIST
    MAXIMUM DATA LENGTH:
    99
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    5
    DESCRIPTION:
    This is an array of clerk IENs.
        (1-n)=IEN to file 200
  5. INPUT PARAMETER:
    DIVISION(S)
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    99
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    4
    DESCRIPTION:
    This parameter is used by the routine if the person requesting this 
report holds the DSIR MDIV key.  It is a list of internal Institution 
File entry numbers (File 4), delimited by '^'.
RETURN PARAMETER DESCRIPTION
Return Array ^ delimited
        1. Division Name
        2. ROI Clerk Name
        3. Patient Name
        4. Patient SSN
        5. Type of Request
        6. External Date Received
        7. External Date Closed (If present)
