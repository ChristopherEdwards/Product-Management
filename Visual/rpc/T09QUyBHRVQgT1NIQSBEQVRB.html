OOPS GET OSHA DATA (1421)
Name Value
NAMEOOPS GET OSHA DATA
TAGOSHA
ROUTINEOOPSGUIF
RETURN VALUE TYPEGLOBAL ARRAY
WORD WRAP ONTRUE
DESCRIPTION
This broker call returns the data needed to produce the Log of Federal 
Occupational Injuries and Illnesses Report.
INPUT PARAMETER
  1. INPUT PARAMETER:
    INPUT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    256
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This parameter contains the Start Date and End date of the Report, the Station
IEN or an 'A' to include all stations at that facilitiy and whether to include
names on the report.  The format is STARTDT^ENDDT^STATION^Y (or N if no).
  2. INPUT PARAMETER:
    CALL
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    10
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This paramter contains the menu that has called the report.  This is mainly
being used to exclude names from displaying on the Union menu.
RETURN PARAMETER DESCRIPTION
The return parameter will contain the data, delimited by ^, for the data to be
displayed in the report.  The return parameter name is RESULTS.
