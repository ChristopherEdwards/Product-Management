DSIR SET ADDRESS INACTIVE (2518)
Name Value
NAMEDSIR SET ADDRESS INACTIVE
TAGTOGLINAC
ROUTINEDSIROI6
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
This RPC sets or resets the Address Inactive flag in the ROI Address file.
INPUT PARAMETER
  1. INPUT PARAMETER:
    ADRPTR
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    ADRPTR - IEN of ADDRESS (NUMERIC)
  2. INPUT PARAMETER:
    FLAGVAL
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    1
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    FLAGVAL - SET OF VALUES FOR THE INACTIVE FLAG.  ZERO(0) = ACTIVE AND ONE 
(1) = INACTIVE. (NUMERIC 0 OR 1)
RETURN PARAMETER DESCRIPTION
RETURN VALUES
  1^UPDATE COMPLETED
  OR
