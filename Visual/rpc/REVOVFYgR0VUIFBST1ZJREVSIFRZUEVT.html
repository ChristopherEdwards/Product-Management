DENTV GET PROVIDER TYPES (1927)
Name Value
NAMEDENTV GET PROVIDER TYPES
TAGTYPE
ROUTINEDENTVRP5
RETURN VALUE TYPEARRAY
AVAILABILITYAGREEMENT
DESCRIPTION
Gets a list of Provider Types from the DENTAL PROVIDER TYPE (220.51)
file.
RETURN PARAMETER DESCRIPTION
Returns an array containing the Provider Types and their 1-2 digit code.
Returns List[1] := -1^error message if no active provider types found.
The format of the return array is:
  list[n] := p1^p2  where
      p1  := Provider Type name
      p2  := code (numeric)
