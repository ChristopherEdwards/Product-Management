DSIF CNSLTS UNHIDE CONSULT (3238)
Name Value
NAMEDSIF CNSLTS UNHIDE CONSULT
TAGUNHIDE
ROUTINEDSIFCNS
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
This RPC will unhide a Consult hidden in the FBCS Consult Tracking 
(#19655) file.
INPUT PARAMETER
  1. INPUT PARAMETER:
    DSIFIEN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    The IEN of the Consult you wish to unhide.
RETURN PARAMETER DESCRIPTION
The RPC will return either 1^successful, or -1^unsuccessful. In the case 
of trying to unhide an already visible consult, it will return 0^message.
