IBARXM TRANS DATA (770)

Name Value
NAMEIBARXM TRANS DATA
TAGTRANS
ROUTINEIBARXMR
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYRESTRICTED
SUPPRESS RDV USER SETUPYes
DESCRIPTION
This remote procedure is used to receive transaction information from a
remote facility and reply with an acknowledgement.
This RPC call does not send user data to remote side.
INPUT PARAMETER
  1. INPUT PARAMETER:
    IBD
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is the data sent by the remote treating facility to be filled in
file 354.71.
  2. INPUT PARAMETER:
    IBICN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the patient's ICN.
RETURN PARAMETER DESCRIPTION
The return is the transaction IEN on this facility, indicating acceptance
of the transmission.  Otherwise a 0 is returned.
