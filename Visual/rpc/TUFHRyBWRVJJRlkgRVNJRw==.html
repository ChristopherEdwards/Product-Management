MAGG VERIFY ESIG (967)

Name Value
NAMEMAGG VERIFY ESIG
TAGESIG
ROUTINEMAGQLOG
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYRESTRICTED
DESCRIPTION
Verifies the Electronic signature of the current user.  
Uses 'DUZ' defined in the partition.
INPUT PARAMETER
  1. INPUT PARAMETER:
    ESIG
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    20
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the encrypted electronic signature entered by the user on the
Imaging window.
RETURN PARAMETER DESCRIPTION
Returns   "1^message"   for success
          "0^message"   for failure or error.
