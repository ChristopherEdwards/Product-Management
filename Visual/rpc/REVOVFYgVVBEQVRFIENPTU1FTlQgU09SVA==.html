This will re-sort a providers canned comments within a category. It will 
return a 1^Successful or -1^Unsuccessful.

  1. INPUT PARAMETER:
    DENTV
    PARAMETER TYPE:
    LIST
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This should be a list of "CATEGORY^SORT ORDER^IEN" for all of the canned 
comments for the provider.
RETURN PARAMETER DESCRIPTION
The return will return a 1^Successful or a -1^Unsuccessful message.
