DENTV REPORT PROVIDERS (1924)
Name Value
NAMEDENTV REPORT PROVIDERS
TAGPROV
ROUTINEDENTVAU
RETURN VALUE TYPEARRAY
AVAILABILITYAGREEMENT
DESCRIPTION
Returns an array containing all dental providers with 8 character 
Provider Ids that belong to the selected division.  If the provider has 
no divisions listed in file 200, then they show up under any division 
selection.
INPUT PARAMETER
  1. INPUT PARAMETER:
    STN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    10
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    Pass if you wish to sort providers by station.
  2. INPUT PARAMETER:
    ACT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    10
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    Pass this flag if you wish to screen for only active providers.
RETURN PARAMETER DESCRIPTION
Return data will be in the following format:
  data(1)=p1^p2^p3^p4 where
     p1 = provider name
     p2 = provider ien (pointer to file 200)
     p3 = 8 character dental provider id
     p4 = user's division(s) separated by commas
