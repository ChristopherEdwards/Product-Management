DSIR STATUS DISCREPANCY RPT (2082)
Name Value
NAMEDSIR STATUS DISCREPANCY RPT
TAGSTATDISC
ROUTINEDSIROIR2
RETURN VALUE TYPEGLOBAL ARRAY
WORD WRAP ONTRUE
VERSION5.4
DESCRIPTION
This RPC returns data from the DSIR STATUS HISTORY file to review which 
request are being entered as closed on one day yet the status date is a 
previous date.
INPUT PARAMETER
  1. INPUT PARAMETER:
    START DATE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    7
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    FileMan format start date.
  2. INPUT PARAMETER:
    END DATE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    7
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    FileMan end date. Optional - defaults to current date.
  3. INPUT PARAMETER:
    CLERK(S)
    PARAMETER TYPE:
    LIST
    MAXIMUM DATA LENGTH:
    99
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    Array where each element is equal to the DUZ of selected clerks.  
Optional - no entries passed will return all clerks for time period.
RETURN PARAMETER DESCRIPTION
Return Values:
 
-1 ^ Missing Start Date!
-2 ^ No Records Found For Specified Clerk(s)!
Clerk Name ^ Request Name ^ Open Date ^ Closed Date ^ Status Code ^ 
Status Date ^ Date User Entered Status
