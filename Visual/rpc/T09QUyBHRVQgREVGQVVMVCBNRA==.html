OOPS GET DEFAULT MD (1392)
Name Value
NAMEOOPS GET DEFAULT MD
TAGDEFMD
ROUTINEOOPSGUI2
RETURN VALUE TYPEARRAY
AVAILABILITYPUBLIC
WORD WRAP ONFALSE
DESCRIPTION
This call will return the Default Medical Doctor information based on an IEN
in the ASISTS 2260 file being passed in. The information returned is from the
file 2262.
INPUT PARAMETER
  1. INPUT PARAMETER:
    IEN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    DESCRIPTION:
    This input value must be a valid IEN from the ASISTS 2260 file.
RETURN PARAMETER DESCRIPTION
The results array will return data in the following format.
 
  Results(0)="1^Valid Results" or "0^No Valid Results"
  Results(1)=PROVIDER NAME
  Results(2)=PROVIDER ADDRESS
  Results(3)=PROVIDER CITY
  Results(4)=PROVIDER STATE
  Results(5)=PROVIDER ZIP CODE
  Results(6)=PROVIDER TITLE
