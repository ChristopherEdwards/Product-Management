DVBAB REPORT DISCHARGE (895)

Name Value
NAMEDVBAB REPORT DISCHARGE
TAGSTRT
ROUTINEDVBAB53
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYRESTRICTED
INACTIVEACTIVE
WORD WRAP ONTRUE
DESCRIPTION
Generates a discharge report, in either standard or delimited format, for 
the specified parameters.
INPUT PARAMETER
  1. INPUT PARAMETER:
    BDATE
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    Beginning date in a date range to use for retrieving results for the 
report.
  2. INPUT PARAMETER:
    EDATE
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    Ending date in a date range to use for retrieving results for the report.
  3. INPUT PARAMETER:
    ADTYPE
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    Valid discharge code values:
  'A' - Recieving A&A
  'P' - Pension
  'S' - Service Connected
  'L' - All discharge types
  4. INPUT PARAMETER:
    DVBADLMTR
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    4
    DESCRIPTION:
    This is an optional parameter, which when defined with the final end date 
selected in the GUI, will produce a delimited report. If this parameter 
is undefined, null or 0 then the standard report format will be generated.
