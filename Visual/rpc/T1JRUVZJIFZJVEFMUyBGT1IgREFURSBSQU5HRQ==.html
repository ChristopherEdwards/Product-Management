ORQQVI VITALS FOR DATE RANGE (490)

Name Value
NAMEORQQVI VITALS FOR DATE RANGE
TAGVITALS
ROUTINEORQQVI
RETURN VALUE TYPEARRAY
AVAILABILITYRESTRICTED
DESCRIPTION
Function returns a patient's vital measurements between start date and 
stop date.
INPUT PARAMETER
  1. INPUT PARAMETER:
    PATIENT ID
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    REQUIRED:
    YES
    DESCRIPTION:
    Patient id (DFN) from Patient File (#2).
  2. INPUT PARAMETER:
    START DATE/TIME
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    DESCRIPTION:
    Start date/time for vital retrieval in Fileman format.
  3. INPUT PARAMETER:
    STOP DATE/TIME
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    DESCRIPTION:
    Stop date/time for vital retrieval in Fileman format.
RETURN PARAMETER DESCRIPTION
Array of patient vitals within start and stop date/times.  If no start and
stop dates are indicated, the most recent are returned.  Vitals are 
returned in the format:
vital ien^vital type^rate/value^date/time taken
