ORQQXQA USER (170)
Name Value
NAMEORQQXQA USER
TAGUSER
ROUTINEORQQXQA
RETURN VALUE TYPEARRAY
AVAILABILITYSUBSCRIPTION
DESCRIPTION
Function returns notifications for current user.
INPUT PARAMETER
  1. INPUT PARAMETER:
    USER ID
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    REQUIRED:
    YES
    DESCRIPTION:
    The record number of the user (DUZ) from the New Person File (#200).
  2. INPUT PARAMETER:
    START DATE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    DESCRIPTION:
    The start date of the period to retrieve notifications for the user in 
Fileman format.
  3. INPUT PARAMETER:
    STOP DATE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    DESCRIPTION:
    The stop date of the period to retrieve notifications for the user in
Fileman format.
RETURN PARAMETER DESCRIPTION
Array of notifications for the user delimited by "^".  Returned data
includes: alert/notification id (XQAID)^alert message^patient id (DFN).
