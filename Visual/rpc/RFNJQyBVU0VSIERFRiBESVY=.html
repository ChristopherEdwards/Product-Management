DSIC USER DEF DIV (1619)
Name Value
NAMEDSIC USER DEF DIV
TAGDIV
ROUTINEDSICDUZ
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYAGREEMENT
DESCRIPTION
This will return the default division for a user.  If no default is 
indicated in the DIVISION multiple, then the return value will be:
 
  1. if the user has only one division, and that division is not
     explicitly marked as NO for default division, then return that
     division
  2. else, if the SITE parameter is passed with a value of 1, then
     return the default value for the facility.
INPUT PARAMETER
  1. INPUT PARAMETER:
    XDUZ
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    20
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the pointer to file 200 for the user for which to return the 
default division.  The default value is DUZ
  2. INPUT PARAMETER:
    SITE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    1
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This boolean flag (1 or 0) will be used if the user has no default 
division specified in the DIVISION multiple in file 200.  If no default 
division, then return the value from $$SITE^VASITE.
RETURN PARAMETER DESCRIPTION
Return the default division for the user:
  ien file 4 ^ institution name ^ institution station number
  on error, or no default, return -1^message
