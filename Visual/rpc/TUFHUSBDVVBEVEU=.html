MAGQ CUPDTE (996)
Name Value
NAMEMAGQ CUPDTE
TAGCUPD
ROUTINEMAGQBPRG
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYRESTRICTED
DESCRIPTION
This RPC is used by the purge routine to update the magnetic references in
the image file(#2005) for images have migrated to heretofore unmapped
network locations.
INPUT PARAMETER
  1. INPUT PARAMETER:
    FILEPATH
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the network path on which the image file was found.
  2. INPUT PARAMETER:
    FNAME
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is the filename, including extension, which requires its location
updated.
  3. INPUT PARAMETER:
    IEN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    This is the internal entry number of the image file (#2005) which requires
updating.
RETURN PARAMETER DESCRIPTION
NULL: no return value is defined.
