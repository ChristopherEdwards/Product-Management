PXRM REMINDER EVALUATION (2020)
Name Value
NAMEPXRM REMINDER EVALUATION
TAGALIST
ROUTINEPXRMRPCA
RETURN VALUE TYPEARRAY
AVAILABILITYSUBSCRIPTION
DESCRIPTION
 Allows evaluation of a list of reminders. Returns a list of clinical
 reminders due/applicable or not applicable to the patient.
INPUT PARAMETER
  1. INPUT PARAMETER:
    PATIENT ID
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    16
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    Patient identifier from the patient file [#2]
  2. INPUT PARAMETER:
    REMINDER ARRAY
    PARAMETER TYPE:
    LIST
    MAXIMUM DATA LENGTH:
    16
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    List of reminders in format :
  
         array(1) = reminder ien1 (from #811.9)
         array(2) = reminder ien2
         etc
RETURN PARAMETER DESCRIPTION
List of reminders in format:
 
IEN^PRINT NAME^DUE DATE/TIME^LAST OCCURRENCE DATE/TIME^PRIORITY^DUE
 
  DUE=0 - Applicable
  DUE=1 - Due
  DUE=2 - Not Applicable
