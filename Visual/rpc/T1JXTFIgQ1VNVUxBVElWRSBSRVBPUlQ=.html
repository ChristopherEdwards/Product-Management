ORWLR CUMULATIVE REPORT (193)
Name Value
NAMEORWLR CUMULATIVE REPORT
TAGCUM
ROUTINEORWLR
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYSUBSCRIPTION
WORD WRAP ONTRUE
DESCRIPTION
This call returns an up to date laboratory cumulative report for a given 
patient.
INPUT PARAMETER
  1. INPUT PARAMETER:
    DFN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    YES
    DESCRIPTION:
    This parameter identifies the patient for whom the lab cumulative report 
should be obtained.
RETURN PARAMETER DESCRIPTION
Data is returned in a table containing two sections.  The first section
lists the lines in the cumulative report where a header begins, in the 
format:
 
  line number ^ header name

The end of this section is delimited by a line of text: [REPORT TEXT]

After [REPORT TEXT] the rest of the lines comprise the text of the 
cumulative report.
