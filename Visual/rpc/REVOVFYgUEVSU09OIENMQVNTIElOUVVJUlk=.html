DENTV PERSON CLASS INQUIRY (3025)
Name Value
NAMEDENTV PERSON CLASS INQUIRY
TAGPROVINQ
ROUTINEDENTVUTL
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYSUBSCRIPTION
DESCRIPTION
This is an RPC wrapper for the Kernal call $$GET^XUA4A72 to determine the 
active providers class information.
INPUT PARAMETER
  1. INPUT PARAMETER:
    PROV
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    The provider DUZ that is to be checked.
  2. INPUT PARAMETER:
    DENTDAT
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is the date used to determine the providers class (in case a 
historical lookup is required).
RETURN PARAMETER DESCRIPTION
Returns a 1^ if the provider information is found and returned.
Returns a -1^ if an error has occured; with a brief message to determine 
what was wrong.
