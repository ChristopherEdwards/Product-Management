DENTV DELETE UNFILED DATA (3015)
Name Value
NAMEDENTV DELETE UNFILED DATA
TAGDELU
ROUTINEDENTVRHD
RETURN VALUE TYPEARRAY
AVAILABILITYAGREEMENT
INACTIVEACTIVE
DESCRIPTION
Input takes in an array of IENS for deletion. Upon completion, the RPC 
will return an array of IENS with a message. 0^File record deleted if 
successful, or -1^Error deleting record if unsuccessful.
INPUT PARAMETER
  1. INPUT PARAMETER:
    DATA
    PARAMETER TYPE:
    LIST
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is an array of IEN records to be deleted from file 228.7
RETURN PARAMETER DESCRIPTION
The return array is formatted the same as the input array. It will return 
an array of IENS with a message specifying if the delete was 
successful or unsuccessful.
