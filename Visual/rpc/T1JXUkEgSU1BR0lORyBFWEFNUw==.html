ORWRA IMAGING EXAMS (241)

Name Value
NAMEORWRA IMAGING EXAMS
TAGEXAMS
ROUTINEORWRA
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYSUBSCRIPTION
WORD WRAP ONTRUE
DESCRIPTION
This remote procedure call returns a list on imaging exams for a
specific patient.
INPUT PARAMETER
  1. INPUT PARAMETER:
    DFN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    20
    REQUIRED:
    YES
    DESCRIPTION:
    Internal entry number of patient in the Patient file [^DPT].
RETURN PARAMETER DESCRIPTION
The format of the returned global array is the following:

   Piece        Description         Format
   -----        -----------         ------
     1          Exam ID:            Inverse FM date/time of Exam_-_Case IEN
     2          Exam date and time  FM date/time format
     3          Procedure name      Free text
     4          Case Number         Number
     5          Status              Free text
     6          Severity            Y or null (ie. Is it an abnormal result?)
