DSIF CNSLTS LEXCODE (3236)

Name Value
NAMEDSIF CNSLTS LEXCODE
TAGLEXCODE
ROUTINEDSIFCNS1
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
Returns the Lexicon code for a given IEN.
INPUT PARAMETER
  1. INPUT PARAMETER:
    DSIFIEN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the Lexicon IEN you wish to know the code for.
  2. INPUT PARAMETER:
    DSIFAPP
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is the Lexicon application for which you are looking.
 
I.e. ICD or CPT.
  3. INPUT PARAMETER:
    DSIFDT
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    NO
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    This is the date you wish to look for the code; if you pass in nothing it 
assumes the current date.
RETURN PARAMETER DESCRIPTION
Returns the code value for the IEN
 
or
 
-1^Error.
