DVBAB FORM DATA BACKUP (2037)
Name Value
NAMEDVBAB FORM DATA BACKUP
TAGBACKUP
ROUTINEDVBABFRM
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYRESTRICTED
DESCRIPTION
Makes a backup copy of a CAPRI template in case of data loss.  The backup 
is restored through the CAPRI GUI.
INPUT PARAMETER
  1. INPUT PARAMETER:
    DVBIEN
    PARAMETER TYPE:
    LITERAL
    DESCRIPTION:
    The IEN of the template for which a backup is being created
  2. INPUT PARAMETER:
    LISTBOX TEXT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    50
    REQUIRED:
    NO
    DESCRIPTION:
    Option field to allow the user to enter custom text to be attached to the 
saved data.  Displayed in the listbox on the GUI side.
