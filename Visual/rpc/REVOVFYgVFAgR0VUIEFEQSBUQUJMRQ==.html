DENTV TP GET ADA TABLE (1896)
Name Value
NAMEDENTV TP GET ADA TABLE
TAGADA
ROUTINEDENTVTP1
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYAGREEMENT
WORD WRAP ONTRUE
DESCRIPTION
This returns the ADA table (file 228) for the Discus Treatment Planning 
system.
INPUT PARAMETER8994.02, , {}
RETURN PARAMETER DESCRIPTION
return ^TMP("DENT",$J,cpt_code) = p1^p2^...^p13
       ^TMP("DENT",$J,cpt_code,1) = long description (250 chars)
 
  p1 = cpt code          p8 = endo (boolean)
  p2 = cpt description   p9 = local code (boolean)
  p3 = condition         p10 = quick code (boolean)
  p4 = area              p11 = apico (boolean)
  p5 = material          p12 = retro (boolean)
  p6 = diagnosis code    p13 = hemi (boolean)
  p7 = ranged (boolean)  p14 = tooth related (boolean)
                         P15 = diagnosis description
  boolean in treatment planning is -1 for true, 0 for false
 
  header records are also returned:
  ^TMP("DENT",$J,cpt_header_code) = cpt code^cpt description
