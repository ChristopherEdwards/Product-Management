DSIF INP INVOICE CHECK (2675)

Name Value
NAMEDSIF INP INVOICE CHECK
TAGINVCHK
ROUTINEDSIFUTL
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYAGREEMENT
VERSION3.2
DESCRIPTION
Pass in the IEN of the 7078 or the 583 file, and the file (7078 or 583)
 
Returns the Invoice number(s) linked to the 7078 or 583 if one exists.
INPUT PARAMETER
  1. INPUT PARAMETER:
    IEN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    25
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    IEN of file 7078 or 583.
  2. INPUT PARAMETER:
    FILE
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    4
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
     Must be "583" (Unauthorized)
     or "7078" (CH)
RETURN PARAMETER DESCRIPTION
Outputs:  
 1^No invoice referenced
 
 or 
 -1^Invoice#1,Invoice#2 (existing referenced invoice numbers)
 -1^Invalid file entry
