DENTV GET ALL PROVIDERS (1926)
Name Value
NAMEDENTV GET ALL PROVIDERS
TAGPROV
ROUTINEDENTVRP5
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYAGREEMENT
WORD WRAP ONTRUE
DESCRIPTION
Gets all providers (including inactive) for edit purposes.
RETURN PARAMETER DESCRIPTION
Returns an array List[n] := dental provider data
Returns List[1] := -1^error message if there are no dental providers.
The format of the dental provider data is List[n]:=p1^p2^p3^p4 where
  p1=provider name (LAST,FIRST MI)
  p2=provider ien (220.5)
  p3=old dental prov id
  p4=new provider id
  p5=status flag (-1=inactive)
  p6=provider ien (200)
