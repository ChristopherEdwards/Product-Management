DSIF CNSLTS SVC W/SYNONYMS (3222)
Name Value
NAMEDSIF CNSLTS SVC W/SYNONYMS
TAGSVCSYN
ROUTINEDSIFCNS1
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYRESTRICTED
WORD WRAP ONTRUE
DESCRIPTION
This RPC returns a list of Services with synonyms.
 
Cloned from ORQQCN SVC W/SYNONYMS.
INPUT PARAMETER
  1. INPUT PARAMETER:
    DSIFROOT
  2. INPUT PARAMETER:
    DSIFSTRT
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the Service IEN you wish to start building the return list from.
  3. INPUT PARAMETER:
    DSIFWHY
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is whether you want to return the list for display only, or for use 
in ordering or forwarding a request.
  4. INPUT PARAMETER:
    DSIFSYN
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    1
    SEQUENCE NUMBER:
    3
    DESCRIPTION:
    This is a boolean inputon whether you want the return to include synonyms:
 
1 = Yes, 0 No.
  5. INPUT PARAMETER:
    DSIFOIEN
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    4
    DESCRIPTION:
    This is the Consult IEN, and is optional.
RETURN PARAMETER DESCRIPTION
Returns a list of services as follows:
 
Service IEN^Service Name or Synonym^Has Children?^Service Usage^Orderable 
Item IEN.
