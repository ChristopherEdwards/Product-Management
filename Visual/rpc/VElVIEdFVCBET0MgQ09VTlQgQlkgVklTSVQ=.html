TIU GET DOC COUNT BY VISIT (354)
Name Value
NAMETIU GET DOC COUNT BY VISIT
TAGDOCCNT
ROUTINETIUSRVLV
RETURN VALUE TYPESINGLE VALUE
AVAILABILITYSUBSCRIPTION
DESCRIPTION
This remote procedure returns the number of documents that are linked to a 
particular visit.
INPUT PARAMETER
  1. INPUT PARAMETER:
    VSIT
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the record number (IEN) of the VISIT in the VISIT FILE.
RETURN PARAMETER DESCRIPTION
This is the COUNT of TIU DOCUMENTS which are currently associated with the 
VISIT passed in the VSIT input parameter.
