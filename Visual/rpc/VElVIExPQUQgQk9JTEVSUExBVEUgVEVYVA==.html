TIU LOAD BOILERPLATE TEXT (89)

Name Value
NAMETIU LOAD BOILERPLATE TEXT
TAGBLRSHELL
ROUTINETIUSRVD
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYSUBSCRIPTION
WORD WRAP ONTRUE
DESCRIPTION
This RPC will load the boilerplate text associated with the selected
title, and execute the methods for any objects embedded in the boilerplate
text.
INPUT PARAMETER
  1. INPUT PARAMETER:
    TITLE
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    DESCRIPTION:
    This is a pointer to the TIU DOCUMENT DEFINITION FILE (#8925.1), which
identifies the title of the document to be loaded.
  2. INPUT PARAMETER:
    DFN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    DESCRIPTION:
    This is the pointer to the patient for whom the document is being entered.
  3. INPUT PARAMETER:
    VSTRING
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    NO
    DESCRIPTION:
    This optional parameter is the visit string, formatted as:

HLOC;VISIT_DATE;SVC_CAT

It's necessary when objects included in the boilerplate are sensitive to the 
attributes of the visit (e.g., the object VISIT DATE).
