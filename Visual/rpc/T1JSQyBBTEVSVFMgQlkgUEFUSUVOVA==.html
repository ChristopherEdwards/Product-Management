ORRC ALERTS BY PATIENT (1748)
Name Value
NAMEORRC ALERTS BY PATIENT
TAGPAT
ROUTINEORRCXQ
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYSUBSCRIPTION

DESCRIPTION
This call returns a list of the patient's non-ADT alerts.

    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:

  2. INPUT PARAMETER:
    USER
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:

RETURN PARAMETER DESCRIPTION
This call returns the list of alert ID's, in the form:
  "Item=" _ ID ^ Text ^ Date in the form YYYYMMDDHHMMSS
where ID is the alert identifier string prefixed by "NOT:".
