OOPS EDIT SITE PARAMETER (1412)
Name Value
NAMEOOPS EDIT SITE PARAMETER
TAGPARMEDT
ROUTINEOOPSGUI6
RETURN VALUE TYPESINGLE VALUE
DESCRIPTION
This broker call files changes to the Site Parameter File (#2262) name and 
District Office, if made.
INPUT PARAMETER
  1. INPUT PARAMETER:
    INPUT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    256
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This input parameter contains the IEN of the Site Parameter File (#2262), the
Site Parameter Name, and the District Office.  The format is:
IEN^SITE NAME^DISTRICT OFFICE NAME.
RETURN PARAMETER DESCRIPTION
The return parameter, RESULTS, contains a message indicating whether changes
made to the top level record were successful (Update Site data Successful)
or were not (Update Site data was NOT Successful) successful.
