DENTV GET VA-DSS PRODUCTS (1962)
Name Value
NAMEDENTV GET VA-DSS PRODUCTS
TAGVADSS
ROUTINEDENTVTP1
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYAGREEMENT
WORD WRAP ONTRUE
DESCRIPTION
Returns all of the VA-DSS product lines from the DENTAL GROUPING FOR 
VA-DSS file.
RETURN PARAMETER DESCRIPTION
Returns ^TMP("DENT",$J,ien) = p1^p2 where
  p1 = ien
  p2 = Code-Description
 
Returns -1^error message if error (or no product lines) found.
