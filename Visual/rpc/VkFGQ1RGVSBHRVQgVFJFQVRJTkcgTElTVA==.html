DESCRIPTION
Given a patient DFN, this will return a list of treating facilities.
INPUT PARAMETER
  1. INPUT PARAMETER:
    Patient DFN.
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    255
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    Patient DFN from PATIENT file (#2).
RETURN PARAMETER DESCRIPTION
This will return a list of treating facilities in the following
format:
STATION #^STATION NAME^DATE OF LAST TREATMENT
 
If there is a problem (patient does not exist, etc.), the first
line item will be a -1^error message.
