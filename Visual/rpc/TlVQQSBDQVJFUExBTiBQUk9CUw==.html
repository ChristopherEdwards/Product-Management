Provides a list of the problems and interventions for a patient's care
plan for the current admission.

    DA

RETURN PARAMETER DESCRIPTION
Returns a list of problems & interventions from the latest care plan in 
^TMP($J).  Global looks like this:
 
^TMP(562303312,"MH",1176)=MH^Mental Health^Anxiety (Actual)^9/25/09@1407^
Minimize the feelings of anxiety related to unidentified sources of
anticipated danger^No change/stable^ 9/29/09@0842^Education - Encourage
meditation to reduce anxiety^ 9/25/09@1407^Not on file^Not on file^
167^1176^
 
Pieces are:
 
1. Tab in the exe
2. Problem name
3. Date problem indentified
4. Desired outcome
5. Last problem evaluation
6. Problem evaluation date
7. Intervention
8. Intervention date
9. Intervention status
10. Intervention status date
11. Problem IEN from file 1927.2
12. Intervention IEN from file 1927.24
