NUPA PAST/FUTURE DATE (2936)
Name Value
NAMENUPA PAST/FUTURE DATE
TAGPF
ROUTINENUPABCL
RETURN VALUE TYPESINGLE VALUE
WORD WRAP ONTRUE
DESCRIPTION
Returns a 1 or a 0 when checking to see if a date is in the past or 
future, based on $H.
INPUT PARAMETER
  1. INPUT PARAMETER:
    VAR 1
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    30
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    The date you want to check.  Should be in "7/7/11@12:30:00 PM" format.
  2. INPUT PARAMETER:
    VAR 2
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    1
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    Variable should be set to "P" if you want to see if the date is in the 
past, or set it to "F" if you want to see if it's in the future.
RETURN PARAMETER DESCRIPTION
Will be a 1 or 0 based upon the future/past parameter.
