OOPS APPROVE SIGN FOR EMPLOYEE (1413)
Name Value
NAMEOOPS APPROVE SIGN FOR EMPLOYEE
TAGEN1
ROUTINEOOPSGUI8
RETURN VALUE TYPEARRAY
DESCRIPTION
This broker call will determine whether the individual can approve the
Workers' Comp person being able to sign for the employee if accessed from the
Safety or Employee Health menus.
INPUT PARAMETER
  1. INPUT PARAMETER:
    INPUT
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    256
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This parameter contains the IEN of the ASISTS record to be evaluated and the
calling menu in the format IEN^CALLING MENU.
RETURN PARAMETER DESCRIPTION
The RESULTS(0) node in the array will contain a 1 or 0.  If a 1, then the
individual approving the Workers' Comp signing for the Employee is Ok, otherwise
the signature form will not be available.
