DSIF FB INSTITUTION (3011)

Name Value
NAMEDSIF FB INSTITUTION
TAGGETINST
ROUTINEDSIFUTL
RETURN VALUE TYPEGLOBAL ARRAY
AVAILABILITYSUBSCRIPTION
WORD WRAP ONTRUE
VERSION3.2
DESCRIPTION
Input Station number, return all populated fields in file #4.
INPUT PARAMETER
  1. INPUT PARAMETER:
    STATION NUMBER
    PARAMETER TYPE:
    LITERAL
    MAXIMUM DATA LENGTH:
    9
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
     Station number lookup, using "D" cross reference (FIELD 99) 1st, then by
 name field.
RETURN PARAMETER DESCRIPTION
 Return populated fields in file #4:
FIELD NUMBER^VALUE
 
 (example)
 
 ^TMP("DSIFUTL",33580,.01)=".01^BELLEVILLE IL CBOC"
 ^TMP("DSIFUTL",33580,.02)=".02^ILLINOIS"
 ^TMP("DSIFUTL",33580,1.01)="1.01^ARCADES CENTER"
 ^TMP("DSIFUTL",33580,1.03)="1.03^BELLEVILLE"
 ^TMP("DSIFUTL",33580,1.04)="1.04^62223"
 ^TMP("DSIFUTL",33580,4.01)="4.01^ARCADES CENTER"
 ^TMP("DSIFUTL",33580,4.03)="4.03^BELLEVILLE"
 ^TMP("DSIFUTL",33580,4.04)="4.04^ILLINOIS"
 ^TMP("DSIFUTL",33580,4.05)="4.05^62223"
 ^TMP("DSIFUTL",33580,11)="11^National"
 ^TMP("DSIFUTL",33580,95)="95^VA"
 ^TMP("DSIFUTL",33580,97)="97^VA"
 ^TMP("DSIFUTL",33580,99)="99^657GA"
 ^TMP("DSIFUTL",33580,100)="100^BELLEVILLE IL CBOC (657GA)"
 or
 
 ^TMP("DSIFUTL",33580,0)="0^entry not found"
 or
 ^TMP("DSIFUTL",33580,0)="-1^Invalid entry"
