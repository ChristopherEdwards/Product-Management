TIU TEMPLATE GET TEMPLATE (3014)
Name Value
NAMETIU TEMPLATE GET TEMPLATE
TAGGETTMPLT
ROUTINETIUSRVT2
RETURN VALUE TYPEARRAY
AVAILABILITYAGREEMENT
DESCRIPTION
This RPC gets basic attributes of a given TIU Template (file 8927).
INPUT PARAMETER
  1. INPUT PARAMETER:
    IEN
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the internal entry number in the TIU TEMPLATE FILE 8927 of the 
template.
RETURN PARAMETER DESCRIPTION
Basic attributes of the given template are returned in '^'-delimited form.
