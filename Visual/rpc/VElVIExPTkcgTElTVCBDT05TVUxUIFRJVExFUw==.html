TIU LONG LIST CONSULT TITLES (137)
Name Value
NAMETIU LONG LIST CONSULT TITLES
TAGLNGCNSLT
ROUTINETIUSRVD
RETURN VALUE TYPEARRAY
AVAILABILITYSUBSCRIPTION
DESCRIPTION
This RPC serves data to a longlist of selectable TITLES for CONSULTS.
INPUT PARAMETER
  1. INPUT PARAMETER:
    FROM
    PARAMETER TYPE:
    LITERAL
    REQUIRED:
    YES
    SEQUENCE NUMBER:
    1
    DESCRIPTION:
    This is the reference title from which the longlist is scrolling.
  2. INPUT PARAMETER:
    DIR
    PARAMETER TYPE:
    LITERAL
    SEQUENCE NUMBER:
    2
    DESCRIPTION:
    This is the direction in which the longlist is scrolling from the
reference title.
RETURN PARAMETER DESCRIPTION
This is an array of the 44 nearest titles to that indicate by the user in
the direction passed by the longlist component.
